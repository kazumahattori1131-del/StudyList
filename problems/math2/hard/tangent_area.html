数学 II 積分・面積 接線と曲線が囲む面積
★★★★☆ 上級目安 15 分
問　題
曲線 $C\colon y = x^3 - 3x$ 上の点 $(1,\ -2)$ における接線 $\ell$ の方程式を求めよ。
また、接線 $\ell$ と曲線 $C$ で囲まれた部分の面積 $S$ を求めよ。
考え方のヒント
$y'$ で接線の傾きを求め、接線の方程式を立てる。
次に接線と曲線の交点を因数分解で求め、定積分で面積を計算する。

（参考：京都大学 2001年 文系数学 第2問 類題）

解説 Step 1：接線の方程式を求める
1 導関数を求める： $$y' = 3x^2 - 3$$
2 $x = 1$ における傾きを計算する： $$y'\big|_{x=1} = 3(1)^2 - 3 = 3 - 3 = 0$$ 傾き $0$ の水平な接線である。
3 点 $(1, -2)$ を通り傾き $0$ の直線： $$\ell\colon y = -2$$
接線の方程式
$$y = -2$$
解説 Step 2：接線と曲線の交点を求める
1 交点は $x^3 - 3x = -2$ を解けばよい： $$x^3 - 3x + 2 = 0$$
2 $x = 1$ を代入して確認：$1 - 3 + 2 = 0$ ✓　→ $(x-1)$ が因子。
3 因数分解する： $$x^3 - 3x + 2 = (x-1)(x^2 + x - 2) = (x-1)(x-1)(x+2) = (x-1)^2(x+2)$$
4 $(x-1)^2(x+2) = 0$ より $x = 1$（重解）または $x = -2$。
交点は $x = -2$ と $x = 1$。
$x=1$ が重解 ⟺ 曲線 $C$ と接線 $\ell$ はここで接触している。
解説 Step 3：積分の準備（上下関係の確認）
1 $-2 < x < 1$ の範囲で曲線と接線の上下を確認する： $$(\text{曲線}) - (\text{接線}) = (x^3 - 3x) - (-2) = x^3 - 3x + 2 = (x-1)^2(x+2)$$
2 $-2 < x < 1$ において $(x-1)^2 > 0$、$(x+2) > 0$ なので： $$(x-1)^2(x+2) > 0$$ よって曲線 $C$ が接線 $\ell$ の上側にある。
3 面積は次の定積分で求められる： $$S = \int_{-2}^{1} \bigl\{(x^3-3x) - (-2)\bigr\}\,dx = \int_{-2}^{1}(x^3-3x+2)\,dx$$
解説 Step 4：定積分の計算
1 不定積分を求める： $$\int(x^3-3x+2)\,dx = \frac{x^4}{4} - \frac{3x^2}{2} + 2x + C$$
2 各端点での値を計算する：
$x = 1$ のとき： $$\frac{1}{4} - \frac{3}{2} + 2 = \frac{1 - 6 + 8}{4} = \frac{3}{4}$$ $x = -2$ のとき： $$\frac{16}{4} - \frac{12}{2} + 2(-2) = 4 - 6 - 4 = -6$$
3 面積を求める： $$S = \frac{3}{4} - (-6) = \frac{3}{4} + 6 = \frac{3}{4} + \frac{24}{4} = \frac{27}{4}$$
まとめ 解答のまとめ
接線 $\ell$ の方程式
点 $(1,-2)$ での傾き $= 0$
水平接線 $$y = -2$$
交点の $x$ 座標
$(x-1)^2(x+2)=0$ より
重解に注意 $$x = -2,\quad x = 1$$
囲まれた面積 $S$
$\displaystyle\int_{-2}^{1}(x-1)^2(x+2)\,dx$ $$S = \dfrac{27}{4}$$
答　え
接線：$y = -2$　　面積：$\displaystyle S = \dfrac{27}{4}$
ポイント 受験テクニック・確認事項
  • 「重解をもつ ⟹ 接している」が面積計算の鍵。交点方程式を因数分解すると重解 $(x=1)$ と単解 $(x=-2)$ が現れ、積分区間が決まる。
  • $(x-\alpha)^2(x-\beta)$ 型の定積分には次の公式が使える： $$\left|\int_{\alpha}^{\beta}(x-\alpha)^2(x-\beta)\,dx\right| = \frac{|\beta - \alpha|^4}{12}$$ ここでは $\alpha = -2$、$\beta = 1$（積分は $\int_{-2}^{1}(x+2)(x-1)^2\,dx$）： $$\frac{|1-(-2)|^4}{12} = \frac{3^4}{12} = \frac{81}{12} = \frac{27}{4} \quad\checkmark$$
  • この $\dfrac{1}{12}$ 公式は「接線と曲線が囲む面積」問題で頻出。「三乗型」面積の公式として覚えておくと大幅な時間短縮になる。
  • 符号確認：積分区間内で（曲線）$-$（接線）$= (x-1)^2(x+2) > 0$ を必ず確認すること。
  • 連続性チェック（接点での一致）：$x=1$ で $y = 1-3 = -2$、接線 $y=-2$ ✓。$x=-2$ で $y = -8+6 = -2$、接線 $y=-2$ ✓。
模範解答 記述答案例（そのまま使えるレベル）
接線と曲線が囲む面積
$y = x^3 - 3x$ を微分すると $y' = 3x^2 - 3$。 $x = 1$ での傾きは $y'\big|_{x=1} = 3 - 3 = 0$。
点 $(1, -2)$ を通り傾き $0$ の接線は $$\ell\colon y = -2$$
次に接線 $\ell$ と曲線 $C$ の交点を求める。$x^3 - 3x = -2$ より $$x^3 - 3x + 2 = 0$$ $x=1$ が解であるから $(x-1)$ で因数分解すると $$(x-1)^2(x+2) = 0$$ ゆえに $x = -2$ または $x = 1$（重解）。
面積の計算
$-2 < x < 1$ において $(x-1)^2(x+2) > 0$ より、曲線 $C$ は接線 $\ell$ の上側にある。よって $$S = \int_{-2}^{1}(x^3-3x+2)\,dx = \int_{-2}^{1}(x-1)^2(x+2)\,dx$$
$$= \left[\frac{x^4}{4} - \frac{3x^2}{2} + 2x\right]_{-2}^{1}$$ $$= \left(\frac{1}{4} - \frac{3}{2} + 2\right) - \left(4 - 6 - 4\right)$$ $$= \frac{3}{4} - (-6) = \frac{3}{4} + 6$$
$$S = \dfrac{27}{4}$$

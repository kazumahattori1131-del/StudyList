【問題】
次の定積分を求めよ。

　∫₀¹ 1/(1+x²)² dx

【解説】

**1. 問題の確認**

分母が (1+x²) の2乗になった定積分です。(1+x²) だけなら tan の逆関数で積分できますが、2乗になると話が違います。この「2乗」をどう処理するかが問題の核心です。

**2. 方針**

分母に (1+x²) が現れたら、三角置換 x = tanθ が有効です。なぜかというと、1 + tan²θ = 1/cos²θ という三角関数の公式が使えて、(1+x²) をきれいに変換できるからです。2乗版でも同じ置換を使います。置換後に cos²θ の積分になり、半角公式で処理できます。

**3. 解法の流れ**

x = tanθ（0 ≤ θ ≤ π/4）とおきます。このとき dx = dθ/cos²θ です。

1+x² = 1/cos²θ なので、(1+x²)² = 1/cos⁴θ。

被積分関数と dx を合わせると：

　dx / (1+x²)² = (dθ/cos²θ) ÷ (1/cos⁴θ) = cos²θ dθ

2乗が消えて、cos²θ だけが残ります。

**4. 計算・ポイント**

積分範囲も変換します。x: 0 → 1 のとき θ: 0 → π/4 です。

cos²θ を積分するには半角公式 cos²θ = (1 + cos2θ)/2 を使います。

　∫₀^(π/4) cos²θ dθ = ∫₀^(π/4) (1 + cos2θ)/2 dθ = [θ/2 + sin2θ/4]₀^(π/4)

θ = π/4 を代入すると sin(π/2) = 1 なので：

　= π/8 + 1/4

**5. 結論**

　∫₀¹ dx/(1+x²)² = π/8 + 1/4

**6. 重要なコツ・理解ポイント**

この問題のポイントは、置換後に dx の変換と分母の変換が「うまく約分される」ことです。cos⁴θ の逆数と cos²θ が掛け合わさることで cos²θ だけが残ります。この約分の流れを事前にイメージできると、計算ミスが減ります。「(1+x²) が分母に出たら x = tanθ」という反射的な判断を身につけましょう。

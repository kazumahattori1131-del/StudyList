【問題】
数列 {aₙ} の初項から第 n 項までの和を Sₙ とするとき、すべての自然数 n に対して

　Sₙ = 2aₙ - n

が成り立つ。a₁ を求め、一般項 aₙ を求めよ。

【解説】

**1. 問題の確認**

Sₙ（和）と aₙ（一般項）が同じ式の中に混在しています。Sₙ は数列の和、aₙ は1つの項です。この2つが絡み合った条件から aₙ を求めます。

**2. 方針**

Sₙ と aₙ が混在する問題は、次の2ステップで解きます。まず n = 1 を代入して a₁ を確定させます。次に n ≥ 2 のとき aₙ = Sₙ - S_{n-1} を使って aₙ と a_{n-1} の関係式（漸化式）を導きます。この漸化式を解いて一般項を求めます。

**3. 解法の流れ**

n = 1 を代入：S₁ = a₁ = 2a₁ - 1 より a₁ = 1。

n ≥ 2 のとき、aₙ = Sₙ - S_{n-1} を使います。

　aₙ = (2aₙ - n) - (2a_{n-1} - (n-1))
　　 = 2aₙ - 2a_{n-1} - 1

整理すると aₙ = 2a_{n-1} + 1 という漸化式が得られます。

**4. 計算・ポイント**

aₙ = 2a_{n-1} + 1 を変形します。aₙ + 1 = 2(a_{n-1} + 1) の形にすると、これは公比 2 の等比数列です。

初項：a₁ + 1 = 2 なので、aₙ + 1 = 2 · 2^{n-1} = 2ⁿ。

よって aₙ = 2ⁿ - 1。

最後に n = 1 のとき 2¹ - 1 = 1 = a₁ となり、一般項の式が n = 1 でも成立することを確認します。この確認を忘れずに！

**5. 結論**

　a₁ = 1、aₙ = 2ⁿ - 1

**6. 重要なコツ・理解ポイント**

「Sₙ と aₙ が混在」する問題の鉄則は「①n = 1 で a₁ を求める、②n ≥ 2 で aₙ = Sₙ - S_{n-1} を使う」です。特に n = 1 を別に確認することは絶対に忘れてはいけません。n ≥ 2 の場合から求めた一般式が n = 1 でも成立するかどうか、最後に必ずチェックする習慣をつけましょう。

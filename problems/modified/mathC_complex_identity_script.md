【問題】
複素数 z が z + 1/z = 1 を満たすとき、z³ + 1/z³ の値を求めよ。

【解説】

**1. 問題の確認**

z + 1/z = 1 という条件から、z³ + 1/z³ の値を求めます。z そのものを求めてから z³ を計算することもできますが、それは遠回りです。条件式を賢く使う方法を考えます。

**2. 方針**

(z + 1/z)³ を展開してみましょう。展開すると中に z³ + 1/z³ が現れます。左辺は 1³ = 1 とわかっているので、右辺を整理すれば z³ + 1/z³ が求まります。z の値を具体的に計算しなくて済むのがポイントです。

**3. 解法の流れ**

(z + 1/z)³ を展開します。

　(z + 1/z)³ = z³ + 3z²·(1/z) + 3z·(1/z²) + 1/z³
　　　　　　 = z³ + 1/z³ + 3(z + 1/z)

左辺は 1³ = 1、右辺の 3(z + 1/z) = 3×1 = 3 なので：

　1 = z³ + 1/z³ + 3

よって z³ + 1/z³ = 1 - 3 = -2。

**4. 計算・ポイント**

展開で間違いやすいのが2乗のときです。(z + 1/z)² = z² + 2 + 1/z²（中間に +2 が入る）を確認しておきましょう。z² + 1/z² と書いてしまうミスが多いです。3乗の展開でも同様に、途中の項 3(z + 1/z) を正確に処理することが大切です。

**5. 結論**

　z³ + 1/z³ = -2

**6. 重要なコツ・理解ポイント**

z + 1/z = k という条件が与えられたとき、z² + 1/z²、z³ + 1/z³ を順に求める技法を覚えましょう。これは (z + 1/z) の累乗を展開して整理するだけです。z を直接求めなくても答えが出るため、計算量を大幅に減らせます。

一般に aₙ = zⁿ + 1/zⁿ とおくと、aₙ₊₁ = k·aₙ - aₙ₋₁ という漸化式が成り立ちます。高い累乗を求めるときもこの漸化式で順々に計算できます。

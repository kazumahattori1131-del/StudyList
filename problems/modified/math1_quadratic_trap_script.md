【問題】
次の不等式がすべての実数 x に対して成り立つような、定数 a の値の範囲を求めよ。

　ax² + 2x + a > 0

【解説】

**1. 問題の確認**

この問題は、二次式 ax² + 2x + a が、どんな x を代入しても必ず正になる a の範囲を求めるものです。「すべての x で成立する」というのが重要なポイントです。

**2. 方針**

ここが最大の落とし穴です。多くの人が「判別式 D < 0 を解けばいい」と考えて進めます。でも実はそれだけでは不十分です。「すべての x で正になる」ためには、放物線が上に開いていること（a > 0）と、x 軸と交わらないこと（D < 0）の2つの条件が同時に必要です。

**3. 解法の流れ**

まず a = 0 の場合を確認します。a = 0 のとき不等式は 2x > 0 となり、x = 0 のとき成立しないので不適です。

次に a ≠ 0 の場合。「すべての x で ax² + 2x + a > 0」が成立するには：
- 条件① : a > 0（上に開く放物線）
- 条件② : 判別式 D = 4 - 4a² < 0（x 軸と交わらない）

この2つを同時に満たす a を求めます。

**4. 計算・ポイント**

条件②を解きます。4 - 4a² < 0 を整理すると a² > 1、つまり a < -1 または a > 1 です。

条件①（a > 0）と条件②（a < -1 または a > 1）の共通部分を求めます。a > 0 かつ a > 1 の共通部分は a > 1 です。a > 0 かつ a < -1 を同時に満たす a は存在しません。

よって答えは a > 1 です。

**5. 結論**

　a > 1

**6. 重要なコツ・理解ポイント**

この問題の鍵は「a の符号の場合分け」です。判別式だけ計算して「a < -1 または a > 1」と答えてしまう人がとても多いですが、a が負なら放物線は下に開くので、どこかで必ず x 軸より下になります。

「二次式がすべての x で正」= 「上に開く（a > 0）かつ x 軸と接触しない（D < 0）」という2条件セットを必ずチェックする習慣をつけましょう。

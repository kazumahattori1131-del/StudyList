解法の核心 両辺を $2^{n+1}$ で割ると等差数列に化ける
🔑 $a_{n+1} = 2a_n + 2^n$ の両辺を $2^{n+1}$ で割ると： $$\frac{a_{n+1}}{2^{n+1}} = \frac{a_n}{2^n} + \frac{1}{2}$$ $b_n = \dfrac{a_n}{2^n}$ とおけば $b_{n+1} = b_n + \dfrac{1}{2}$（等差数列！）
右辺の $2^n$ の部分が鍵。$\dfrac{2^n}{2^{n+1}} = \dfrac{1}{2}$（定数）になるのがポイント。
解説 等差数列として解く
1 $b_n = \dfrac{a_n}{2^n}$ とおく。漸化式を $2^{n+1}$ で割ると： $$b_{n+1} = b_n + \frac{1}{2}$$
2 これは公差 $\dfrac{1}{2}$ の 等差数列。初項： $$b_1 = \frac{a_1}{2^1} = \frac{1}{2}$$
3 等差数列の一般項： $$b_n = \frac{1}{2} + (n-1)\cdot\frac{1}{2} = \frac{n}{2}$$
4 $b_n = \dfrac{a_n}{2^n}$ を戻す： $$a_n = 2^n \cdot b_n = 2^n \cdot \frac{n}{2} = n \cdot 2^{n-1}$$

一般化 この発想はどこでも使える
一般化
$a_{n+1} = p\,a_n + q\cdot p^n$（$p \neq 0$）の形：
両辺を $p^{n+1}$ で割ると $\dfrac{a_{n+1}}{p^{n+1}} = \dfrac{a_n}{p^n} + \dfrac{q}{p}$
→ $b_n = \dfrac{a_n}{p^n}$ は公差 $\dfrac{q}{p}$ の等差数列。
→ $a_n = p^n \cdot b_n = p^n \!\left(b_1 + (n-1)\dfrac{q}{p}\right)$
類題
$a_1 = 2$、$a_{n+1} = 3a_n + 3^n$ のとき、$a_n$ を求めよ。
$b_n = \dfrac{a_n}{3^n}$：$b_{n+1} = b_n + \dfrac{1}{3}$、$b_1 = \dfrac{2}{3}$
$b_n = \dfrac{2}{3} + (n-1)\dfrac{1}{3} = \dfrac{n+1}{3}$
$a_n = 3^n \cdot \dfrac{n+1}{3} = (n+1)\cdot 3^{n-1}$
模範解答 記述答案例
漸化式・等差化
$b_n = \dfrac{a_n}{2^n}$ とおく。
漸化式 $a_{n+1} = 2a_n + 2^n$ の両辺を $2^{n+1}$ で割ると：
$$\frac{a_{n+1}}{2^{n+1}} = \frac{a_n}{2^n} + \frac{1}{2}$$
すなわち $b_{n+1} = b_n + \dfrac{1}{2}$。
これは公差 $\dfrac{1}{2}$ の等差数列。初項 $b_1 = \dfrac{a_1}{2} = \dfrac{1}{2}$。
$$b_n = \frac{1}{2} + (n-1)\cdot\frac{1}{2} = \frac{n}{2}$$
$b_n = \dfrac{a_n}{2^n}$ より $a_n = 2^n \cdot \dfrac{n}{2}$。
$$a_n = n\cdot 2^{n-1}$$

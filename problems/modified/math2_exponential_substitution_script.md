【問題】
次の方程式を解け。

　4ˣ - 3·2ˣ - 4 = 0

【解説】

**1. 問題の確認**

4ˣ と 2ˣ が混在した指数方程式です。このまま解こうとしても、指数同士を直接足したり引いたりはできません。まずこの2つの関係に気づくことが出発点です。

**2. 方針**

4ˣ = (2²)ˣ = (2ˣ)² という変形がポイントです。つまり 4ˣ は 2ˣ の2乗と見ることができます。ここで t = 2ˣ という置き換えをすると、方程式が t の2次方程式に変わります。2次方程式なら因数分解で解けます。

**3. 解法の流れ**

t = 2ˣ（t > 0）とおきます。このとき 4ˣ = t² なので、方程式は次のようになります。

　t² - 3t - 4 = 0

これを因数分解すると (t-4)(t+1) = 0 となり、t = 4 または t = -1 です。

**4. 計算・ポイント**

ここで絶対に忘れてはいけないのが、t = 2ˣ > 0 という条件です。2の何乗も必ず正になるので、t は正の値しかとれません。よって t = -1 は不適で、t = 4 のみ採用します。

t = 2ˣ = 4 = 2² より、底が同じなので指数を比較して x = 2 です。

**5. 結論**

　x = 2

**6. 重要なコツ・理解ポイント**

この問題の型を覚えましょう。「4ˣ と 2ˣ が混在」「9ˣ と 3ˣ が混在」という形は、t 置き換えで2次方程式に帰着できます。大切なのは2点です。まず 4ˣ = (2ˣ)² という変形に気づくこと。次に t > 0 の条件を忘れずに確認すること。解いた後に必ず t > 0 のチェックを行う習慣をつけてください。

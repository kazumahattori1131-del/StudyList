⚠ 落とし穴①：$n = 1$ の確認を忘れる
$a_n = S_n - S_{n-1}$ は $n \geq 2$ でのみ成り立つ。
$n = 1$ のとき $S_0$ は定義されないので、$n = 1$ を直接代入して $a_1$ を求める必要がある。
この確認を怠ると $a_1$ が一般項の式と一致するかどうか確認できない。
⚠ 落とし穴②：漸化式の整理の符号ミス
$a_n = S_n - S_{n-1}$ を展開するとき、$S_{n-1} = 2a_{n-1} - (n-1)$ の符号に注意。
$-(2a_{n-1} - (n-1)) = -2a_{n-1} + n - 1$ と正しく展開すること。

①まず $n = 1$ で $a_1$ を確定
② $n \geq 2$ で $a_n = S_n - S_{n-1}$ を使い $a_n$ の漸化式を導く

解法の核心 $a_n = S_n - S_{n-1}$ で漸化式に帰着
🔑 $n \geq 2$ のとき $a_n = S_n - S_{n-1}$ を利用： $$a_n = (2a_n - n) - (2a_{n-1} - (n-1)) = 2a_n - 2a_{n-1} - 1$$ 整理すると $a_n = 2a_{n-1} + 1$（漸化式！）
→ $a_n + 1 = 2(a_{n-1} + 1)$ の形に変形すれば等比数列。
$a_n + 1$ が公比 $2$ の等比数列になることがポイント。初項 $a_1 + 1 = 2$ から求める。
解説 $n = 1$ から順に解く
1 $n = 1$：$S_1 = a_1 = 2a_1 - 1$ より $a_1 = 1$。
2 $n \geq 2$：$a_n = S_n - S_{n-1} = (2a_n - n) - (2a_{n-1} - (n-1))$ $$= 2a_n - 2a_{n-1} - 1$$ 整理：$-a_n = -2a_{n-1} - 1$ → $a_n = 2a_{n-1} + 1$
3 $a_n + 1 = 2(a_{n-1} + 1)$ → 公比 $2$ の等比数列。
初項：$a_1 + 1 = 2$
$$a_n + 1 = 2 \cdot 2^{n-1} = 2^n$$

一般化 この発想はどこでも使える
一般化
$S_n = pa_n + f(n)$ の形（$p \neq 1$）は $a_n = S_n - S_{n-1}$ で $a_n$ の漸化式が出る。
漸化式の形を $a_n + c = r(a_{n-1} + c)$ に変形して等比数列に帰着する。
必ず $n = 1$ を別に確認すること！
類題
$S_n = 2a_n + n$ のとき、$a_1$ と一般項 $a_n$ を求めよ。
$n=1$：$a_1 = 2a_1 + 1$ → $a_1 = -1$
$n \geq 2$：$a_n = 2a_n - 2a_{n-1} + 1$ → $a_n = 2a_{n-1} - 1$
$a_n - 1 = 2(a_{n-1} - 1)$、初項 $a_1 - 1 = -2$
$a_n - 1 = -2 \cdot 2^{n-1} = -2^n$ → $a_n = 1 - 2^n$
模範解答 記述答案例
数列の和・漸化式
$n = 1$ のとき：$S_1 = a_1 = 2a_1 - 1$ より $a_1 = 1$。
$n \geq 2$ のとき：$a_n = S_n - S_{n-1}$
$$= (2a_n - n) - (2a_{n-1} - (n-1)) = 2a_n - 2a_{n-1} - 1$$
整理すると $a_n = 2a_{n-1} + 1$、すなわち $a_n + 1 = 2(a_{n-1} + 1)$。
これは公比 $2$ の等比数列。初項 $a_1 + 1 = 2$ より：
$$a_n + 1 = 2 \cdot 2^{n-1} = 2^n$$
$n = 1$ のとき $2^1 - 1 = 1 = a_1$ より式は $n = 1$ でも成立。
$$a_n = 2^n - 1$$

ありがちなミス ここで間違えやすい
⚠ 落とし穴①：$z$ を直接求めようとする
$z + \dfrac{1}{z} = 1$ を整理すると $z^2 - z + 1 = 0$ となり、
$z = \dfrac{1 \pm \sqrt{-3}}{2}$ と求められるが、
その後 $z^3$ を計算するのは非常に手間がかかる。条件式を展開して利用する方が効率的。
⚠ 落とし穴②：$(z + 1/z)^2$ の展開ミス
$\left(z + \dfrac{1}{z}\right)^2 = z^2 + \dfrac{1}{z^2}$ と書いてしまうミス。
正しくは $\left(z + \dfrac{1}{z}\right)^2 = z^2 + 2 \cdot z \cdot \dfrac{1}{z} + \dfrac{1}{z^2} = z^2 + 2 + \dfrac{1}{z^2}$。
中間の $+2$ を忘れないこと。
💡 $(z + 1/z)^n$ の展開を利用して
　$z^2 + 1/z^2$、$z^3 + 1/z^3$ を順に求めよう！

ヒント
① $\left(z + \dfrac{1}{z}\right)^2$ を展開すると $z^2 + \dfrac{1}{z^2}$ が求まる。
② $\left(z + \dfrac{1}{z}\right)^3$ を展開すると $z^3 + \dfrac{1}{z^3}$ が現れる。
③ $z + \dfrac{1}{z} = 1$ であることを使って数値を計算しよう。
解法の核心 $(z + 1/z)^3$ の展開を利用する
🔑 $(z + 1/z)^3$ を展開すると： $$\left(z + \frac{1}{z}\right)^3 = z^3 + 3z + \frac{3}{z} + \frac{1}{z^3} = z^3 + \frac{1}{z^3} + 3\left(z + \frac{1}{z}\right)$$ $z + \dfrac{1}{z} = 1$ を代入すれば $z^3 + \dfrac{1}{z^3}$ が直接求まる！
$z$ を具体的に求めなくても、条件式の累乗を展開するだけで答えが出る。
解説 段階的に求める
1 $\left(z + \dfrac{1}{z}\right)^2 = z^2 + 2 + \dfrac{1}{z^2} = 1^2 = 1$ より： $$z^2 + \frac{1}{z^2} = -1$$
2 $\left(z + \dfrac{1}{z}\right)^3$ を展開： $$\left(z + \frac{1}{z}\right)^3 = z^3 + \frac{1}{z^3} + 3\left(z + \frac{1}{z}\right)$$
3 左辺 $= 1^3 = 1$、右辺の $3\left(z + \dfrac{1}{z}\right) = 3 \times 1 = 3$ より： $$1 = z^3 + \frac{1}{z^3} + 3$$
4 $$z^3 + \frac{1}{z^3} = 1 - 3 = -2$$
答え 解答まとめ
2乗を展開
$\left(z + \dfrac{1}{z}\right)^2 = 1$
$$z^2 + \frac{1}{z^2} = -1$$

$\left(z + \dfrac{1}{z}\right)^3 = 1$
$= z^3 + \dfrac{1}{z^3} + 3$
答え
$z^3 + \dfrac{1}{z^3}$ $$= 1 - 3 = -2$$

一般化 この発想はどこでも使える
一般化
$a_n = z^n + \dfrac{1}{z^n}$ とおくと、$z + \dfrac{1}{z} = k$ のとき漸化式が成り立つ： $$a_{n+1} = k \cdot a_n - a_{n-1}, \quad a_0 = 2,\; a_1 = k$$ これを使えば $z^n + \dfrac{1}{z^n}$ を順に求められる（$z$ を求めずに）。
類題
$z + \dfrac{1}{z} = -1$ のとき、$z^3 + \dfrac{1}{z^3}$ を求めよ。
$\left(z + \dfrac{1}{z}\right)^3 = z^3 + \dfrac{1}{z^3} + 3\left(z + \dfrac{1}{z}\right)$
$(-1)^3 = z^3 + \dfrac{1}{z^3} + 3(-1)$
$-1 = z^3 + \dfrac{1}{z^3} - 3$ より $z^3 + \dfrac{1}{z^3} = 2$
模範解答 記述答案例
複素数・恒等式
$\left(z + \dfrac{1}{z}\right)^3$ を展開すると：
$$\left(z + \frac{1}{z}\right)^3 = z^3 + 3z^2 \cdot \frac{1}{z} + 3z \cdot \frac{1}{z^2} + \frac{1}{z^3} = z^3 + \frac{1}{z^3} + 3\left(z + \frac{1}{z}\right)$$
$z + \dfrac{1}{z} = 1$ より左辺 $= 1$、右辺の $3\left(z + \dfrac{1}{z}\right) = 3$ なので：
$$1 = z^3 + \frac{1}{z^3} + 3$$
$$z^3 + \frac{1}{z^3} = -2$$

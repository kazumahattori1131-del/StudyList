【問題】
数列 {aₙ} が次の条件を満たすとき、一般項 aₙ を求めよ。

　a₁ = 1,　a_{n+1} = 2aₙ + 2ⁿ　(n ≧ 1)

【解説】

**1. 問題の確認**

この漸化式、右辺に「2aₙ」と「2ⁿ」の2つの項があります。求めるのは aₙ の一般項、つまりどんな式で表せるかです。

**2. 方針**

「2ⁿ が右辺にあるから特性方程式で…」と考えた人、ちょっと待ってください。実はもっとシンプルな方法があります。右辺の 2ⁿ を見てください。係数の「2」と底が同じですよね。こういうときは、両辺を 2^{n+1} でそっくり割るのがコツです。なぜかというと、割ることで右辺の 2ⁿ が「定数」に変わるからです。そうすると、等差数列の形が見えてきます。

**3. 解法の流れ**

まず、bₙ = aₙ / 2ⁿ という新しい数列を定義します。漸化式の両辺を 2^{n+1} で割ってみましょう。

左辺は a_{n+1} / 2^{n+1}、これがそのまま b_{n+1} です。

右辺は、2aₙ / 2^{n+1} と 2ⁿ / 2^{n+1} に分けて計算します。第1項は aₙ / 2ⁿ = bₙ、第2項は 1/2（定数！）です。

つまり、b_{n+1} = bₙ + 1/2 という等差数列になりました。

**4. 計算・ポイント**

初項を求めます。b₁ = a₁ / 2¹ = 1/2 です。ここで要注意。「b₁ = a₁ = 1」とそのまま書いてしまうミスが非常に多いです。bₙ は aₙ を 2ⁿ で割ったものなので、必ず 2¹ = 2 で割ってください。

公差が 1/2 の等差数列なので、一般項は次のように求まります。

　bₙ = 1/2 + (n-1) · 1/2 = n/2

あとは bₙ = aₙ / 2ⁿ に戻すだけです。

　aₙ = 2ⁿ · bₙ = 2ⁿ · n/2 = n · 2^{n-1}

**5. 結論**

　aₙ = n · 2^{n-1}

**6. 重要なコツ・理解ポイント**

この問題の型をしっかり覚えておきましょう。「a_{n+1} = p·aₙ + q·pⁿ」の形、つまり右辺の指数の底と係数の底が同じときは、両辺を p^{n+1} で割ると必ず等差数列に変換できます。

なぜ等差数列になるかというと、pⁿ / p^{n+1} = 1/p で、n が消えて定数になるからです。この「割り算で n を消す」発想が、この型の問題を解くための核心です。

見た目は難しそうでも、パターンに気づければ計算はとてもシンプルです。「右辺の底と係数の底が同じ → 両辺を割る」これを今日の持ち帰りポイントにしてください。

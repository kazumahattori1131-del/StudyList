⚠ 落とし穴①：AとBが隣り合う場合の数え忘れ
AとBをひとかたまりとして数えるとき、A・B の並び順（AB と BA）の2通りを忘れがち。
「AとBがひとまとまり = 1通り」と思い込んでしまうと答えが半分になる。
⚠ 落とし穴②：全体を直線順列で考えてしまう
分母を $5!$ としてしまうミス。円順列では「回転して同じ配置は1通り」なので
全体は $(5-1)! = 4! = 24$ 通り。
💡 「〜しない確率」は余事象が有効。
　$P(\text{隣り合わない}) = 1 - P(\text{隣り合う})$
　AとBが隣り合う場合：（A,B）を1人として $(4-1)!\times 2!$

解法の核心 余事象「隣り合う」を数える
🔑 余事象：A と B が隣り合う場合を数える

（A,B）をひとまとまりにすると → 4人の円順列 $(4-1)! = 6$ 通り
A と B の内部順列 $\times 2! = 2$ 通り
→ 合計 $6 \times 2 = 12$ 通り
全体 $24$ 通りのうち隣り合う場合が $12$ 通り → $P(\text{隣り合う}) = \dfrac{12}{24} = \dfrac{1}{2}$
解説 余事象を使って計算する
1 全体の場合の数：5人の円順列 $(5-1)! = 4! = 24$（通り）
2 A と B が隣り合う場合：(A,B) を1人とみなして4人の円順列 $$(4-1)! \times 2! = 6 \times 2 = 12 \text{（通り）}$$
3 A と B が隣り合う確率： $$P(\text{隣り合う}) = \frac{12}{24} = \frac{1}{2}$$
4 余事象より： $$P(\text{隣り合わない}) = 1 - \frac{1}{2} = \frac{1}{2}$$

一般化 この発想はどこでも使える
一般化
$n$ 人の円順列で特定の2人が隣り合う確率： $$P(\text{隣り合う}) = \frac{(n-2)! \times 2}{(n-1)!} = \frac{2}{n-1}$$ → $P(\text{隣り合わない}) = 1 - \dfrac{2}{n-1} = \dfrac{n-3}{n-1}$
$n=5$：$P = \dfrac{2}{4} = \dfrac{1}{2}$ ✓
類題
A〜F の6人が円形に並ぶとき、A と B が隣り合わない確率を求めよ。
全体：$(6-1)! = 120$、隣り合う：$(5-1)! \times 2 = 48$
$P(\text{隣り合う}) = \dfrac{48}{120} = \dfrac{2}{5}$
$P(\text{隣り合わない}) = 1 - \dfrac{2}{5} = \dfrac{3}{5}$

円順列・余事象
5人が円形に並ぶ場合の数は $(5-1)! = 24$（通り）。
余事象として A と B が隣り合う場合を数える。
(A,B) をひとまとまり（1人）とみると4人の円順列：$(4-1)! = 6$（通り）
A と B の内部の並び方：$2! = 2$（通り）
隣り合う場合の数：$6 \times 2 = 12$（通り）
$$P(\text{隣り合わない}) = 1 - \frac{12}{24} = \frac{1}{2}$$

【問題】
次の方程式を解け。ただし x > 0、x ≠ 1 とする。

　log_x(2) + log_2(x) = 3

【解説】

**1. 問題の確認**

log_x(2) と log_2(x)、底と真数が入れ替わった2つの対数が足し合わさった方程式です。一見似ているようで、この2つは全く別の値です。これをどう整理するかが問題のカギです。

**2. 方針**

底の変換公式を使って log_x(2) を log_2(x) で表します。

　log_x(2) = log₂(2) / log₂(x) = 1 / log₂(x)

これに気づけば、t = log₂(x) とおくことで方程式が 1/t + t = 3 という形になり、t の2次方程式に帰着します。

**3. 解法の流れ**

t = log₂(x) とおきます。x > 0 かつ x ≠ 1 なので、t は 0 以外の任意の実数です。

　1/t + t = 3

両辺に t をかけると（t ≠ 0 なので大丈夫です）：

　1 + t² = 3t　→　t² - 3t + 1 = 0

**4. 計算・ポイント**

解の公式を使います。t = (3 ± √5) / 2 です。この2つの解はどちらも 0 ではない（√5 ≠ 3 なので）ので、両方適します。

t = log₂(x) を元に戻すと x = 2^t なので：

　x = 2^((3+√5)/2)　または　x = 2^((3-√5)/2)

**5. 結論**

　x = 2^((3+√5)/2)、x = 2^((3-√5)/2)

**6. 重要なコツ・理解ポイント**

底と真数が互いに入れ替わった形「log_x(a) と log_a(x)」を見たら、底の変換公式で log_x(a) = 1 / log_a(x) という逆数の関係に変換するのが定石です。この関係を知っていれば、t = log_a(x) の置き換えで必ず2次方程式に帰着できます。log の問題では「置き換え」が武器になることを覚えておいてください。

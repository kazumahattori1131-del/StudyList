次の定積分を求めよ。 $$\int_0^1 \frac{1}{(1+x^2)^2}\,dx$$

ヒント
① $x = \tan\theta$ とおくと $1 + x^2 = 1 + \tan^2\theta = \dfrac{1}{\cos^2\theta}$（$= \sec^2\theta$）
② $dx = \dfrac{1}{\cos^2\theta}\,d\theta$ を使って被積分関数全体を $\theta$ で表してみよう。
③ 積分範囲：$x: 0 \to 1$ のとき $\theta: 0 \to \dfrac{\pi}{4}$。
解法の核心 $x = \tan\theta$ で分母が $\cos^4\theta$ に変わる
🔑 $x = \tan\theta$ とおくと $$\frac{1}{(1+x^2)^2} = \frac{1}{\sec^4\theta} = \cos^4\theta,\quad dx = \sec^2\theta\,d\theta$$ かけ合わせると $\cos^4\theta \cdot \sec^2\theta\,d\theta = \cos^2\theta\,d\theta$
→ 半角公式で簡単に積分できる形になる！
$\sec^2\theta$（分母）と $dx$（分子の $\sec^2\theta$）が約分されるのがポイント。
解説 置換から積分まで
1 $x = \tan\theta$（$0 \leq \theta \leq \dfrac{\pi}{4}$）とおく。$dx = \sec^2\theta\,d\theta$。
2 $(1+x^2)^2 = \sec^4\theta$ より： $$\frac{dx}{(1+x^2)^2} = \frac{\sec^2\theta\,d\theta}{\sec^4\theta} = \cos^2\theta\,d\theta$$
3 半角公式 $\cos^2\theta = \dfrac{1+\cos 2\theta}{2}$ を使う： $$\int_0^{\pi/4}\cos^2\theta\,d\theta = \int_0^{\pi/4}\frac{1+\cos 2\theta}{2}\,d\theta = \left[\frac{\theta}{2} + \frac{\sin 2\theta}{4}\right]_0^{\pi/4}$$
4 $$= \left(\frac{\pi/4}{2} + \frac{\sin(\pi/2)}{4}\right) - 0 = \frac{\pi}{8} + \frac{1}{4}$$

一般化 この発想はどこでも使える
一般化
$\int \dfrac{1}{(a^2+x^2)^n}\,dx$ の形は $x = a\tan\theta$ で置換すると
$a^2 + x^2 = a^2\sec^2\theta$ となり、次数が下がる。
特に $n=1$ のとき $\dfrac{1}{a}\arctan\dfrac{x}{a} + C$。
類題
$\displaystyle\int_0^{\sqrt{3}} \frac{1}{(1+x^2)^2}\,dx$ を求めよ。
$x = \tan\theta$、範囲 $\theta: 0 \to \dfrac{\pi}{3}$： $\displaystyle\int_0^{\pi/3}\cos^2\theta\,d\theta = \left[\dfrac{\theta}{2}+\dfrac{\sin 2\theta}{4}\right]_0^{\pi/3} = \dfrac{\pi}{6}+\dfrac{\sqrt{3}}{8}$
模範解答 記述答案例
三角置換・定積分
$x = \tan\theta$（$0 \leq \theta \leq \dfrac{\pi}{4}$）とおくと $dx = \sec^2\theta\,d\theta$、$1+x^2 = \sec^2\theta$。
積分範囲：$x: 0 \to 1$ のとき $\theta: 0 \to \dfrac{\pi}{4}$。
$$\int_0^1\frac{dx}{(1+x^2)^2} = \int_0^{\pi/4}\frac{\sec^2\theta}{\sec^4\theta}\,d\theta = \int_0^{\pi/4}\cos^2\theta\,d\theta$$
半角公式 $\cos^2\theta = \dfrac{1+\cos 2\theta}{2}$ を用いると：
$$= \int_0^{\pi/4}\frac{1+\cos 2\theta}{2}\,d\theta = \left[\frac{\theta}{2}+\frac{\sin 2\theta}{4}\right]_0^{\pi/4}$$
$$= \left(\frac{\pi}{8}+\frac{\sin(\pi/2)}{4}\right)-0 = \frac{\pi}{8}+\frac{1}{4}$$
$$\int_0^1\frac{dx}{(1+x^2)^2} = \frac{\pi}{8}+\frac{1}{4}$$

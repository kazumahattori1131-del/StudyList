【問題】
次の不定積分を求めよ。

　∫ eˣ sinx dx

【解説】

**1. 問題の確認**

指数関数と三角関数の積の積分です。部分積分を使おうとしても、積分のたびに eˣ sinx が形を変えて出てきて、一見終わりが見えないように感じます。でも実は、これが解法の鍵になります。

**2. 方針**

I = ∫ eˣ sinx dx とおいて、部分積分を2回行います。すると、右辺に元の I が再び登場します。「I = ～ - I」の形が出たら、それを方程式として解けばいいのです。終わらないように見える積分を、逆に「I を含む方程式」として利用します。

**3. 解法の流れ**

I = ∫ eˣ sinx dx とおきます。

第1回の部分積分（sinx を微分する側にします）：

　I = eˣ sinx - ∫ eˣ cosx dx

次に ∫ eˣ cosx dx を同じ方針（cosx を微分する側）で部分積分します：

　∫ eˣ cosx dx = eˣ cosx - ∫ eˣ(-sinx) dx = eˣ cosx + I

**4. 計算・ポイント**

2回目の部分積分で元の I が再登場しました。代入すると：

　I = eˣ sinx - (eˣ cosx + I)
　I = eˣ sinx - eˣ cosx - I
　2I = eˣ(sinx - cosx)

両辺を2で割って：

　I = eˣ(sinx - cosx) / 2 + C

注意点は2つです。1回目と2回目の部分積分で役割を逆にしないこと。逆にすると 0 = 0 という式になってしまいます。また2回目で (cosx)' = -sinx の符号を忘れないようにしましょう。この「-」があるから I が再登場するのです。

**5. 結論**

　∫ eˣ sinx dx = eˣ(sinx - cosx)/2 + C

**6. 重要なコツ・理解ポイント**

「指数関数 × 三角関数」の積分は、部分積分を同じ方向で2回行うと元の積分が再登場するというパターンがあります。これを知っていれば、「終わらない！」と焦らず「あ、I が出てきた、方程式として解ける」と切り替えられます。「2I = ～」の形に整理して 2 で割る、この流れを体で覚えてください。

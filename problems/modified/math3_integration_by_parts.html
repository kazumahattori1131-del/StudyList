⚠ 落とし穴①：2回目の部分積分で役割を逆にしてしまう
1回目の部分積分で「三角関数を微分する（$\sin x$ を微分）、$e^x$ を積分」と進めた後、
2回目で 役割を逆にする（$e^x$ を微分、$\cos x$ を積分）と $0 = 0$ が出てしまう。
2回とも同じ方針（三角関数の側を微分し続ける）で進めることが重要。
⚠ 落とし穴②：$\cos x$ の微分の符号ミス
2回目の部分積分で $(\cos x)' = -\sin x$ を使うとき、符号 $-$ を忘れがち。
符号が変わることで 元の積分がマイナスで再登場するのがこの問題の核心。
💡 部分積分を2回行うと元の積分が再び現れる。
　「元の積分 $= \ldots -$ 元の積分」の形になったら、
　両辺を整理して解く！

ヒント
① 第1回：$\sin x$ を微分する側、$e^x$ を積分する側として部分積分を行う。
② $\int e^x \cos x\, dx$ が現れる。同じ方針でもう1回部分積分を行う。
③ 元の $\int e^x \sin x\, dx$ が再び出てきたら、方程式として解こう。
解法の核心 元の積分を $I$ とおいて方程式を解く
🔑 $I = \displaystyle\int e^x \sin x \, dx$ とおく。部分積分を2回行うと $$I = e^x \sin x - e^x \cos x - I$$ が得られる。これを整理すると： $$2I = e^x(\sin x - \cos x)$$ → $I$ を求める方程式として解ける！
「積分が終わらない」ように見えても、元に戻ることを逆手にとる発想が鍵。
解説 部分積分を2回行う
1 $I = \displaystyle\int e^x \sin x\, dx$ とおく。
第1回（$\sin x$ を微分、$e^x$ を積分）： $$I = e^x \sin x - \int e^x \cos x \, dx$$
2 $\displaystyle\int e^x \cos x\, dx$ を第2回の部分積分（$\cos x$ を微分、$e^x$ を積分）： $$\int e^x \cos x\, dx = e^x \cos x - \int e^x (-\sin x)\, dx = e^x \cos x + I$$
3 代入：$I = e^x \sin x - (e^x \cos x + I)$ $$I = e^x \sin x - e^x \cos x - I$$ $$2I = e^x(\sin x - \cos x)$$
4 $$I = \frac{e^x(\sin x - \cos x)}{2} + C$$

一般化 この発想はどこでも使える
一般化
$\displaystyle\int e^x \sin x\, dx$ や $\displaystyle\int e^x \cos x\, dx$ のように
「指数関数 × 三角関数」の積分は、部分積分を同じ方向で2回行うと
元の積分が再登場し、方程式として解ける。
この形：$I = (\text{式}) - I$ → $2I = (\text{式})$ → $I = \dfrac{(\text{式})}{2} + C$
類題
$\displaystyle\int e^x \cos x \, dx$ を求めよ。
$J = \displaystyle\int e^x \cos x\, dx$ とおく。
第1回：$J = e^x \cos x + \displaystyle\int e^x \sin x\, dx = e^x \cos x + I$
上の結果 $I = \dfrac{e^x(\sin x - \cos x)}{2} + C$ を代入： $$J = e^x \cos x + \frac{e^x(\sin x - \cos x)}{2} + C = \frac{e^x(\cos x + \sin x)}{2} + C$$
模範解答 記述答案例
部分積分・連鎖
$I = \displaystyle\int e^x \sin x\, dx$ とおく。
部分積分（$\sin x$ を微分、$e^x$ を積分）：
$$I = e^x \sin x - \int e^x \cos x\, dx$$
$\displaystyle\int e^x \cos x\, dx$ にも同じ方針で部分積分（$\cos x$ を微分、$e^x$ を積分）：
$$\int e^x \cos x\, dx = e^x \cos x - \int e^x(-\sin x)\, dx = e^x \cos x + I$$
代入すると：$I = e^x \sin x - (e^x \cos x + I)$
$$2I = e^x(\sin x - \cos x)$$
$$\int e^x \sin x\, dx = \frac{e^x(\sin x - \cos x)}{2} + C$$

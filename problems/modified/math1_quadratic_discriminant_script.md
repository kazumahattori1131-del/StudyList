【問題】
放物線 y = x² - 2ax + a + 2 がx軸と2点で交わるような、定数 a の値の範囲を求めよ。

【解説】

**1. 問題の確認**

この問題は、放物線がx軸と「ちょうど2点」で交わる条件を求めるものです。定数 a の値によって交点の数が変わります。どんな a のときに2点で交わるかを調べます。

**2. 方針**

放物線とx軸の交点の個数は、判別式 D の符号で決まります。「2点で交わる」ということは、D が正になる必要があります。ここで注意してほしいのが、「交わる」という言葉を見て D ≥ 0 としてしまう間違いです。D = 0 のときは「接する」つまり1点で触れるだけなので、2点で交わるには D > 0 が必要です。

**3. 解法の流れ**

y = x² - 2ax + (a+2) の形から、判別式 D/4 を計算します。D/4 を使うと、係数が2の倍数のとき計算がシンプルになります。x の係数は -2a なので、その半分は -a。これを2乗して定数項を引きます。

D/4 = (-a)² - (a+2) = a² - a - 2

**4. 計算・ポイント**

「D/4 = (-a)² - (a+2)」の計算をしっかり確認しましょう。(-a)² = a²（マイナスが消えます）です。a² のままで計算してください。

D/4 > 0 なので、a² - a - 2 > 0 という a についての2次不等式を解きます。左辺を因数分解すると (a-2)(a+1) > 0 となります。2次不等式の解は、両端の値の外側なので a < -1 または a > 2 です。

**5. 結論**

　a < -1 または a > 2

**6. 重要なコツ・理解ポイント**

この問題の核心は、「交点の条件を判別式に翻訳し、それを a の不等式として解く」という2段構えの流れです。「放物線とx軸の関係 → D の符号」という変換が最初の一手。そこから D を a の式として整理し、a の2次不等式を解くという流れを体に染み込ませましょう。

「2点で交わる = D > 0」「接する = D = 0」「交わらない = D < 0」この3つをセットで覚えておいてください。

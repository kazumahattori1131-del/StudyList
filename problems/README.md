## ディレクトリ構造

```
problems/
├── math1/       # 数学I（二次関数・三角比・データ分析）
├── math2/       # 数学II（図形と方程式・三角関数・微積分）
├── mathA/       # 数学A（場合の数・確率・図形の性質）
└── mathB/       # 数学B（数列・ベクトル）

```

数学C 複素数の累乗とド・モアブルの定理
★★★★☆ 応用偏差値60〜70
$z = 1+i$ のとき $z^8$ を求めよ。

$(1+i)^8 = 1^8 + i^8$ と展開しようとしていませんか？　それは禁じ手です。

よくある間違い 二項定理なしに累乗しようとする

❌ よくある間違い①：各項を独立に累乗する

$$(1+i)^8 \;\overset{?}{=}\; 1^8 + i^8 = 1 + (i^2)^4 = 1+1 = 2$$

→ $(a+b)^n \neq a^n + b^n$　累乗は絶対に分配できない！

❌ よくある間違い②：$|z|$ を忘れて角度だけ計算

$$z = \cos\frac{\pi}{4}+i\sin\frac{\pi}{4} \;\overset{?}{\Rightarrow}\; z^8 = \cos 2\pi + i\sin 2\pi = 1$$

→ $|z|=\sqrt{2}$ を掛け忘れ！　極形式は $r(\cos\theta+i\sin\theta)$ の $r$ が必須。

⚠️ 複素数の累乗を直接計算しようとするのは罠。極形式か段階的累乗を使う！
本質ポイント 極形式に変換してド・モアブルを使う

$z=1+i$ を極形式 $r(\cos\theta+i\sin\theta)$ に変換する：

$r = |z| = \sqrt{1^2+1^2} = \sqrt{2}$
$\theta = \dfrac{\pi}{4}$ （偏角：実部・虚部がともに正で等しい）
$\therefore\; z = \sqrt{2}\!\left(\cos\dfrac{\pi}{4}+i\sin\dfrac{\pi}{4}\right)$
ド・モアブルの定理：$z^n = r^n(\cos n\theta + i\sin n\theta)$

※ 絶対値 $r$ の $n$ 乗を忘れると全く違う答えになるので要注意

解説 2つの方法で完全解答
方法A：極形式＋ド・モアブル
$z = \sqrt{2}\!\left(\cos\dfrac{\pi}{4}+i\sin\dfrac{\pi}{4}\right)$
$z^8 = (\sqrt{2})^8\!\left(\cos\dfrac{8\pi}{4}+i\sin\dfrac{8\pi}{4}\right)$
$\phantom{z^8} = 16(\cos 2\pi + i\sin 2\pi)$
$\phantom{z^8} = 16\cdot 1 = \boldsymbol{16}$
方法B：段階的累乗
$z^2 = (1+i)^2 = 1+2i+i^2 = 2i$
$z^4 = (z^2)^2 = (2i)^2 = 4i^2 = -4$
$z^8 = (z^4)^2 = (-4)^2 = \boldsymbol{16}$
どちらの方法でも $z^8 = 16$（実数）。$n$ が大きいほど方法Aが速い。
一般化 ド・モアブルの定理と累乗の戦略
$\bigl[r(\cos\theta+i\sin\theta)\bigr]^n = r^n(\cos n\theta + i\sin n\theta)$

累乗の解法選択

  • $n$ が大きい（6以上など）→ 極形式＋ド・モアブルが最速
  • $z^2$ が簡単な形になる → 段階的累乗（$z^2 \to z^4 \to \cdots$）
  • どちらも：必ず $r = |z|$ を極形式に含める。忘れると答えが $1$ になって終わる
複素数の累乗は必ず「極形式」または「段階的累乗」。直接展開は禁じ手！
類題 同じ手順で解いてみよう
$z = 1+\sqrt{3}\,i$ のとき $z^6$ を求めよ。
答　え
極形式に変換：
$r = |z| = \sqrt{1^2+(\sqrt{3})^2} = \sqrt{4} = 2$、$\theta = \dfrac{\pi}{3}$
$z = 2\!\left(\cos\dfrac{\pi}{3}+i\sin\dfrac{\pi}{3}\right)$

ド・モアブルの定理を適用：
$z^6 = 2^6\!\left(\cos\dfrac{6\pi}{3}+i\sin\dfrac{6\pi}{3}\right) = 64(\cos 2\pi + i\sin 2\pi)$

$\therefore\; z^6 = 64$

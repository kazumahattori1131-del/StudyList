## スライド①　フック

0から3まで x の2乗 マイナス 2x を積分すると、答えは 0 になります。
では、面積も 0 でしょうか？
もちろん違います。正しい面積は 3分の8 です。
なぜ積分値と面積が異なるのか、今から説明します。

## スライド②　問題提示

今回の問題です。
y イコール x の2乗 マイナス 2x、定義域は 0以上3以下 で、x 軸との間に囲まれた面積を求めます。
f カッコ x カッコ閉じ イコール x かける カッコ x マイナス 2 カッコ閉じ なので、x イコール 0 と x イコール 2 で x 軸と交わります。
グラフを見ると、0以上2以下では x 軸より下、2以上3以下では x 軸より上になっています。
この x 軸より下の部分が、今回の落とし穴の正体です。

---

## スライド③　よくある間違い

典型的な間違いを確認します。
0から3まで x の2乗 マイナス 2x を積分すると、x の3乗 割る 3 マイナス x の2乗 の 0から3 で、カッコ 9 マイナス 9 カッコ閉じ マイナス 0 イコール 0 となります。
ここで「面積はゼロ」と答えてしまうのが典型的な間違いです。
なぜ間違いかというと、0以上2以下の区間ではグラフが x 軸より下にあり、この部分の積分値は負になります。
2以上3以下の正の部分と打ち消し合って、合計がゼロになってしまったのです。
積分値をそのまま面積と思ってしまうと、必ずここで失点します。

---

## スライド④　本質ポイント

整理します。x 軸と囲む面積の正しい求め方は4ステップです。
ステップ1。f カッコ x カッコ閉じ イコール 0 の解を求め、x 軸との交点を見つけます。
ステップ2。各区間で f カッコ x カッコ閉じ の正・負を確認し、グラフを描きます。
ステップ3。区間を分割して積分します。f カッコ x カッコ閉じ が 0以下 の区間では、積分の前にマイナスをかけます。ここが核心です。
ステップ4。すべての区間の面積を足し合わせます。
全区間を一本で積分してOKというのは罠です。必ずグラフで正・負を確認してください。

## スライド⑤　解説

それでは正しく解いていきます。
ステップ1。f カッコ x カッコ閉じ イコール x の2乗 マイナス 2x イコール x かける カッコ x マイナス 2 カッコ閉じ イコール 0 より、x イコール 0 または x イコール 2。
ステップ2。0以上2以下では f カッコ x カッコ閉じ が 0以下、2以上3以下では f カッコ x カッコ閉じ が 0以上です。
ステップ3。区間を分割して計算します。
S イコール マイナス 0から2の積分 カッコ x の2乗 マイナス 2x カッコ閉じ プラス 2から3の積分 カッコ x の2乗 マイナス 2x カッコ閉じ。
最初の区間を計算します。x の3乗 割る 3 マイナス x の2乗 の 0から2 は、3分の8 マイナス 4 イコール マイナス 3分の4。これにマイナスをかけて 3分の4。
次の区間。x の3乗 割る 3 マイナス x の2乗 の 2から3 は、カッコ 9 マイナス 9 カッコ閉じ マイナス カッコ 3分の8 マイナス 4 カッコ閉じ イコール 0 マイナス マイナス 3分の4 イコール 3分の4。
合計すると、3分の4 プラス 3分の4 イコール 3分の8 が答えです。

## スライド⑥　一般化

まとめます。
x 軸と y イコール f カッコ x カッコ閉じ で囲まれた面積は、f カッコ x カッコ閉じ の符号で区間を分けて、各区間の積分値の絶対値を足します。
f カッコ x カッコ閉じ が 0以上 の区間はそのまま積分、0以下 の区間はマイナスをかけて積分します。
全区間を一本で積分すると、正と負が打ち消し合って面積が小さくなったり、ゼロになったりします。
これは入試で最もよくある積分ミスのひとつです。グラフを描いて符号を確認する習慣をつけましょう。

## スライド⑦　類題

では類題を一問。
y イコール x の2乗 マイナス x、定義域は 0以上2以下 での x 軸との面積を求める問題です。

f カッコ x カッコ閉じ イコール x かける カッコ x マイナス 1 カッコ閉じ より、x 軸との交点は x イコール 0 と x イコール 1 です。
0以上1以下では f カッコ x カッコ閉じ が 0以下、1以上2以下では f カッコ x カッコ閉じ が 0以上となります。
S イコール マイナス 0から1の積分 カッコ x の2乗 マイナス x カッコ閉じ プラス 1から2の積分 カッコ x の2乗 マイナス x カッコ閉じ。
0から1の積分を計算します。x の3乗 割る 3 マイナス x の2乗 割る 2 の 0から1 は、3分の1 マイナス 2分の1 イコール マイナス 6分の1。これにマイナスをかけて 6分の1。
1から2の積分を計算します。カッコ 3分の8 マイナス 2 カッコ閉じ マイナス カッコ 3分の1 マイナス 2分の1 カッコ閉じ イコール 3分の2 マイナス マイナス 6分の1 イコール 6分の4 プラス 6分の1 イコール 6分の5。
合計すると、6分の1 プラス 6分の5 イコール 1 が答えです。

## スライド⑧　まとめ

今日の一言まとめです。
積分値は面積ではありません。
f カッコ x カッコ閉じ が 0より小さい区間が含まれたら、区間を分けてマイナスをかける。グラフを描く習慣が、この落とし穴を防ぎます。
全区間を一本で積分すると正と負が打ち消し合い、面積がゼロになる罠にハマります。ここだけは絶対に忘れないでください。

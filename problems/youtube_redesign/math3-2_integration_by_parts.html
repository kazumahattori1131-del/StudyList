数学III 部分積分を繰り返す
★★★★☆ 発展偏差値60〜70
問　題
次の不定積分を求めよ。 $$\int e^x \sin x \, dx$$
💬 部分積分を続けると ―― 元に戻ってくる？ それを逆手にとれるか。
ヒント $I = \int e^x \sin x\, dx$ とおいて2回部分積分
HINT： $I = \displaystyle\int e^x \sin x\, dx$ とおく。

第1回：$\sin x$ を微分する側、$e^x$ を積分する側で部分積分する。
→ $\displaystyle\int e^x \cos x\, dx$ が現れる。

第2回：同じ方針（三角関数を微分する側） でもう一度部分積分する。
→ 何が出てくるか？
本質ポイント $I$ が再登場 → 方程式として解く
🔑 部分積分を2回行うと $$I = e^x \sin x - e^x \cos x - I$$ が得られる。$I$ を含む方程式として解けば： $$2I = e^x(\sin x - \cos x) \;\Rightarrow\; I = \frac{e^x(\sin x - \cos x)}{2} + C$$ 「終わらない」ように見えて、元に戻ることが解法の鍵。
解説 部分積分を2回行う
1 第1回（$\sin x$ を微分）： $$I = e^x \sin x - \int e^x \cos x \, dx$$
2 第2回（$\cos x$ を微分）： $$\int e^x \cos x\, dx = e^x \cos x + \int e^x \sin x\, dx = e^x \cos x + I$$
3 代入：$I = e^x \sin x - (e^x \cos x + I)$ → $2I = e^x(\sin x - \cos x)$
答　え
$$\int e^x \sin x \, dx = \frac{e^x(\sin x - \cos x)}{2} + C$$
一般化 指数関数×三角関数の積分
$\displaystyle\int e^x \sin x\, dx$ や $\displaystyle\int e^x \cos x\, dx$ は
部分積分を同じ方向で2回行うと元の積分が再登場。
「$I = (\text{式}) - I$」→「$2I = (\text{式})$」→「$I = \dfrac{(\text{式})}{2} + C$」

※ 2回目で役割を逆にすると $0 = 0$ になるので注意。
類題 同じ手順で解いてみよう
$\displaystyle\int e^x \cos x \, dx$ を求めよ。
解答： $J = \displaystyle\int e^x \cos x\, dx$ とおく。
同様に2回部分積分すると $2J = e^x(\cos x + \sin x)$： $$J = \frac{e^x(\cos x + \sin x)}{2} + C$$

数学II 微分の最大・最小（閉区間上での端点チェック）
★★★☆☆ 標準偏差値55〜65
問　題
関数 $f(x) = x^3 - 3x^2 \quad (0 \leq x \leq 4)$ の最大値・最小値を求めよ。
💬 増減表を作れば終わり…と思っていませんか？
閉区間では端点の値も必ず確認する必要があります！
出典：教科書例題・数学II（改題）
よくある間違い 端点 $x=4$ の値を確認しない
❌ 間違いの典型パターン
$f'(x) = 3x^2 - 6x = 3x(x-2)$ より $x=0$ で極大、$x=2$ で極小。

❌ 間違い： 「増減表より 最大値 $= f(0) = 0$、最小値 $= f(2) = -4$」と答えてしまう。

→ 端点 $x=4$ を忘れている！
$f(4) = 64 - 48 = \boldsymbol{16}$ — これが本当の最大値！
ポイント：極値 ≠ 最大・最小値。閉区間では端点も候補に入れること。
本質ポイント 閉区間の最大・最小は「極値 ＋ 端点」を比べる
🔑 閉区間 $[a,b]$ での最大・最小の求め方：
  1. $f'(x)=0$ の解（臨界点）を求め、増減表を書く
  2. 極値（臨界点での値）を計算する
  3. 端点 $f(a)$・$f(b)$ も計算する
  4. 極値と端点の値をすべて比べ、最大・最小を決定する
⚠️ 端点は極値にならない（増減表で「極」と書かれない）ため見落としやすい。
必ず別途代入して確認すること！
解説 $f(x) = x^3 - 3x^2$ の最大・最小を正しく求める
1 $f'(x) = 3x^2 - 6x = 3x(x-2)$
$0 \leq x \leq 4$ での増減表：
$x$$0$$\cdots$$2$$\cdots$$4$
$f'(x)$$-$$0$$+$
$f(x)$$0$↘$-4$（極小）↗$16$
2 各候補の値を確認：
$f(0) = 0$，　$f(2) = 8-12 = -4$（極小），　$f(4) = 64-48 = \boldsymbol{16}$（端点）
3 比較すると：
最大値 $\boldsymbol{16}$（$x=4$ のとき）
最小値 $\boldsymbol{-4}$（$x=2$ のとき）
一般化 閉区間上の最大・最小 — 手順まとめ
$$\text{閉区間} [a,b] \text{ での最大・最小} = \max\!\bigl(f(a),\;f(b),\;\text{区間内の極値}\bigr)$$
  • $f'(x)=0$ の解が区間外なら、極値の候補から除外する
  • 区間内の極値がなくても端点の大小で決まる場合がある
  • 「極大 = 最大」は開区間での話。閉区間では必ず端点チェック！
⚠️ 定義域が $[a,b]$ か $(a,b)$ かによって話が変わる。問題文の括弧を確認しよう。
類題 端点も臨界点も多い場合
$g(x) = x^3 - 3x + 2 \quad (-2 \leq x \leq 2)$ の最大値・最小値を求めよ。
答　え
$g'(x) = 3x^2 - 3 = 3(x+1)(x-1)$ より $x=-1$ で極大、$x=1$ で極小。
各値：$g(-2)=0$，$g(-1)=4$（極大），$g(1)=0$（極小），$g(2)=4$（端点）

最大値 $\boldsymbol{4}$（$x=-1$ または $x=2$ のとき）
最小値 $\boldsymbol{0}$（$x=-2$ または $x=1$ のとき）

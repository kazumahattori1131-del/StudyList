数学II 定積分と面積（1/12公式・符号ミス）
★★★☆☆ 標準偏差値55〜65
$y = x^2 - x - 2$ と $x$ 軸で囲まれた図形の面積を求めよ。

「とりあえず積分」は危険！　符号を無視すると面積がマイナスになる。

よくある間違い 符号を無視してそのまま積分する

❌ よくある間違い①：そのまま積分して負の値

$$\int_{-1}^{2}(x^2 - x - 2)\,dx = \left[\frac{x^3}{3} - \frac{x^2}{2} - 2x\right]_{-1}^{2} = -\frac{9}{2}$$

→ 面積は正の値。$x$ 軸の下側にある領域は「マイナスをつける」必要がある！

❌ よくある間違い②：交点を求めずに積分

「$x$ 軸との交点」＝「積分区間の端点」。これを特定しないと正しい区間で積分できない。

→ まず因数分解して交点を求めること。

⚠️ $x$ 軸の下にある領域の面積は、積分値の符号を反転して加える！
本質ポイント 交点を求めて「符号をつけて積分」

$y = x^2 - x - 2 = (x+1)(x-2)$ より $x$ 軸との交点は $x = -1,\; 2$。
$-1 \leq x \leq 2$ では $y \leq 0$（$x$ 軸の下側）なので：

面積 $= -\displaystyle\int_{-1}^{2}(x^2 - x - 2)\,dx$
曲線が $x$ 軸の下にある区間では
面積 $= -\displaystyle\int_{\alpha}^{\beta} f(x)\,dx\;（> 0）$

※ 1/12 公式が使える形 $a(x-\alpha)(x-\beta)$ を確認しよう

解説 2通りの方法で面積を求める
STEP 1因数分解と交点の確認
$y = (x+1)(x-2)$　交点：$x = -1,\; 2$　区間内で $y \leq 0$
STEP 2定積分で計算
$S = -\displaystyle\int_{-1}^{2}(x^2-x-2)\,dx = -\left[\dfrac{x^3}{3} - \dfrac{x^2}{2} - 2x\right]_{-1}^{2}$
$= -\!\left\{\left(\dfrac{8}{3}-2-4\right)-\left(-\dfrac{1}{3}-\dfrac{1}{2}+2\right)\right\} = -\!\left(-\dfrac{9}{2}\right) = \dfrac{9}{2}$
STEP 31/12 公式で検算
$S = \dfrac{|a|}{6}(\beta-\alpha)^3 = \dfrac{1}{6}(2-(-1))^3 = \dfrac{1}{6} \times 27 = \dfrac{9}{2}$ ✓
$S = \dfrac{9}{2}$
一般化 1/12 公式と面積計算のパターン
$y = a(x-\alpha)(x-\beta)$ と $x$ 軸の間の面積：
$$S = \frac{|a|}{6}(\beta - \alpha)^3 \quad (\alpha < \beta)$$

面積計算の手順

  • 因数分解（または解の公式）で $x$ 軸との交点 $\alpha,\, \beta$ を求める
  • 区間内で曲線が $x$ 軸の上・下どちらにあるか確認する
  • 下側なら積分値にマイナスをつけて面積を正にする
  • 1/12 公式で素早く計算・検算する
複数区間にまたがる場合は区間ごとに分けて符号を確認し、それぞれの面積を加算する！
類題 同じ手順で解いてみよう
$y = -x^2 + 4$ と $x$ 軸で囲まれた図形の面積を求めよ。
答　え
$y = -(x+2)(x-2)$ より $x$ 軸との交点は $x = -2,\; 2$。
$-2 \leq x \leq 2$ では $y \geq 0$（$x$ 軸の上側）。

1/12 公式を使う：$a = -1$、$\alpha = -2$、$\beta = 2$
$$S = \frac{|-1|}{6}(2-(-2))^3 = \frac{1}{6} \times 64 = \frac{32}{3}$$ （定積分で確認）
$\displaystyle\int_{-2}^{2}(-x^2+4)\,dx = \left[-\dfrac{x^3}{3}+4x\right]_{-2}^{2} = \left(-\dfrac{8}{3}+8\right)-\left(\dfrac{8}{3}-8\right) = \dfrac{32}{3}$ ✓

$\therefore\; S = \dfrac{32}{3}$

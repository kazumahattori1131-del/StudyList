数学A 条件付き確率（$P(A|B)$ と $P(B|A)$ の違い）
★★★☆☆ 標準偏差値55〜65
問　題
袋の中に赤玉4個・青玉2個がある。1個ずつ順に2個取り出す（非復元）。
2個目が青玉だったとき、1個目が赤玉である確率を求めよ。
💬 「2個目が青という条件のもとで、1個目が赤」 — これは条件付き確率。
条件が「後ろ」に来るのが紛らわしいポイントです！
出典：センター試験・数学A（改題）
よくある間違い $P(A|B)$ と $P(B|A)$ の混同、または単純確率で答える
❌ 間違いの典型パターン
事象 $A$＝「1個目が赤」，事象 $B$＝「2個目が青」とする。

❌ 間違い①： 「1個目が赤の確率は $\dfrac{4}{6}=\dfrac{2}{3}$」 — 条件（2個目が青）を無視している。

❌ 間違い②： 「1個目が赤のとき2個目が青 → $P(B|A)=\dfrac{2}{5}$」 — 条件と求める事象が逆！

✅ 正しくは $P(A|B) = \dfrac{P(A \cap B)}{P(B)}$ を使う。
覚え方：「〜のとき」の後ろが条件 $B$、求めたいのが $A$ → $P(A|B)$
本質ポイント 条件付き確率の定義と計算手順
🔑 条件付き確率の定義： $$P(A|B) = \frac{P(A \cap B)}{P(B)} \quad (P(B) > 0)$$ 計算手順：
  1. $P(A \cap B)$：両方同時に起きる確率を求める
  2. $P(B)$：条件側の確率を求める（余事象・全確率の公式を使う場合も）
  3. 割り算して $P(A|B)$ を求める
解説 条件付き確率を定義にしたがって計算する
1 $P(A \cap B)$：1個目が赤 かつ 2個目が青 $$P(A \cap B) = \frac{4}{6} \times \frac{2}{5} = \frac{8}{30} = \frac{4}{15}$$
2 $P(B)$：2個目が青（全確率の公式） $$P(B) = \underbrace{\frac{4}{6} \times \frac{2}{5}}_{1\text{個目赤}} + \underbrace{\frac{2}{6} \times \frac{1}{5}}_{1\text{個目青}} = \frac{8}{30} + \frac{2}{30} = \frac{10}{30} = \frac{1}{3}$$
3 条件付き確率： $$P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{\dfrac{4}{15}}{\dfrac{1}{3}} = \frac{4}{15} \times 3 = \boldsymbol{\frac{4}{5}}$$
一般化 条件付き確率のまとめ
$$P(A|B) = \frac{P(A \cap B)}{P(B)}, \quad P(B|A) = \frac{P(A \cap B)}{P(A)}$$
  • $P(A|B) \neq P(B|A)$ — 条件と結果を入れ替えると値が変わる
  • $P(B)$ の計算には全確率の公式を使うことが多い
  • 「〜のとき」＝条件側、「〜である」＝求める側 と読み分けること
⚠️ 非復元抽出（引いた玉を戻さない）か復元抽出かで分母が変わることに注意！
類題 条件付き確率の練習
袋の中に赤玉3個・青玉3個がある。1個ずつ順に2個取り出す（非復元）。
2個目が赤玉だったとき、1個目も赤玉である確率を求めよ。
答　え
事象 $A$＝「1個目が赤」，$B$＝「2個目が赤」とする。
$P(A \cap B) = \dfrac{3}{6} \times \dfrac{2}{5} = \dfrac{6}{30} = \dfrac{1}{5}$

$P(B) = \dfrac{3}{6} \times \dfrac{2}{5} + \dfrac{3}{6} \times \dfrac{3}{5} = \dfrac{6}{30} + \dfrac{9}{30} = \dfrac{15}{30} = \dfrac{1}{2}$

$P(A|B) = \dfrac{1/5}{1/2} = \boldsymbol{\dfrac{2}{5}}$

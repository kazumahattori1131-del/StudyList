数学III ∫1/(1+x²) の発展形
★★★★☆ 発展偏差値65〜70
問　題
次の定積分を求めよ。 $$\int_0^1 \frac{1}{(1+x^2)^2}\,dx$$
💬 分母の 2乗 ―― どうやって崩す？
ヒント $x = \tan\theta$ の三角置換
HINT： $x = \tan\theta$ とおくと $$1 + x^2 = 1 + \tan^2\theta = \frac{1}{\cos^2\theta}$$ このとき $dx = \dfrac{d\theta}{\cos^2\theta}$。

被積分関数と $dx$ をまとめて変換するとどうなる？
積分範囲：$x: 0 \to 1$ のとき $\theta: 0 \to \dfrac{\pi}{4}$
本質ポイント 置換後に $\cos^2\theta$ だけが残る
🔑 $x = \tan\theta$ とおくと $(1+x^2)^2 = \dfrac{1}{\cos^4\theta}$、$dx = \dfrac{d\theta}{\cos^2\theta}$

$$\frac{dx}{(1+x^2)^2} = \frac{\frac{1}{\cos^2\theta}\,d\theta}{\frac{1}{\cos^4\theta}} = \cos^2\theta\,d\theta$$ → 半角公式で積分できる形に変換完了！
解説 置換から積分まで
1 $x = \tan\theta$（$0 \leq \theta \leq \dfrac{\pi}{4}$）： $$\int_0^1 \frac{dx}{(1+x^2)^2} = \int_0^{\pi/4}\cos^2\theta\,d\theta$$
2 半角公式 $\cos^2\theta = \dfrac{1+\cos 2\theta}{2}$ を使う： $$= \left[\frac{\theta}{2} + \frac{\sin 2\theta}{4}\right]_0^{\pi/4}$$
3 $\theta = \pi/4$ を代入（$\sin(\pi/2) = 1$）： $$= \frac{\pi}{8} + \frac{1}{4}$$
答　え
$$\int_0^1 \frac{dx}{(1+x^2)^2} = \frac{\pi}{8} + \frac{1}{4}$$
一般化 分母に $(a^2+x^2)^n$ がある積分
$\displaystyle\int \frac{1}{(a^2+x^2)^n}\,dx$ の形は $x = a\tan\theta$ で置換。
$a^2 + x^2 = \dfrac{a^2}{\cos^2\theta}$ となり次数が下がる。

「分母に $(1+x^2)$ → $x = \tan\theta$」 を反射的に出せるようにしよう。
類題 同じ手順で解いてみよう
$\displaystyle\int_0^{\sqrt{3}} \frac{1}{(1+x^2)^2}\,dx$ を求めよ。
解答： $x = \tan\theta$、範囲 $\theta: 0 \to \dfrac{\pi}{3}$： $$\int_0^{\pi/3}\cos^2\theta\,d\theta = \left[\frac{\theta}{2}+\frac{\sin 2\theta}{4}\right]_0^{\pi/3} = \frac{\pi}{6}+\frac{\sqrt{3}}{8}$$
1 / 6 ← → キーでも操作できます

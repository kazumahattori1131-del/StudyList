数学I 絶対値と不等式（場合分けの考え方）
★★☆☆☆ 基本〜標準偏差値50〜60
$|2x - 3| < 5$ を解け。

絶対値を「外す」だけではダメ！ 場合分けの意味を理解すれば一生使える。

よくある間違い 絶対値をそのまま外してしまう

❌ よくある間違い①：絶対値を無視して解く

$$|2x - 3| < 5 \;\overset{?}{\Rightarrow}\; 2x - 3 < 5 \;\Rightarrow\; x < 4$$

→ 負の範囲を捨てている！ $x = -1$ は $|2(-1)-3| = 5$ で満たさないが $x = -10$ は？

❌ よくある間違い②：$2x-3$ の正負で場合分けして連立し忘れる

$2x - 3 \geq 0$ のとき $2x - 3 < 5$ → $x < 4$、として場合分けと連立を忘れ $x < 4$ と答える。

→ 連立条件 $x \geq \dfrac{3}{2}$ を忘れると不正確！

⚠️ $|A| < k$ は「$A$ の数直線上の距離が $k$ 未満」← 公式 $-k < A < k$ を使えば場合分け不要！
本質ポイント $|A| < k \Leftrightarrow -k < A < k$ の図形的意味

絶対値 $|A|$ は「$A$ と $0$ の距離」。$|A| < k$ は「$A$ が $0$ から距離 $k$ 未満にある」ことを意味する。

数直線上のイメージ：
$\longleftarrow \underbrace{\quad\quad\quad}_{-k \text{ から } k \text{ の範囲}} \longrightarrow$
$|A| < k \;\Longleftrightarrow\; -k < A < k$

$|A| \geq k \;\Longleftrightarrow\; A \leq -k \;\text{ または }\; A \geq k$
$k > 0$ のとき：$|A| < k \Leftrightarrow -k < A < k$
この変換で絶対値を一気に外せる！
解説 公式で一発解答
STEP 1公式 $|A| < k \Leftrightarrow -k < A < k$ を適用
$|2x - 3| < 5 \;\Leftrightarrow\; -5 < 2x - 3 < 5$
STEP 2各辺に $3$ を足す
$-5 + 3 < 2x < 5 + 3 \;\Rightarrow\; -2 < 2x < 8$
STEP 3各辺を $2$ で割る
$-1 < x < 4$
答え $-1 < x < 4$
一般化 絶対値不等式の変換パターン
$k > 0$ のとき
  • $|A| < k \;\Longleftrightarrow\; -k < A < k$（絞り込む型）
  • $|A| \leq k \;\Longleftrightarrow\; -k \leq A \leq k$（等号あり）
  • $|A| > k \;\Longleftrightarrow\; A < -k \;\text{ または }\; A > k$（広がる型）
  • $|A| \geq k \;\Longleftrightarrow\; A \leq -k \;\text{ または }\; A \geq k$（等号あり）
不等号の向きで「絞り込むか（and）・広がるか（or）」が決まる！
$<,\leq$ → 連立（and）　　$>,\geq$ → 「または」（or）
類題 同じ手順で解いてみよう
$|x + 1| \geq 3$ を解け。
答　え
$|A| \geq k \Leftrightarrow A \leq -k$ または $A \geq k$ を使う。

$x + 1 \leq -3$ または $x + 1 \geq 3$

$x \leq -4$ または $x \geq 2$

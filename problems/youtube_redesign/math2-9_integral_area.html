落とし穴 積分値が 0 でも面積は 0 じゃない？
❓ 計算したら 0 になった
$$\int_0^3(x^2-2x)\,dx = (9-9)-0 = 0$$
→「面積 $= 0$」と答えてしまう！
✅ 正しい面積は $\dfrac{8}{3}$
積分値 0 は正と負が打ち消し合っただけ！
グラフの赤（$-\tfrac{4}{3}$）と緑（$+\tfrac{4}{3}$）が打ち消し合って積分値は 0 に。
面積は 各領域の大きさを足したもの → $\tfrac{4}{3}+\tfrac{4}{3} = \boldsymbol{\dfrac{8}{3}}$
数学II 積分と面積（符号の落とし穴）
★★★☆☆ 標準偏差値55〜65
問　題
$y = x^2-2x \quad (0 \leq x \leq 3)$ と $x$ 軸で囲まれた図形の面積を求めよ。
💬 グラフを描いてみると…
$f(x) = x(x-2)$ なので
$x=0$, $x=2$ で $x$ 軸と交わる。

$0 \leq x \leq 2$：$f(x) \leq 0$（x 軸より下！）
$2 \leq x \leq 3$：$f(x) \geq 0$
出典：教科書例題・数学II（改題）
よくある間違い 積分値をそのまま面積と思ってしまう
❌ 間違いの典型パターン
$$\int_0^3(x^2-2x)\,dx = \left[\frac{x^3}{3}-x^2\right]_0^3 = (9-9)-0 = 0$$
❌「面積 $= 0$」と答えてしまう
なぜ間違い？
$0 \leq x \leq 2$ ではグラフが $x$ 軸より下にあり、この区間の積分値は負の値になる。
$2 \leq x \leq 3$ の正の部分と打ち消し合って 0 になってしまうのが落とし穴！
⚠️ $\displaystyle\int_a^b f(x)\,dx$ は面積ではない。$f(x) < 0$ の区間では符号が逆になる！
本質ポイント $x$ 軸と囲む面積の正しい求め方
🔑 面積 $S = \displaystyle\int_a^b |f(x)|\,dx$ — 絶対値が鍵！
1 $f(x)=0$ の解（$x$ 軸との交点）を求める
2 各区間で $f(x)$ の正・負を確認する（グラフを描く）
3 区間を分割して積分する
$f(x) \leq 0$ の区間は $-\displaystyle\int f(x)\,dx$（マイナスをかける）
4 すべての区間の面積を足し合わせる
⚠️「全区間で1本の積分でOK」は罠。必ずグラフで正・負を確認！
解説 $y = x^2-2x$ の面積を正しく求める
1 $f(x)=x^2-2x=x(x-2)=0$ より $x=0,\;2$（$x$ 軸との交点）
2 $0 \leq x \leq 2$：$f(x)\leq 0$，　$2 \leq x \leq 3$：$f(x)\geq 0$
3 区間を分割して計算： $$S = -\int_0^2(x^2-2x)\,dx + \int_2^3(x^2-2x)\,dx$$ $$= -\!\left[\frac{x^3}{3}-x^2\right]_0^2 + \left[\frac{x^3}{3}-x^2\right]_2^3$$ $$= -\!\left(\frac{8}{3}-4\right)+\left\{0-\!\left(\frac{8}{3}-4\right)\right\} = \frac{4}{3}+\frac{4}{3} = \boldsymbol{\dfrac{8}{3}}$$
一般化 $x$ 軸との面積 — 手順まとめ
$x$ 軸と $y=f(x)$ で囲まれた面積は、
$f(x)$ の符号で区間を分け、各区間の $|\text{積分値}|$ を足す。
  • $f(x) \geq 0$ の区間：$\displaystyle\int_a^b f(x)\,dx$（そのまま）
  • $f(x) \leq 0$ の区間：$\displaystyle-\!\int_a^b f(x)\,dx$（マイナスをかける）
⚠️ 全区間を1本で積分すると正・負が打ち消し合い、面積が小さくなる（または 0 になる）。
これは入試で最もよくある積分ミスのひとつ！
類題 符号に注意して面積を求める
$y = x^2-x \quad (0 \leq x \leq 2)$ と $x$ 軸で囲まれた図形の面積を求めよ。
答　え
$f(x)=x^2-x=x(x-1)$ より交点：$x=0,\;1$
$0\leq x\leq 1$：$f(x)\leq 0$，　$1\leq x\leq 2$：$f(x)\geq 0$ $$S = -\int_0^1(x^2-x)\,dx + \int_1^2(x^2-x)\,dx$$ $$= -\!\left[\frac{x^3}{3}-\frac{x^2}{2}\right]_0^1+\left[\frac{x^3}{3}-\frac{x^2}{2}\right]_1^2$$ $$= \frac{1}{6}+\frac{5}{6} = \boldsymbol{1}$$
まとめ 今日の一言
🔑 積分値 $\neq$ 面積。
$f(x) < 0$ の区間が含まれたら
区間を分けてマイナスをかける。
グラフを描く習慣が命取りを防ぐ！
⚠️ 全区間を1本で積分すると正・負が打ち消し合い、面積が 0 になる罠にハマる。

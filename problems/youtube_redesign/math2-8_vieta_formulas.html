数学II 解と係数の関係（実際に解かなくて解ける問題）
★★★☆☆ 標準偏差値55〜65
問　題
$x^2 - 3x + 1 = 0$ の 2 解を $\alpha,\ \beta$ とするとき、$\alpha^2 + \beta^2$ の値を求めよ。
💬 解の公式で $\alpha$ と $\beta$ を実際に計算しようとしていませんか？
解と係数の関係を使えば、解を求めずに一行で答えが出ます。
出典：教科書例題・数学II（改題）
よくある間違い 解の公式で実際に $\alpha,\,\beta$ を求めようとする
❌ 間違いの典型パターン
解の公式より $\alpha = \dfrac{3 + \sqrt{5}}{2}$、$\beta = \dfrac{3 - \sqrt{5}}{2}$ を求める。

すると $\alpha^2 = \dfrac{14 + 6\sqrt{5}}{4}$、$\beta^2 = \dfrac{14 - 6\sqrt{5}}{4}$ となり…

→ 計算が複雑になりミスが生じやすい！
実は $\alpha$ と $\beta$ の個々の値は不要。
係数だけから $\alpha + \beta$ と $\alpha\beta$ がわかれば十分です。
本質ポイント 解と係数の関係で $\alpha+\beta$ と $\alpha\beta$ を読む
🔑 $x^2 - 3x + 1 = 0$ の 2 解 $\alpha,\,\beta$ に対して：
$\alpha + \beta = 3$  （$x$ の係数の符号を反転）
$\alpha\beta = 1$   （定数項）
あとは恒等式の変形だけ！
$\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta = 3^2 - 2 \times 1 = \boldsymbol{7}$
解を求めずに係数から直接計算できる。これが解と係数の関係の本質です。
解説 $\alpha^2 + \beta^2$ を解と係数の関係で求める
1 解と係数の関係より：
$\alpha + \beta = 3$、  $\alpha\beta = 1$
2 $\alpha^2 + \beta^2$ を $\alpha+\beta$ と $\alpha\beta$ で表す：
$$\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta$$
3 ①の値を代入する：
$(\alpha + \beta)^2 - 2\alpha\beta = 3^2 - 2 \times 1 = 9 - 2 = \boldsymbol{7}$
ポイント：$\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta$ の変形は必ず覚えること。
一般化 解と係数の関係 — 公式と頻出変形まとめ
$ax^2 + bx + c = 0$ の 2 解 $\alpha,\,\beta$ に対して：
$\alpha + \beta = -\dfrac{b}{a}$、  $\alpha\beta = \dfrac{c}{a}$
頻出の変形パターン：
求めたい式変形のしかた
$\alpha^2 + \beta^2$$(\alpha+\beta)^2 - 2\alpha\beta$
$(\alpha - \beta)^2$$(\alpha+\beta)^2 - 4\alpha\beta$
$\dfrac{1}{\alpha} + \dfrac{1}{\beta}$$\dfrac{\alpha+\beta}{\alpha\beta}$
$\alpha^3 + \beta^3$$(\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta)$
すべて $\alpha+\beta$ と $\alpha\beta$ だけで表せる — これが解と係数の関係の強さ。
類題 解と係数の関係を使って求めよ
問　題
$2x^2 - 5x + 3 = 0$ の 2 解を $\alpha,\ \beta$ とするとき、$\dfrac{1}{\alpha} + \dfrac{1}{\beta}$ の値を求めよ。
解　答
1 解と係数の関係より：$\alpha + \beta = \dfrac{5}{2}$、  $\alpha\beta = \dfrac{3}{2}$
2 $\dfrac{1}{\alpha} + \dfrac{1}{\beta} = \dfrac{\alpha + \beta}{\alpha\beta} = \dfrac{\,\frac{5}{2}\,}{\,\frac{3}{2}\,} = \boldsymbol{\dfrac{5}{3}}$

## スライド①　問題提示

はい、今回の問題はこちらです。
「数列 a の n 番目 が次の条件を満たすとき、一般項を求めよ。a の 1 番目 イコール 1、a の n プラス 1 番目 イコール 2 かける a の n 番目 プラス 2 の n 乗。」
右辺に 2 の n 乗 がある漸化式です。
「特性方程式を使おう」と思うかもしれませんが、右辺の係数と同じ底の指数関数があるとき、別の手が使えます。

---

## スライド②　ヒント

ヒントです。両辺を 2 の n プラス 1 乗 で割るとどうなるか、やってみましょう。
a の n プラス 1 番目 わる 2 の n プラス 1 乗 イコール a の n 番目 わる 2 の n 乗 プラス 2 の n 乗 わる 2 の n プラス 1 乗。
最後の項は 2分の1 に簡単になります。
b の n 番目 イコール a の n 番目 わる 2 の n 乗 とおくと、b の n プラス 1 番目 イコール b の n 番目 プラス 2分の1 という等差数列になります。

---

## スライド③　本質ポイント

右辺の 2 の n 乗 を 2 の n プラス 1 乗 で割ったら定数 2分の1 になる、これが変換の鍵です。
b の n 番目 イコール a の n 番目 わる 2 の n 乗 とおくと、
b の n プラス 1 番目 イコール b の n 番目 プラス 2分の1 という公差 2分の1 の等差数列に帰着できます。

---

## スライド④　解説

では実際に解いていきます。
ステップ1。b の n 番目 イコール a の n 番目 わる 2 の n 乗 とおく。初項 b の 1 番目 イコール 1 わる 2 イコール 2分の1。
公差 2分の1 の等差数列なので、b の n 番目 イコール 2分の1 プラス n マイナス 1 かける 2分の1 イコール n わる 2。
ステップ2。b の n 番目 イコール a の n 番目 わる 2 の n 乗 に戻す。a の n 番目 イコール 2 の n 乗 かける n わる 2。
答えは a の n 番目 イコール n かける 2 の n マイナス 1 乗 です。

---

## スライド⑤　一般化

一般化しておきましょう。
a の n プラス 1 番目 イコール p かける a の n 番目 プラス q かける p の n 乗 の形なら、
両辺を p の n プラス 1 乗 で割ると、b の n 番目 イコール a の n 番目 わる p の n 乗 は公差 q わる p の等差数列になります。
「右辺の底と係数の底が同じ → 両辺を割る」これが鉄則です。

## スライド⑥　類題

では類題を一問。
「a の 1 番目 イコール 2、a の n プラス 1 番目 イコール 3 かける a の n 番目 プラス 3 の n 乗 のとき、a の n 番目 を求めよ。」
b の n 番目 イコール a の n 番目 わる 3 の n 乗 とおく。b の n プラス 1 番目 イコール b の n 番目 プラス 3分の1。初項 b の 1 番目 イコール 3分の2。
b の n 番目 イコール 3分の2 プラス n マイナス 1 かける 3分の1 イコール n プラス 1 わる 3。
答えは a の n 番目 イコール n プラス 1 かける 3 の n マイナス 1 乗 です。

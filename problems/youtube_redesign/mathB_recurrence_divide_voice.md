## スライド①　問題提示

今回の問題です。
「a の 1 番目 イコール 1、a の n プラス 1 番目 イコール 2 かける a の n 番目 プラス 2 の n 乗。一般項を求めよ。」

右辺に 2 の n 乗 がある漸化式です。
「特性方程式で処理しよう」と思うと、少し詰まります。
実は、もっとシンプルな手があります。

---

## スライド②　ヒント

試しに、両辺を 2 の n プラス 1 乗 で割ってみます。
a の n プラス 1 番目 わる 2 の n プラス 1 乗 イコール a の n 番目 わる 2 の n 乗 プラス 2 の n 乗 わる 2 の n プラス 1 乗。

あ、最後の項が 2分の1 になりました。
b の n 番目 イコール a の n 番目 わる 2 の n 乗 とおくと、
b の n プラス 1 番目 イコール b の n 番目 プラス 2分の1。

これ、等差数列ですね。

---

## スライド③　本質ポイント

ここがポイントです。
右辺の 2 の n 乗 を 2 の n プラス 1 乗 で割ると定数 2分の1 になる。
だから等差数列に帰着できる。

「右辺の底と係数の底が同じ → 両辺をその底で割る」
この発想、押さえておいてください。

---

## スライド④　解説

解いていきます。

ステップ1。b の n 番目 イコール a の n 番目 わる 2 の n 乗 とおく。
初項 b の 1 番目 イコール 2分の1、公差 2分の1 の等差数列。
b の n 番目 イコール 2分の1 プラス n マイナス 1 かける 2分の1 イコール n わる 2。

ステップ2。a の n 番目 に戻します。
a の n 番目 イコール 2 の n 乗 かける n わる 2。

答えは a の n 番目 イコール n かける 2 の n マイナス 1 乗。

---

## スライド⑤　一般化

a の n プラス 1 番目 イコール p かける a の n 番目 プラス q かける p の n 乗 の形は、
両辺を p の n プラス 1 乗 で割ると公差 q わる p の等差数列になります。

「右辺の底と係数の底が同じ → 両辺を割る」これが鉄則です。

---

## スライド⑥　類題

では一問。
「a の 1 番目 イコール 2、a の n プラス 1 番目 イコール 3 かける a の n 番目 プラス 3 の n 乗 のとき、a の n 番目 を求めよ。」

b の n 番目 イコール a の n 番目 わる 3 の n 乗 とおく。
b の n プラス 1 番目 イコール b の n 番目 プラス 3分の1、初項 b の 1 番目 イコール 3分の2。
b の n 番目 イコール n プラス 1 わる 3。
答えは a の n 番目 イコール n プラス 1 かける 3 の n マイナス 1 乗。

自分でも確認してみてください。

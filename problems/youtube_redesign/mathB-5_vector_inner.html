数学B ベクトルの内積と大きさ
★★★☆☆ 標準偏差値55〜65
$|\vec{a}|=2$、$|\vec{b}|=3$、$\vec{a}\cdot\vec{b}=3$ のとき、次を求めよ。

(1)　$|2\vec{a}-\vec{b}|$
(2)　$\vec{a}$ と $\vec{b}$ のなす角 $\theta$

$|2\vec{a}-\vec{b}|$ をそのまま引き算しようとしていませんか？

よくある間違い 大きさの差として計算してしまう

❌ よくある間違い①：大きさを直接引き算する

$$|2\vec{a}-\vec{b}| \;\overset{?}{=}\; 2|\vec{a}|-|\vec{b}| = 2\cdot 2 - 3 = 1$$

→ ベクトルの大きさは引き算できない！　$|{\vec{u}}-{\vec{v}}| \neq |\vec{u}|-|\vec{v}|$

❌ よくある間違い②：展開で内積の係数を間違える

$$|2\vec{a}-\vec{b}|^2 \;\overset{?}{=}\; 4|\vec{a}|^2 \underbrace{- 2}_{\text{係数を }-2\text{ にする}}\vec{a}\cdot\vec{b} + |\vec{b}|^2$$

→ $(2\vec{a})\cdot\vec{b}$ の部分が $2(\vec{a}\cdot\vec{b})$ なので係数は $-4$（$2 \times (-2)$ ではない）

⚠️ ベクトルの大きさを求めるには必ず「2乗して内積展開」が基本！
本質ポイント 「2乗→内積展開」の公式を使う

ベクトルの大きさは直接計算できない。必ず 2乗して内積に変換 する：

$|\vec{p}|^2 = \vec{p}\cdot\vec{p}$
$|2\vec{a}-\vec{b}|^2 = (2\vec{a}-\vec{b})\cdot(2\vec{a}-\vec{b})$
$\phantom{|2\vec{a}-\vec{b}|^2} = 4\vec{a}\cdot\vec{a} - 2\cdot 2(\vec{a}\cdot\vec{b}) + \vec{b}\cdot\vec{b}$
$\phantom{|2\vec{a}-\vec{b}|^2} = 4|\vec{a}|^2 - 4\vec{a}\cdot\vec{b} + |\vec{b}|^2$
展開の係数：$(2\vec{a}-\vec{b})^2$ の中間項は $2\cdot 2\vec{a}\cdot(-\vec{b}) = -4\vec{a}\cdot\vec{b}$
解説 2問とも完全解答
(1)$|2\vec{a}-\vec{b}|$ を求める
$|2\vec{a}-\vec{b}|^2 = 4|\vec{a}|^2 - 4\vec{a}\cdot\vec{b} + |\vec{b}|^2$
$\phantom{|2\vec{a}-\vec{b}|^2} = 4\cdot 4 - 4\cdot 3 + 9 = 16 - 12 + 9 = 13$
$\therefore\; |2\vec{a}-\vec{b}| = \sqrt{13}$
(2)なす角 $\theta$ を求める
内積の定義：$\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\theta$ より
$\cos\theta = \dfrac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|} = \dfrac{3}{2\cdot 3} = \dfrac{3}{6} = \dfrac{1}{2}$
$\therefore\; \theta = \dfrac{\pi}{3}$（60°）　$\bigl(0 \leq \theta \leq \pi\bigr)$
一般化 内積計算の2大公式
$|\vec{p}\pm\vec{q}|^2 = |\vec{p}|^2 \pm 2\vec{p}\cdot\vec{q} + |\vec{q}|^2$
$\cos\theta = \dfrac{\vec{p}\cdot\vec{q}}{|\vec{p}||\vec{q}|} \quad (0 \leq \theta \leq \pi)$

使い方の鉄則

  • $|\vec{p}|$（大きさ）を求めたい → まず $|\vec{p}|^2$ を計算し、内積展開する
  • なす角を求めたい → $\cos\theta = \dfrac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|}$ の公式一択
  • $(a\vec{p}+b\vec{q})$ 型の展開：中間項は $2ab(\vec{p}\cdot\vec{q})$、係数のミスに注意
類題 同じ手順で解いてみよう
$|\vec{a}|=\sqrt{3}$、$|\vec{b}|=2$、$\vec{a}$ と $\vec{b}$ のなす角が $\dfrac{\pi}{6}$ のとき、$|\vec{a}+\vec{b}|$ を求めよ。
答　え
まず内積を求める：
$\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\dfrac{\pi}{6} = \sqrt{3}\cdot 2\cdot\dfrac{\sqrt{3}}{2} = 3$

次に2乗して展開：
$|\vec{a}+\vec{b}|^2 = |\vec{a}|^2 + 2\vec{a}\cdot\vec{b} + |\vec{b}|^2 = 3 + 2\cdot 3 + 4 = 13$

$\therefore\; |\vec{a}+\vec{b}| = \sqrt{13}$

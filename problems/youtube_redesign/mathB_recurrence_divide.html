数学B 右辺に $2^n$ がある漸化式
★★★☆☆ 標準+偏差値60〜70
問　題
数列 $\{a_n\}$ が次の条件を満たすとき、一般項 $a_n$ を求めよ。 $$a_1 = 1, \quad a_{n+1} = 2a_n + 2^n \quad (n \geq 1)$$
💬 右辺の $2^n$、特性方程式で処理しようとすると詰まる…別の手がある。
ヒント 両辺を $2^{n+1}$ で割るとどうなる？
HINT： $a_{n+1} = 2a_n + 2^n$ の両辺を $2^{n+1}$ で割ってみよう。

$$\frac{a_{n+1}}{2^{n+1}} = \frac{a_n}{2^n} + \frac{2^n}{2^{n+1}} = \frac{a_n}{2^n} + \frac{1}{2}$$
$b_n = \dfrac{a_n}{2^n}$ とおくと、$b_{n+1}$ と $b_n$ の関係は？

※ $b_1 = \dfrac{a_1}{2^1}$（$a_1$ をそのまま代入しないこと！）
本質ポイント $b_n = a_n/2^n$ とおくと等差数列に化ける
🔑 $b_n = \dfrac{a_n}{2^n}$ とおくと $$b_{n+1} = b_n + \frac{1}{2}$$ → 公差 $\dfrac{1}{2}$ の等差数列！

右辺の $2^n$ を $2^{n+1}$ で割ると $\dfrac{1}{2}$（定数）になるのが変換の鍵。
解説 等差数列として解く
1 $b_n = \dfrac{a_n}{2^n}$ とおく。初項 $b_1 = \dfrac{1}{2}$、公差 $\dfrac{1}{2}$ の等差数列： $$b_n = \frac{1}{2} + (n-1)\cdot\frac{1}{2} = \frac{n}{2}$$
2 $b_n = \dfrac{a_n}{2^n}$ を戻す：$a_n = 2^n \cdot \dfrac{n}{2}$
答　え
$$a_n = n \cdot 2^{n-1}$$
一般化 $a_{n+1} = p\,a_n + q\cdot p^n$ の型
$a_{n+1} = p\,a_n + q\cdot p^n$（右辺の底 $= p$）の形：
両辺を $p^{n+1}$ で割ると $b_n = \dfrac{a_n}{p^n}$ は公差 $\dfrac{q}{p}$ の等差数列。

「右辺の底と係数の底が同じ → 両辺を割る」
類題 同じ手順で解いてみよう
$a_1 = 2$、$a_{n+1} = 3a_n + 3^n$ のとき、$a_n$ を求めよ。
解答： $b_n = \dfrac{a_n}{3^n}$：$b_{n+1} = b_n + \dfrac{1}{3}$、$b_1 = \dfrac{2}{3}$
$b_n = \dfrac{2}{3} + (n-1)\dfrac{1}{3} = \dfrac{n+1}{3}$ $$a_n = (n+1)\cdot 3^{n-1}$$

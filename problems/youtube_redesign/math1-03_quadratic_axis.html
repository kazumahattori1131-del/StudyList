数学I 2次関数の最大・最小（定義域が動く場合分け）
★★★☆☆ 標準偏差値55〜65
$f(x) = x^2 - 2x$ の $a \leq x \leq a+2$ における最小値を求めよ。

定義域が文字 $a$ を含む…。軸との位置関係で場合分けが必要！

よくある間違い 場合分けせず「軸の値が最小」と答える

❌ よくある間違い①：場合分けをしない

$$f(x) = (x-1)^2 - 1 \;\Rightarrow\; \text{軸は } x=1 \;\Rightarrow\; \text{最小値は } f(1) = -1$$

→ 軸 $x=1$ が区間 $[a,\, a+2]$ の外にある場合は成り立たない！

❌ よくある間違い②：場合分けの境界を誤る

「軸が区間内」の条件を $a \leq 1$ だけ（つまり $a < 1$）で終わらせ、
右端の条件 $1 \leq a+2$ を見落とす。

→ 正しくは $a \leq 1$ かつ $1 \leq a+2$、つまり $-1 \leq a \leq 1$。

⚠️ 定義域が文字を含む場合は、軸と定義域の位置関係で必ず3通り場合分け！
本質ポイント 軸と区間の位置関係で3通りに分ける

まず $f(x) = (x-1)^2 - 1$ と平方完成。軸は $x = 1$。
定義域 $[a,\, a+2]$ に対して軸の位置を3通りに分類する：

場合 1
軸が区間の左外
$a + 2 < 1$
$(a < -1)$
場合 2
軸が区間の内側
$a \leq 1 \leq a+2$
$(-1 \leq a \leq 1)$
場合 3
軸が区間の右外
$1 < a$
$(a > 1)$
上に凸でないので「軸が区間内 → 最小」。
軸が外なら、軸に近い端点が最小になる！
解説 3つの場合ごとに最小値を求める
場合 1$a < -1$：区間全体で単調減少 → 右端 $x = a+2$ が最小
$f(a+2) = (a+2-1)^2 - 1 = (a+1)^2 - 1$
場合 2$-1 \leq a \leq 1$：軸 $x=1$ が区間内 → 頂点が最小
$f(1) = (1-1)^2 - 1 = -1$
場合 3$a > 1$：区間全体で単調増加 → 左端 $x = a$ が最小
$f(a) = (a-1)^2 - 1$
$$\text{最小値} = \begin{cases} (a+1)^2 - 1 & (a < -1) \\ -1 & (-1 \leq a \leq 1) \\ (a-1)^2 - 1 & (a > 1) \end{cases}$$
一般化 「定義域が動く」問題の解き方パターン
下に凸の $f(x)$、定義域 $[a,\, a+L]$（$L > 0$）、軸 $x = p$

手順

  • 平方完成して軸 $x=p$ を求める
  • 「軸が区間の左外・内側・右外」の3通りに場合分け
  • 左外：右端 $f(a+L)$ が最小　／　内側：頂点 $f(p)$ が最小　／　右外：左端 $f(a)$ が最小
  • 各境界条件（$a+L < p$, $a \leq p \leq a+L$, $p < a$）を $a$ の不等式に直す
最大値を求めるとき（下に凸）は「軸から遠い端点」が最大 → 左右どちらが遠いか比較も必要！
類題 同じ手順で解いてみよう
$g(x) = x^2 - 4x + 3$ の $a \leq x \leq a+2$ における最小値を求めよ。
答　え
$g(x) = (x-2)^2 - 1$ より軸 $x = 2$。
場合 1　$a < 0$：右端 $g(a+2) = a^2 - 1$
場合 2　$0 \leq a \leq 2$：頂点 $g(2) = -1$
場合 3　$a > 2$：左端 $g(a) = (a-2)^2 - 1$

$\therefore\;\text{最小値} = \begin{cases} a^2 - 1 & (a < 0) \\ -1 & (0 \leq a \leq 2) \\ (a-2)^2 - 1 & (a > 2) \end{cases}$

## スライド①　フック

増減表を書いて「最大値はゼロ」と答えたことはありませんか？
実は、この答えは間違いです。
本当の最大値は 16 で、端点の値を確認すれば一発でわかります。
なぜ端点を見なければならないのか、今から説明します。

## スライド②　問題提示

今回の問題です。
f カッコ x カッコ閉じ イコール x の3乗 マイナス 3 かける x の2乗、定義域は 0以上4以下 の閉区間、での最大値・最小値を求める問題です。

微分して増減表を作れば終わり、と思っていませんか？
閉区間の最大・最小は、増減表だけでは不十分です。
端点の値を必ず確認しなければならない、というのが今回の落とし穴です。

---

## スライド③　よくある間違い

まず最初に、典型的な間違いを確認しておきます。

f'カッコ x カッコ閉じ イコール 3x の2乗 マイナス 6x イコール 3x かける カッコ x マイナス 2 カッコ閉じ
となりますから、x イコール 0 で極大、x イコール 2 で極小です。

ここで増減表を見ただけで
「最大値 イコール f カッコ 0 カッコ閉じ イコール 0」
「最小値 イコール f カッコ 2 カッコ閉じ イコール マイナス4」
と答えてしまう人がいます。

ところが、定義域の右端 x イコール 4 の値を確認してみると、
f カッコ 4 カッコ閉じ イコール 64 マイナス 48 イコール 16 となります。
これが本当の最大値です。端点の確認を忘れていました。

覚え方は「極値は最大・最小の候補の一つに過ぎない。閉区間では端点も候補に入れる」です。

---

## スライド④　本質ポイント

整理します。

閉区間 a以上b以下 での最大・最小の求め方は4ステップです。

ステップ1。f'カッコ x カッコ閉じ イコール 0 となる x の値を求め、増減表を書きます。

ステップ2。増減表から読み取れる極値を計算します。

ステップ3。端点 f カッコ a カッコ閉じ と f カッコ b カッコ閉じ も計算します。
ここが見落としやすいポイントです。

ステップ4。極値と端点の値をすべて比べて、最大値・最小値を決定します。

端点は増減表の「極」欄に書かれないので見落としやすい。必ず別途代入して確認してください。

---

## スライド⑤　解説

それでは正しく解いていきます。

ステップ1。
f'カッコ x カッコ閉じ イコール 3x の2乗 マイナス 6x イコール 3x かける カッコ x マイナス 2 カッコ閉じ です。
0以上4以下 での増減表を書くと、x イコール 0 から x イコール 2 まで減少、x イコール 2 から x イコール 4 まで増加となります。

ステップ2。
増減表から極値を確認します。
f カッコ 0 カッコ閉じ イコール 0、f カッコ 2 カッコ閉じ イコール 8 マイナス 12 イコール マイナス4 で極小となります。

ステップ3。端点の値を確認します。ここが落とし穴です。
端点 f カッコ 4 カッコ閉じ イコール 64 マイナス 48 イコール 16 です。

ステップ4。
0、マイナス4、16 を比べると、
最大値は 16 で x イコール 4 のとき、
最小値は マイナス4 で x イコール 2 のとき、です。

## スライド⑥　一般化

まとめます。

閉区間 a以上b以下 での最大・最小は、極値と端点の値すべての中から最大・最小を選びます。

注意点が3つあります。

1つ目。f'カッコ x カッコ閉じ イコール 0 の解が区間の外にある場合は、極値の候補から除外します。

2つ目。区間内に極値がなくても、端点の大小だけで最大・最小が決まる場合があります。

3つ目。「極大 イコール 最大」という等式が成り立つのは開区間での話です。閉区間では必ず端点をチェックしてください。

また、問題文の定義域が 角かっこ、つまり閉区間か、丸かっこつまり開区間かによって扱いが変わります。問題文をよく確認しましょう。

## スライド⑦　類題

では類題を一問。
g カッコ x カッコ閉じ イコール x の3乗 マイナス 3x プラス 2、定義域は マイナス2以上2以下 の閉区間、の最大値・最小値を求めてください。
今度は極値を与える x が2つあります。端点の値と合わせてすべて比べてみてください。

g' カッコ x カッコ閉じ イコール 3x の2乗 マイナス 3 イコール 3 かける カッコ x プラス 1 カッコ閉じ かける カッコ x マイナス 1 カッコ閉じ より、
x イコール マイナス1 で極大、x イコール 1 で極小です。

各値を計算すると、
g カッコ マイナス2 カッコ閉じ イコール 0、
g カッコ マイナス1 カッコ閉じ イコール 4 で極大、
g カッコ 1 カッコ閉じ イコール 0 で極小、
g カッコ 2 カッコ閉じ イコール 4 が端点です。

最大値は 4 で x イコール マイナス1 または x イコール 2 のとき、
最小値は 0 で x イコール マイナス2 または x イコール 1 のときです。

## スライド⑧　まとめ

最後に今日の一言まとめです。
閉区間での最大・最小は、増減表と端点の代入がセット。
この2つをセットで確認することが、正解への道です。
増減表だけで終わると、端点が最大・最小になるケースで必ず失点します。ここだけは絶対に忘れないでください。

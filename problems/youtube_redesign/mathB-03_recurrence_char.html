数学B 漸化式の一般項（特性方程式）
★★★☆☆ 標準偏差値55〜65
数列 $\{a_n\}$ が以下を満たすとき、一般項 $a_n$ を求めよ。

$$a_{n+1} = 3a_n - 4, \quad a_1 = 3$$

等差でも等比でもない… どうやって解く？

よくある間違い 等差・等比の公式を無理やり使おうとする

❌ よくある発想①：隣の差を取る

$$a_{n+1} - a_n = 2a_n - 4$$

→ 右辺に $a_n$ が残る → 等差数列にはならない！

❌ よくある発想②：比を取る

$$\frac{a_{n+1}}{a_n} = 3 - \frac{4}{a_n}$$

→ $a_n$ に依存する → 等比数列にもならない！

⚠️ この形はそのままでは等差・等比のどちらでもない
本質ポイント 特性方程式で「定数項を消す」

数列が定数 $\alpha$ に収束するなら $a_{n+1} = a_n = \alpha$ のはず。これを利用する：

特性方程式
$\alpha = 3\alpha - 4 \;\Rightarrow\; \alpha = 2$
$b_n = a_n - 2$ と置くと定数項が消えて等比数列に変換できる！

※ 京大生はこの形を見た瞬間に「特性方程式を立てる」と判断します

解説 4ステップで完全解答
STEP 1特性方程式を解く
$\alpha = 3\alpha - 4 \;\Rightarrow\; \alpha = 2$
STEP 2$b_n = a_n - 2$ とおいて漸化式を変形
$b_{n+1} = a_{n+1} - 2 = (3a_n - 4) - 2 = 3a_n - 6 = 3(a_n - 2) = 3b_n$
STEP 3$\{b_n\}$ は公比 $3$ の等比数列
$b_1 = a_1 - 2 = 1 \;\Rightarrow\; b_n = 3^{n-1}$
STEP 4$a_n$ に戻す： $a_n = 3^{n-1} + 2$
一般化 1次変換型漸化式のパターン
$a_{n+1} = p\,a_n + q \quad (p \neq 1,\ q \neq 0)$

解法の流れ

  • 特性方程式 $\alpha = p\alpha + q$ を解いて $\alpha = \dfrac{q}{1-p}$ を求める
  • $b_n = a_n - \alpha$ とおくと $b_{n+1} = p\,b_n$（等比数列になる！）
  • $b_n = b_1 \cdot p^{n-1}$ から $a_n = b_n + \alpha$ で一般項を完成
「特性方程式」→「置き換え」→「等比数列」の3ステップが鉄板！
類題 同じ手順で解いてみよう
数列 $\{a_n\}$ が $a_{n+1} = 2a_n + 3,\; a_1 = 1$ を満たすとき、一般項 $a_n$ を求めよ。
答　え
特性方程式：$\alpha = 2\alpha + 3 \;\Rightarrow\; \alpha = -3$
$b_n = a_n - \alpha = a_n - (-3) = a_n + 3$ とおくと
$b_{n+1} = a_{n+1} + 3 = (2a_n + 3) + 3 = 2a_n + 6 = 2(a_n + 3) = 2b_n$

$b_1 = 1 + 3 = 4 \;\Rightarrow\; b_n = 4 \cdot 2^{n-1} = 2^{n+1}$

$\therefore\; a_n = 2^{n+1} - 3$

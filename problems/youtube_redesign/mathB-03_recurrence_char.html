この漸化式、どう解きますか？
$a_{n+1} = 3a_n - 4,\quad a_1 = 3$ のとき一般項 $a_n$ を求めよ。
❌ よくある誤答
差を取る → $a_{n+1}-a_n = 2a_n-4$
等差数列にならない！
比を取る → $\dfrac{a_{n+1}}{a_n} = 3-\dfrac{4}{a_n}$
等比数列にもならない！
手が止まってそのまま0点…という典型パターン
✅ 正しい答え
$a_n = 3^{n-1} + 2$
特性方程式で等比数列に変換すれば解ける！
🔑 $a_{n+1} = pa_n + q$ の形 → 特性方程式が鉄板。
$\alpha = p\alpha + q$ を解いて $b_n = a_n - \alpha$ と置く。これが攻略の入り口。

問　題
数列 $\{a_n\}$ が $a_{n+1} = 3a_n - 4,\; a_1 = 3$ を満たすとき、一般項 $a_n$ を求めよ。
💬 最初の4項を確認すると…
$a_1=3,\;a_2=5,\;a_3=11,\;a_4=29$

差： $2,\;6,\;18,\;\ldots$（一定でない → 等差でない）
比： $\frac{5}{3},\;\frac{11}{5},\;\frac{29}{11},\;\ldots$（一定でない → 等比でない）
⚠️ 等差でも等比でもない形。特性方程式を使った変換が必要。
出典：教科書例題・数学B（改題）
よくある間違い 「等差・等比で解けるはず」という思い込み
「真面目に勉強した人」ほどハマる間違いを確認します。
❌ 間違った解法①：差を取る
$a_{n+1} - a_n = (3a_n - 4) - a_n = 2a_n - 4$
→ 右辺に $a_n$ が残る ＝ 等差数列にはならない！
❌ 間違った解法②：比を取る
$\dfrac{a_{n+1}}{a_n} = \dfrac{3a_n - 4}{a_n} = 3 - \dfrac{4}{a_n}$
→ $a_n$ に依存する ＝ 等比数列にもならない！
⛔ 原因：教科書の等差・等比を"そのまま当てはめようとする"ことが落とし穴。
この形はそのままでは等差・等比のどちらでもない
等差数列の条件
$a_{n+1} - a_n = $ 定数
右辺が $a_n$ を含まないこと
この問題
$a_{n+1} - a_n = 2a_n - 4$
右辺に $a_n$ → 成立しない！
本質ポイント 特性方程式で等比数列に変換する4ステップ
✅ この手順を身につければ、$a_{n+1}=pa_n+q$ 型をすべて同じ手順で解ける！
1特性方程式を立てる（$a_{n+1}=a_n=\alpha$ とおいて $\alpha$ を求める）
2$b_n = a_n - \alpha$ と置いて漸化式を変形する（定数項が消えて等比型になる）
3$\{b_n\}$ の一般項を求める（公比 $p$ の等比数列として処理）
4$a_n = b_n + \alpha$ として元に戻す（これが最終答え）
💡 特性方程式を立てた瞬間に「あとは等比数列」と確信できる。試験で10秒の差がつく思考法。
解説 $a_{n+1} = 3a_n - 4$ を正しく解く
1 特性方程式を立てる：$\alpha = 3\alpha - 4 \;\Rightarrow\; \alpha = 2$
2 $b_n = a_n - 2$ と置いて変形する：
　$b_{n+1} = a_{n+1} - 2 = (3a_n - 4) - 2 = 3a_n - 6 = 3(a_n - 2) = 3b_n$
　$\Rightarrow \{b_n\}$ は公比 $3$ の等比数列！
3 $\{b_n\}$ の一般項：$b_1 = a_1 - 2 = 1 \;\Rightarrow\; b_n = 1 \cdot 3^{n-1} = 3^{n-1}$
4 $a_n$ に戻す：$a_n = b_n + 2$ $$\boxed{a_n = 3^{n-1} + 2}$$
一般化 1次変換型漸化式の鉄則まとめ
📌 $a_{n+1} = p\,a_n + q$（$p \neq 1,\; q \neq 0$）の鉄板3ステップ：
STEP 1：特性方程式
$\alpha = p\alpha + q$
$\Rightarrow \alpha = \dfrac{q}{1-p}$
STEP 2：置き換え
$b_n = a_n - \alpha$ とおく
$\Rightarrow b_{n+1} = p\,b_n$
試験で差がつくポイント
① $p = 1$ のときは等差型（特性方程式ではなく $b_n = a_{n+1} - a_n$ を計算）
② $\alpha$ の符号を間違えない（$b_n = a_n - \alpha$ で必ず「引く」こと）
⏱️ 特性方程式を立てた瞬間に等比数列と確信できれば、この型で失点しない。
類題 同じ3ステップで解いてみよう
数列 $\{a_n\}$ が $a_{n+1} = 2a_n + 3,\; a_1 = 1$ を満たすとき、一般項 $a_n$ を求めよ。
1特性方程式：$\alpha = 2\alpha + 3 \;\Rightarrow\; \alpha = -3$
2$b_n = a_n + 3$ とおくと：
　$b_{n+1} = a_{n+1}+3 = (2a_n+3)+3 = 2(a_n+3) = 2b_n$
3$b_1 = 1+3 = 4$ なので $b_n = 4\cdot 2^{n-1} = 2^{n+1}$
4 $a_n = b_n - 3$：$$\boxed{a_n = 2^{n+1} - 3}$$
まとめ 今日の一言
🔑 $a_{n+1}=pa_n+q$ を見たら、最初にやること：特性方程式を立てる。
$\alpha = p\alpha + q$ を解いて $b_n = a_n - \alpha$ と置く。
すると $b_{n+1} = p\,b_n$（等比数列）に変わる。
特性方程式 → 置き換え → 等比数列の3ステップを習慣に。
⚠️ $p=1$ のときは特性方程式が使えない。その場合は両辺を $a_n$ で割るか階差数列を使う。

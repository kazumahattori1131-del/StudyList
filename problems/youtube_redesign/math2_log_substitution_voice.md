## スライド①　問題提示

今回の問題です。
「x を底とする 2 の対数 プラス 2 を底とする x の対数 イコール 3。ただし x が 0より大きく x が 1でない。」

見てみると、底と真数が入れ替わっています。
「x を底とする 2 の対数」と「2 を底とする x の対数」。

この2つの間に、ある関係があります。
そこに気づくかどうかが、この問題のカギです。

---

## スライド②　よくある間違い

底の変換公式を使います。
log a b イコール log c b わる log c a。
変換先の底が分母にくる、という公式です。

これを使うと、x を底とする 2 の対数 イコール 2 を底とする 2 の対数 わる 2 を底とする x の対数。
2 を底とする 2 の対数 は 1 なので、結局 1 わる 2 を底とする x の対数 になります。

ここで間違えやすいのが、分子と分母が逆になるケースです。
変換先の底が分母、この原則だけ確認してください。

---

## スライド③　本質ポイント

t イコール 2 を底とする x の対数 とおきます。
x が 1でない ことから、t は 0でない。

x を底とする 2 の対数 は 1 わる t になりますから、
式全体は 1 わる t プラス t イコール 3 という形になります。

なるほど、こう繋がるんですね。
t の方程式になった。あとは解くだけです。

---

## スライド④　解説

解いていきます。

ステップ1。1 わる t プラス t イコール 3。

ステップ2。両辺に t をかけます。t が 0でない ので問題ありません。
tの2乗 マイナス 3t プラス 1 イコール 0。
解の公式より t イコール 3 プラスマイナス ルート5 のすべてを 2 で割ったもの。

ステップ3。両方とも t が 0でない を満たします。
x イコール 2 の t 乗 に戻して、
答えは x イコール 2 の 3プラスルート5 を2で割った乗 および x イコール 2 の 3マイナスルート5 を2で割った乗。

## スライド⑤　一般化

整理します。
x を底とする a の対数 プラス a を底とする x の対数 イコール k の形は、
t イコール a を底とする x の対数 と置くだけで tの2乗 マイナス kt プラス 1 イコール 0 に変わります。

実数解が存在する条件は k が マイナス2 以下 または k が 2 以上。
これも頭に入れておくと便利です。

## スライド⑥　類題

では一問。
「x を底とする 3 の対数 プラス 3 を底とする x の対数 イコール 2。x が 0より大きく x が 1でない。」

t イコール 3 を底とする x の対数 とおくと tの2乗 マイナス 2t プラス 1 イコール 0。
カッコ t マイナス 1 カッコ の2乗 イコール 0、つまり t イコール 1。
3 を底とする x の対数 イコール 1 より x イコール 3。

きれいに解けましたね。自分でも確認してみてください。

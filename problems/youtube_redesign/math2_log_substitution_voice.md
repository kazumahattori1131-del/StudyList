## スライド①　問題提示

はい、今回の問題はこちらです。
「次の方程式を解け。ただし x が 0より大きく、x が 1でないとする。x を底とする 2 の対数 プラス 2 を底とする x の対数 イコール 3。」
底と真数が入れ替わった2つの対数が並んでいますね。
このまま計算しようとすると、どこから手をつければいいか迷います。
「x を底とする 2 の対数」と「2 を底とする x の対数」、この2つの間にある関係に気づけるかがポイントです。

---

## スライド②　よくある間違い

ここで底の変換公式を使います。
底の変換公式、log a b イコール log c b わる log c a。変換先の底が分母にきます。
これを使うと、x を底とする 2 の対数 イコール 2 を底とする 2 の対数 わる 2 を底とする x の対数。
2 を底とする 2 の対数 は 1 なので、x を底とする 2 の対数 イコール 2 を底とする x の対数 の逆数、つまり 1 わる 2 を底とする x の対数 です。
よくある間違いとして、分子と分母が逆になるケースがあります。変換先の底が分母、この原則を必ず確認してください。

---

## スライド③　本質ポイント

核心はここです。t イコール 2 を底とする x の対数 とおくと、x が 1でない ことから t が 0でない。
x を底とする 2 の対数 は 1 わる t になります。
式全体は 1 わる t プラス t イコール 3 という t の方程式に変わります。
t の2次方程式に帰着できるんです。

---

## スライド④　解説

では実際に解いていきます。
ステップ1。t イコール 2 を底とする x の対数、t が 0でない とおく。1 わる t プラス t イコール 3。
ステップ2。両辺に t をかけます。t が 0でない ので問題ありません。tの2乗 マイナス 3t プラス 1 イコール 0。
解の公式より t イコール 3 プラスマイナス ルート5 のすべてを 2 で割ったもの。
ステップ3。両方とも t が 0でない を満たします。x イコール 2 の t 乗 より戻すと、
答えは x イコール 2 の 3プラスルート5 を2で割った乗 および x イコール 2 の 3マイナスルート5 を2で割った乗 です。

## スライド⑤　一般化

この手法を一般化しておきましょう。
x を底とする a の対数 プラス a を底とする x の対数 イコール k の形は、t イコール a を底とする x の対数 とおくと、
1 わる t プラス t イコール k、整理すると tの2乗 マイナス kt プラス 1 イコール 0 に帰着できます。
実数解を持つ条件は k が マイナス2 以下 または k が 2 以上 です。

## スライド⑥　類題

では類題を一問。
「x を底とする 3 の対数 プラス 3 を底とする x の対数 イコール 2 を解け。x が 0より大きく x が 1でない。」
t イコール 3 を底とする x の対数 とおくと tの2乗 マイナス 2t プラス 1 イコール 0。
カッコ t マイナス 1 カッコ の2乗 イコール 0 なので t イコール 1。
3 を底とする x の対数 イコール 1 より x イコール 3 です。

数学II 底が変数の対数方程式
★★★☆☆ 標準+偏差値55〜70
問　題
次の方程式を解け。ただし $x > 0$、$x \neq 1$ とする。 $$\log_x 2 + \log_2 x = 3$$
💬 $\log_x 2$ と $\log_2 x$ ―― 底と真数が入れ替わっている。この関係に気づけるか？
よくある間違い 底の変換の方向を間違える
❌ 間違い変換例
$\log_x 2 = \dfrac{\log_2 x}{\log_2 2}$ と逆向きに変換してしまうミス。

正しい底の変換公式： $$\log_a b = \frac{\log_c b}{\log_c a}$$ 変換先の底 $c$ が分母にくる。よって： $$\log_x 2 = \frac{\log_2 2}{\log_2 x} = \frac{1}{\log_2 x}$$
本質ポイント $\log_x 2 = \dfrac{1}{\log_2 x}$ ― 逆数の関係
🔑 底の変換公式より $\log_x 2 = \dfrac{1}{\log_2 x}$

$t = \log_2 x$（$t \neq 0$：$x \neq 1$ より）とおくと $$\frac{1}{t} + t = 3$$ → $t$ の2次方程式に帰着！
解説 方程式を解く
1 $t = \log_2 x$ とおく（$t \neq 0$）。$\log_x 2 = \dfrac{1}{t}$ より： $$\frac{1}{t} + t = 3$$
2 両辺に $t$ をかける（$t \neq 0$ なので OK）： $$t^2 - 3t + 1 = 0 \;\Rightarrow\; t = \frac{3 \pm \sqrt{5}}{2}$$
3 両解とも $t \neq 0$ を満たす。$x = 2^t$ より：
答　え
$$x = 2^{\dfrac{3+\sqrt{5}}{2}},\quad x = 2^{\dfrac{3-\sqrt{5}}{2}}$$
一般化 $\log_x a + \log_a x = k$ の型
$\log_x a + \log_a x = k$ の形は、$t = \log_a x$ とおくと $$\frac{1}{t} + t = k \;\Rightarrow\; t^2 - kt + 1 = 0$$ 実数解の条件：$k^2 - 4 \geq 0$ → $k \leq -2$ または $k \geq 2$
類題 同じ手順で解いてみよう
$\log_x 3 + \log_3 x = 2$ を解け（$x > 0$, $x \neq 1$）。
解答： $t = \log_3 x$：$t^2 - 2t + 1 = 0$ → $(t-1)^2 = 0$ → $t = 1$
$\log_3 x = 1$ より $x = 3$

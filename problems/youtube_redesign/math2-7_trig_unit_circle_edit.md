📌 問題・解法まとめ
【問題】単位円を用いて sin(π-θ) = sinθ を証明せよ
【キーポイント】角度 π-θ の点は P(cosθ, sinθ) の y 軸対称 → y 座標（= sin）は変わらない

変換公式まとめ
・sin(π-θ) = sinθ, cos(π-θ) = -cosθ （y 軸対称）
・sin(π+θ) = -sinθ, cos(π+θ) = -cosθ （原点対称）
・sin(-θ) = -sinθ, cos(-θ) = cosθ （x 軸対称）

変換公式に迷ったら、単位円の「どの対称か」を考えましょう！
コメントで「どの変換公式がいちばん混乱する？」を教えてください 👇

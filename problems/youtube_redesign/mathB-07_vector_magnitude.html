落とし穴 ベクトルの大きさは「足せない」！
この問題、どう解きますか？
$|\vec{a}|=3,\ |\vec{b}|=2,\ \vec{a}\cdot\vec{b}=4$ のとき $|\vec{a}+\vec{b}|$ を求めよ。
❌ よくある誤答
$|\vec{a}+\vec{b}| = |\vec{a}|+|\vec{b}|$
$= 3 + 2 = 5$
スカラーと同じ感覚で足してしまう罠！
✅ 正しい答え
$\sqrt{21}$
二乗 → 内積展開 → 平方根 が正解！
🔑 ベクトルの大きさは足し算できない。
$|\vec{a}+\vec{b}|^2 = |\vec{a}|^2 + 2\vec{a}\cdot\vec{b} + |\vec{b}|^2$ の展開が必須。
数学B ベクトルの大きさ（内積を使う）
★★★☆☆ 標準偏差値55〜65
問　題
$|\vec{a}|=3,\ |\vec{b}|=2,\ \vec{a}\cdot\vec{b}=4$ のとき $|\vec{a}+\vec{b}|$ の値を求めよ。
💬 $\vec{a}$ と $\vec{b}$ を図で見ると…
$\vec{a}+\vec{b}$ の長さは 平行四辺形の対角線。

大きさを単純に足すと $3+2=5$（❌）
実際は角度によって長さが変わる！
正しくは 二乗して展開 する。
⚠️ $|\vec{a}+\vec{b}|$ を直接求めることはできない。必ず二乗してから展開する。
出典：教科書例題・数学B（改題）
よくある間違い 「大きさを足せばいい」という思い込み
「真面目に勉強した人」ほどハマる間違いを確認します。
❌ 間違った解法
$|\vec{a}+\vec{b}| = |\vec{a}| + |\vec{b}|$
$= 3 + 2 = \mathbf{5}$ ← 完全に間違い！
⛔ 原因：数（スカラー）と同じ感覚でベクトルの大きさを「足せる」と思い込んでいる。
ベクトルの大きさは向きに依存するため、単純な足し算は成り立たない。
スカラーとベクトルの大きさの違い（必須知識）
スカラーの足し算
$a + b = b + a$
数値を直接足せる ✓
ベクトルの大きさ
$|\vec{a}+\vec{b}| \neq |\vec{a}|+|\vec{b}|$
直接足せない！
二乗→展開が必須
本質ポイント ベクトルの大きさの4ステップ解法
✅ この手順を身につければ、$|\vec{a}\pm\vec{b}|$ を確実に求められる！
1大きさを二乗する（$|\vec{a}+\vec{b}|^2$ の形にする）
2内積展開の公式を使う（$|\vec{a}|^2 + 2\vec{a}\cdot\vec{b} + |\vec{b}|^2$ に書き換える）
3数値を代入して計算する（$|\vec{a}|^2,\ |\vec{b}|^2,\ \vec{a}\cdot\vec{b}$ を代入）
4平方根を取る（$|\vec{a}+\vec{b}| = \sqrt{\cdots}$ として答えを出す）
💡 $-$ のときは $|\vec{a}-\vec{b}|^2 = |\vec{a}|^2 - 2\vec{a}\cdot\vec{b} + |\vec{b}|^2$（符号に注意！）
解説 $|\vec{a}+\vec{b}|$ を正しく求める
1 二乗する：$|\vec{a}+\vec{b}|^2$ を求める
2 内積展開： $|\vec{a}+\vec{b}|^2 = |\vec{a}|^2 + 2\vec{a}\cdot\vec{b} + |\vec{b}|^2$
3 代入： $|\vec{a}|^2=9,\ \vec{a}\cdot\vec{b}=4,\ |\vec{b}|^2=4$ を代入
　$= 9 + 2 \times 4 + 4 = 9 + 8 + 4 = 17+4$ $= 21$
4 平方根：$|\vec{a}+\vec{b}| > 0$ より $$\boxed{|\vec{a}+\vec{b}| = \sqrt{21}}$$
一般化 ベクトルの大きさの鉄則まとめ
📌 ベクトルの大きさは必ず二乗してから展開する：
$|\vec{a}+\vec{b}|^2$
$= |\vec{a}|^2 + 2\vec{a}\cdot\vec{b} + |\vec{b}|^2$
$|\vec{a}-\vec{b}|^2$
$= |\vec{a}|^2 - 2\vec{a}\cdot\vec{b} + |\vec{b}|^2$
符号が逆転！
試験で差がつくポイント
① なす角 $\theta$ が与えられた場合は先に内積を計算する（$\vec{a}\cdot\vec{b}=|\vec{a}||\vec{b}|\cos\theta$）
② $|\vec{a}-\vec{b}|^2$ の符号ミスに注意（中項は $-2\vec{a}\cdot\vec{b}$）
⏱️ この2点を押さえるだけで、ベクトルの大きさ問題は確実に得点できる。
類題 なす角が与えられたパターン
$|\vec{a}|=2,\ |\vec{b}|=3$，$\vec{a}$ と $\vec{b}$ のなす角が $60°$ のとき $|\vec{a}-\vec{b}|$ を求めよ。
1内積を計算：$\vec{a}\cdot\vec{b}=2\times3\times\cos60°=3$
2二乗して展開：$|\vec{a}-\vec{b}|^2 = |\vec{a}|^2 - 2\vec{a}\cdot\vec{b} + |\vec{b}|^2$
3代入：$= 4 - 2\times3 + 9 = 4-6+9 = 7$
4 平方根：$$\boxed{|\vec{a}-\vec{b}|=\sqrt{7}}$$
まとめ 今日の一言
🔑 $|\vec{a}+\vec{b}|$ を求めるとき、最初にやること：二乗する。
$|\vec{a}|^2 + 2\vec{a}\cdot\vec{b} + |\vec{b}|^2$ に展開
$|\vec{a}-\vec{b}|$ なら中項が $-2\vec{a}\cdot\vec{b}$（符号逆転！）
この展開を習慣にすれば、ベクトルの大きさ問題の失点はほぼゼロ。
⚠️ なす角が与えられた場合は先に $\vec{a}\cdot\vec{b}=|\vec{a}||\vec{b}|\cos\theta$ で内積を求めてから代入。

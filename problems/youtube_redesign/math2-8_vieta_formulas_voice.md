## スライド①　問題提示

今回の問題です。
x の2乗 マイナス 3x プラス 1 イコール 0 の2つの解を アルファ、ベータ とするとき、アルファの2乗 プラス ベータの2乗 の値を求める問題です。

この問題、解の公式を使って アルファ と ベータ の値を実際に求めようとしていませんか？
解と係数の関係を使えば、解を求めることなく一行で答えが出ます。
それが今回のテーマです。

## スライド②　よくある間違い

まず、典型的な間違いから確認しましょう。

解の公式を使うと アルファ イコール 3プラスルート5わる2、ベータ イコール 3マイナスルート5わる2 と求まります。

ここでアルファの2乗とベータの2乗をそれぞれ計算しようとすると、14プラス6ルート5わる4、14マイナス6ルート5わる4 という複雑な式になってしまいます。

計算が複雑になりミスが生じやすい、典型的な遠回りです。

実は アルファ と ベータ の個々の値は不要です。係数だけから アルファプラスベータ と アルファかけるベータ がわかれば、答えが出ます。

## スライド③　本質ポイント

整理しましょう。

x の2乗 マイナス 3x プラス 1 イコール 0 の2つの解 アルファ、ベータ に対して、解と係数の関係から次のことが言えます。

アルファ プラス ベータ イコール 3。これは x の係数 マイナス3 の符号を反転した値です。
アルファ かける ベータ イコール 1。これはそのまま定数項です。

あとは恒等式の変形だけです。
アルファの2乗プラスベータの2乗 イコール カッコアルファプラスベータカッコ閉じの2乗 マイナス 2かけるアルファベータ イコール 3の2乗 マイナス 2かけるいち イコール 7。

解と係数の関係を使えば、係数から直接計算できます。これが本質です。

## スライド⑥　類題

では類題です。
2かけるxの2乗 マイナス 5x プラス 3 イコール 0 の2解を アルファ、ベータ とするとき、いちわるアルファ プラス いちわるベータ の値を求める問題です。

解と係数の関係と、一般化スライドの変形パターンを使って、自分で解いてみてください。

ステップいち。解と係数の関係より、アルファプラスベータ イコール 5わる2、アルファかけるベータ イコール 3わる2。

ステップに。いちわるアルファ プラス いちわるベータ イコール アルファプラスベータ わる アルファかけるベータ。分子と分母に代入すると、5わる2 を 3わる2 で割って、5わる3。

答えは 5わる3 です。

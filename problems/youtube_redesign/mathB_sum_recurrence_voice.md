# 【音声台本】数学B｜和と一般項の関係式（Sn型漸化式）

対象スライド：mathB_sum_recurrence.html
難易度：標準+（偏差値55〜65）

※ 数式はAI音声読み上げ向けに日本語表記へ変換済み

---

## スライド①　問題提示

はい、今回の問題はこちらです。
「数列 a の n 番目 の初項から第 n 項までの和を S の n 番目 とするとき、すべての自然数 n に対して S の n 番目 イコール 2 かける a の n 番目 マイナス n が成り立つ。a の 1 番目 を求め、一般項 a の n 番目 を求めよ。」
S の n 番目 と a の n 番目 が混在していますね。どこから手をつければいいでしょうか。

---

## スライド②　よくある間違い

ここで1つ、よくある失点パターンを確認しておきます。
a の n 番目 イコール S の n 番目 マイナス S の n マイナス 1 番目 をすぐ使ってしまうケースです。
この公式は n が 2 以上 でのみ成立します。n イコール 1 のとき S の 0 番目 は定義されないので要注意。
まず n イコール 1 を直接代入して a の 1 番目 を求め、それから n が 2 以上 の漸化式を導く。この順番が鉄則です。
さらに最後に、求めた一般項が n イコール 1 でも成立するか確認することも忘れずに。

---

## スライド③　本質ポイント

核心はここです。n が 2 以上 のとき、a の n 番目 イコール S の n 番目 マイナス S の n マイナス 1 番目 を使います。
S の n 番目 イコール 2 かける a の n 番目 マイナス n から、
a の n 番目 イコール カッコ 2 かける a の n 番目 マイナス n カッコ マイナス カッコ 2 かける a の n マイナス 1 番目 マイナス n マイナス 1 カッコ。
整理すると a の n 番目 イコール 2 かける a の n マイナス 1 番目 プラス 1。
さらに変形すると a の n 番目 プラス 1 イコール 2 かける カッコ a の n マイナス 1 番目 プラス 1 カッコ。これは等比数列の形です。

---

## スライド④　解説

では実際に解いていきます。
ステップ1。n イコール 1 のとき。S の 1 番目 イコール a の 1 番目 イコール 2 かける a の 1 番目 マイナス 1。整理して a の 1 番目 イコール 1。
ステップ2。n が 2 以上 のとき。a の n 番目 プラス 1 イコール 2 かける カッコ a の n マイナス 1 番目 プラス 1 カッコ。公比 2 の等比数列。初項 a の 1 番目 プラス 1 イコール 2 なので、
a の n 番目 プラス 1 イコール 2 かける 2 の n マイナス 1 乗 イコール 2 の n 乗。
ステップ3。n イコール 1 でも 2 の 1 乗 マイナス 1 イコール 1 イコール a の 1 番目。式は全ての n で成立します。
答えは a の 1 番目 イコール 1、a の n 番目 イコール 2 の n 乗 マイナス 1 です。

---

## スライド⑤　一般化

一般化しておきましょう。S の n 番目 イコール p かける a の n 番目 プラス f(n)、p が 1 でない の形は4ステップで解けます。
n イコール 1 を代入して a の 1 番目 を確定。
n が 2 以上 で漸化式を導出。
等比か等差の形に変形して解く。
最後に n イコール 1 で確認。

## スライド⑥　類題

では類題を一問。
「S の n 番目 イコール 2 かける a の n 番目 プラス n のとき、a の n 番目 を求めよ。」
n イコール 1：a の 1 番目 イコール 2 かける a の 1 番目 プラス 1、a の 1 番目 イコール マイナス1。
n が 2 以上：a の n 番目 イコール 2 かける a の n マイナス 1 番目 マイナス 1。a の n 番目 マイナス 1 イコール 2 かける カッコ a の n マイナス 1 番目 マイナス 1 カッコ。初項 マイナス2 の公比 2 の等比数列。
答えは a の n 番目 イコール 1 マイナス 2 の n 乗 です。
手順はまったく同じですね。ぜひ自分でも確認してみてください。

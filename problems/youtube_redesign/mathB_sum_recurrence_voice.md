## スライド①　問題提示

今回の問題です。
「S の n 番目 イコール 2 かける a の n 番目 マイナス n が成り立つ。a の 1 番目 と一般項を求めよ。」

S の n 番目 と a の n 番目 が混在していますね。
どこから手をつけるか、少し考えてみてください。

---

## スライド②　よくある間違い

「a の n 番目 イコール S の n 番目 マイナス S の n マイナス 1 番目 を使えばいい」
…ちょっと待ってください。

この公式、n が 2 以上 でしか使えません。
n イコール 1 のとき S の 0 番目 は定義されていないので。

だから最初にすることは、n イコール 1 を直接代入して a の 1 番目 を求めること。
そのあと n が 2 以上 の漸化式を導く。この順番が大事です。

それから最後に、求めた一般項が n イコール 1 でも成立するか確認すること。
この3つのステップ、セットで覚えておいてください。

---

## スライド③　本質ポイント

n が 2 以上 のとき、a の n 番目 イコール S の n 番目 マイナス S の n マイナス 1 番目 を使います。
S の n 番目 イコール 2 かける a の n 番目 マイナス n から計算すると、
a の n 番目 イコール 2 かける a の n マイナス 1 番目 プラス 1 という漸化式が出てきます。

あ、変形すると a の n 番目 プラス 1 イコール 2 かける カッコ a の n マイナス 1 番目 プラス 1 カッコ。
これは等比数列の形です。

---

## スライド④　解説

解いていきます。

ステップ1。n イコール 1 のとき。a の 1 番目 イコール 2 かける a の 1 番目 マイナス 1。
整理して a の 1 番目 イコール 1。

ステップ2。n が 2 以上 のとき。a の n 番目 プラス 1 イコール 2 かける カッコ a の n マイナス 1 番目 プラス 1 カッコ。
公比 2 の等比数列。初項 a の 1 番目 プラス 1 イコール 2 なので、
a の n 番目 プラス 1 イコール 2 の n 乗。

ステップ3。n イコール 1 でも 2 の 1 乗 マイナス 1 イコール 1 イコール a の 1 番目。合っています。

答えは a の 1 番目 イコール 1、a の n 番目 イコール 2 の n 乗 マイナス 1。

---

## スライド⑤　一般化

S の n 番目 イコール p かける a の n 番目 プラス f(n)、p が 1 でない の形は、いつも同じ4ステップです。
n イコール 1 を代入して a の 1 番目 を確定。
n が 2 以上 で漸化式を導出。
等比か等差の形に変形して解く。
n イコール 1 で確認。

この流れ、型として身につけてください。

## スライド⑥　類題

では一問。
「S の n 番目 イコール 2 かける a の n 番目 プラス n のとき、a の n 番目 を求めよ。」

n イコール 1：a の 1 番目 イコール 2 かける a の 1 番目 プラス 1、a の 1 番目 イコール マイナス1。
n が 2 以上：a の n 番目 マイナス 1 イコール 2 かける カッコ a の n マイナス 1 番目 マイナス 1 カッコ。初項 マイナス2 の公比 2 の等比数列。
答えは a の n 番目 イコール 1 マイナス 2 の n 乗。

自分でも確認してみてください。

数学II 三角関数の合成（sin + √3 cos の最大・最小）
★★★☆☆ 標準偏差値55〜65
問　題
$f(\theta) = \sin\theta + \sqrt{3}\cos\theta \quad (0 \leq \theta < 2\pi)$
の最大値・最小値と、それを与える $\theta$ の値を求めよ。
💬 $\sin$ と $\cos$ が混在していてそのままでは扱いにくい。
「合成」で1つの三角関数にまとめよう！
出典：センター試験・数学II（改題）
よくある間違い $R\sin\alpha$ と $R\cos\alpha$ の対応を逆にしてしまう
❌ 間違いの典型パターン
$R\sin(\theta+\alpha) = R(\sin\theta\cos\alpha + \cos\theta\sin\alpha)$ を展開すると：

正しい対応：
$\sin\theta$ の係数 $\Rightarrow R\cos\alpha = 1$， $\cos\theta$ の係数 $\Rightarrow R\sin\alpha = \sqrt{3}$
$\Rightarrow \tan\alpha = \dfrac{\sqrt{3}}{1} = \sqrt{3}$ → $\alpha = \dfrac{\pi}{3}$ ✅

❌ よくある間違い：
「$R\sin\alpha = 1$，$R\cos\alpha = \sqrt{3}$」と逆に対応させて
$\tan\alpha = \dfrac{1}{\sqrt{3}}$ → $\alpha = \dfrac{\pi}{6}$ と答えてしまう。
覚え方：sin に対しては cos、cos に対しては sin — 「たすき掛け」の対応
本質ポイント 展開して係数比較が最も確実
🔑 $a\sin\theta + b\cos\theta = R\sin(\theta+\alpha)$ の両辺を展開： $$R\cos\alpha \cdot \sin\theta \;+\; R\sin\alpha \cdot \cos\theta$$ 係数を比較： $$R\cos\alpha = a, \quad R\sin\alpha = b$$ $$\Rightarrow R = \sqrt{a^2+b^2}, \quad \alpha \text{ は } \cos\alpha,\,\sin\alpha \text{ の符号で象限を決定}$$
解説 $\sin\theta + \sqrt{3}\cos\theta$ を合成して最大・最小を求める
1 $R\cos\alpha = 1$，$R\sin\alpha = \sqrt{3}$ より $$R = \sqrt{1^2+(\sqrt{3})^2} = 2, \quad \cos\alpha = \tfrac{1}{2},\;\sin\alpha = \tfrac{\sqrt{3}}{2} \;\Rightarrow\; \alpha = \frac{\pi}{3}$$
2 よって $f(\theta) = 2\sin\!\left(\theta+\dfrac{\pi}{3}\right)$。
$0 \leq \theta < 2\pi$ より $\dfrac{\pi}{3} \leq \theta+\dfrac{\pi}{3} < \dfrac{7\pi}{3}$。
3 $\sin\!\left(\theta+\dfrac{\pi}{3}\right) = 1$（$\theta+\dfrac{\pi}{3}=\dfrac{\pi}{2}$）のとき $\theta = \dfrac{\pi}{6}$ → 最大値 $2$
$\sin\!\left(\theta+\dfrac{\pi}{3}\right) = -1$（$\theta+\dfrac{\pi}{3}=\dfrac{3\pi}{2}$）のとき $\theta = \dfrac{7\pi}{6}$ → 最小値 $-2$
一般化 $a\sin\theta + b\cos\theta = R\sin(\theta+\alpha)$ の公式まとめ
$$R = \sqrt{a^2+b^2}, \quad R\cos\alpha = a, \quad R\sin\alpha = b$$
  • $a>0,\;b>0$ → $\alpha$ は第1象限（$0 < \alpha < \frac{\pi}{2}$）
  • $a>0,\;b<0$ → $\alpha$ は第4象限（$-\frac{\pi}{2} < \alpha < 0$）
  • $a<0,\;b>0$ → $\alpha$ は第2象限
  • $a<0,\;b<0$ → $\alpha$ は第3象限
⚠️ 定義域がある場合、合成後の $\theta+\alpha$ の範囲を必ず計算すること！
類題 合成してみよう — b が負の場合
$g(\theta) = \sqrt{3}\sin\theta - \cos\theta \quad (0 \leq \theta < 2\pi)$ の最大値・最小値を求めよ。
答　え
$R\cos\alpha = \sqrt{3}$，$R\sin\alpha = -1$ より $R = 2$，$\cos\alpha = \dfrac{\sqrt{3}}{2}$，$\sin\alpha = -\dfrac{1}{2}$
$\Rightarrow \alpha = -\dfrac{\pi}{6}$ （第4象限）
$g(\theta) = 2\sin\!\left(\theta - \dfrac{\pi}{6}\right)$
最大値 $\boldsymbol{2}$（$\theta = \dfrac{2\pi}{3}$），　最小値 $\boldsymbol{-2}$（$\theta = \dfrac{5\pi}{3}$）

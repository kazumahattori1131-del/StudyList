## スライド①　フック

対数不等式。この問題、解けますか。
2分の1を底とする x の対数 が マイナス1 より大きい、定義域は x が0より大きい。

最初に「x が 2 より大きい」と答えた方、要注意です。
その答えは正反対の間違いです。

正しい答えは 0 より大きく 2 未満。不等号の向きが逆なんです。
原因はひとつ。底が 2分の1 、つまり 1 より小さいとき、対数の不等号は反転します。
今日はこの落とし穴を完全に攻略します。

## スライド②　問題提示

今回の問題です。
2分の1を底とする x の対数 が マイナス1 より大きい、定義域は x が0より大きい、の解を求めます。

グラフで確認しましょう。y イコール 2分の1を底とする x の対数、このグラフの形は何でしょうか。
底が 2分の1 、つまり 1 より小さいので、このグラフは単調減少、右肩下がりです。
x が大きくなるにつれ y が小さくなっていきます。

y が マイナス1 より大きい領域はどこか。グラフで見ると、点 x イコール 2 よりも左側の領域です。
ここが解のイメージです。では計算で確認します。

## スライド③　よくある間違い

典型的な間違いを確認します。
2分の1を底とする x の対数 が マイナス1 より大きい。
マイナス1 を 2分の1を底とする対数に変換すると、2分の1を底とする 2 の対数 になります。
ここで「対数は単調増加だから不等号はそのまま」と考えて x が 2 より大きい と答えてしまう。
これが典型的な間違いです。完全に逆です。
理由は、底が 2分の1 で 1 より小さいからです。
底が 1 より大きいときは単調増加。底が 0 より大きく 1 より小さいときは単調減少。
単調減少のとき、不等号の向きは必ず逆転します。

## スライド④　本質ポイント

対数不等式の解き方を4ステップで整理します。
ステップ1。真数条件を確認する。x が 0 より大きい、2 x マイナス 1 が 0 より大きい、などです。
ステップ2。底 a を確認する。a が 1 より大きいか、0 と 1 の間にあるかを判断します。
ステップ3。不等号の向きを決める。底が 1 より大きければ不等号そのまま。底が 1 より小さければ不等号を逆にします。
ステップ4。解を求め、真数条件との共通部分を取ります。

模試で大切なのはステップ2です。底の確認は 2秒 でできます。でもここを飛ばすと答えが丸ごと逆になる。このステップを習慣化することが合否を分けます。

---

## スライド⑤　解説

では実際に解いていきます。
ステップ1。真数条件は x が 0 より大きい。
ステップ2。底は 2分の1。これは 1 より小さいので単調減少です。
ステップ3。マイナス1 を 2分の1を底とする対数に変換します。
2分の1の マイナス1乗 は 2 ですから、マイナス1 イコール 2分の1を底とする 2 の対数 と書けます。
これを代入すると、2分の1を底とする x の対数 が 2分の1を底とする 2 の対数 より大きい。
底が 1 より小さく単調減少なので、不等号を逆にして x が 2 より小さい。
ステップ4。真数条件 x が 0 より大きい、との共通部分を取ります。
答えは 0 より大きく 2 未満。

## スライド⑥　一般化

対数不等式のルールをまとめます。

a を底とする f の対数 が a を底とする g の対数 より大きい、という不等式を解くとき。
底 a が 1 より大きければ単調増加ですから不等号の向きはそのまま、f が g より大きい。
底 a が 0 と 1 の間にあれば単調減少ですから不等号の向きが逆転し、f が g より小さい。

試験で差がつくポイントがふたつあります。
ひとつ目、底の確認を先にする。不等号の方向を確定してから計算に入ること。
ふたつ目、真数条件を最後に忘れない。f が 0 より大きい、g が 0 より大きい、この条件との共通部分を取るまでが解答です。

このふたつを習慣にするだけで、対数不等式の失点がほぼゼロになります。

## スライド⑦　類題

では類題を解いてみましょう。
3分の1を底とする x の対数 が マイナス2 以下の解を求めよ。

ステップ1。真数条件は x が 0 より大きい。
ステップ2。底は 3分の1。これは 1 より小さいので単調減少です。不等号は逆転します。
ステップ3。マイナス2 を 3分の1を底とする対数に変換します。
3分の1の マイナス2乗 は 9 ですから、マイナス2 イコール 3分の1を底とする 9 の対数 と書けます。
3分の1を底とする x の対数 が 3分の1を底とする 9 の対数 以下。
底が 1 より小さく単調減少なので、不等号を逆にして x が 9 以上。
ステップ4。真数条件 x が 0 より大きい、との共通部分を取ります。
答えは x が 9 以上。

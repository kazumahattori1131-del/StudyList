数学C 複素数の恒等式活用
★★★★☆ 発展偏差値60〜70
問　題
複素数 $z$ が $z + \dfrac{1}{z} = 1$ を満たすとき、 $z^3 + \dfrac{1}{z^3}$ の値を求めよ。
💬 $z$ を直接求めてから $z^3$ を計算する？ ―― もっと賢い方法がある。
ヒント $\left(z + \dfrac{1}{z}\right)^3$ を展開してみよう
HINT： $\left(z + \dfrac{1}{z}\right)^3$ を展開すると？

$$\left(z + \frac{1}{z}\right)^3 = z^3 + \frac{1}{z^3} + 3\left(z + \frac{1}{z}\right)$$
左辺は $1^3 = 1$。右辺の $3\left(z + \dfrac{1}{z}\right) = 3 \times 1 = 3$。

※ 展開で $(z+1/z)^2 = z^2 + \mathbf{2} + 1/z^2$（中間の $+2$ を忘れずに）
本質ポイント 条件式の累乗に $z^3 + 1/z^3$ が含まれる
🔑 $\left(z + \dfrac{1}{z}\right)^3 = z^3 + \dfrac{1}{z^3} + 3\left(z + \dfrac{1}{z}\right)$

$z + \dfrac{1}{z} = 1$ を代入： $$1 = z^3 + \frac{1}{z^3} + 3$$ → $z$ を求めずに $z^3 + \dfrac{1}{z^3}$ が直接求まる！
解説 3乗の展開を利用する
1 $\left(z + \dfrac{1}{z}\right)^3$ を展開： $$= z^3 + 3z \cdot \frac{1}{z^2} + 3z^2 \cdot \frac{1}{z^3}\cdot z^3 + \frac{1}{z^3}$$ 整理すると： $$= z^3 + \frac{1}{z^3} + 3\!\left(z + \frac{1}{z}\right)$$
2 $z + \dfrac{1}{z} = 1$ を代入： $$1 = z^3 + \frac{1}{z^3} + 3$$
答　え
$$z^3 + \frac{1}{z^3} = -2$$
一般化 $a_n = z^n + 1/z^n$ を漸化式で求める
$a_n = z^n + \dfrac{1}{z^n}$ とおくと、$z + \dfrac{1}{z} = k$ のとき： $$a_{n+1} = k \cdot a_n - a_{n-1}, \quad a_0 = 2,\; a_1 = k$$ この漸化式で $z^n + \dfrac{1}{z^n}$ を$z$ を求めずに順々に計算できる。
類題 同じ手順で解いてみよう
$z + \dfrac{1}{z} = -1$ のとき、$z^3 + \dfrac{1}{z^3}$ を求めよ。
解答： $\left(z + \dfrac{1}{z}\right)^3 = z^3 + \dfrac{1}{z^3} + 3\left(z + \dfrac{1}{z}\right)$
$(-1)^3 = z^3 + \dfrac{1}{z^3} + 3(-1)$
$-1 = z^3 + \dfrac{1}{z^3} - 3$ $$z^3 + \frac{1}{z^3} = 2$$

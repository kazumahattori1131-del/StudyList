## ① 追い装飾指示

### [Slide②　ヒント]

**強調①**
- 強調対象：`$b_n = \dfrac{a_n}{2^n}$ とおくと、$b_{n+1}$ と $b_n$ の関係は？`
- 方法：黄色ハイライト
- アニメーション：slide_up
- タイミング：「b の n 番目 イコール a の n 番目 わる 2 の n 乗 とおけば、」
- 理由：「何を置くか」というアイデアの発生タイミングで視覚強調する

**強調②**
- 強調対象：`※ $b_1 = \dfrac{a_1}{2^1}$（$a_1$ をそのまま代入しないこと！）`
- 方法：赤枠
- アニメーション：fade
- タイミング：「初項 b の 1 番目 イコール 2分の1」
- 理由：「a₁ = 1 をそのまま代入する」という典型ミスを防ぐ

---

### [Slide③　本質ポイント]

**強調①**
- 強調対象：`右辺の $2^n$ を $2^{n+1}$ で割ると $\dfrac{1}{2}$（定数）になるのが変換の鍵。`
- 方法：赤枠
- アニメーション：zoom
- タイミング：「右辺の底と係数の底が同じなので、両辺をその底で割ることができます。」
- 理由：なぜ等差数列に化けるかの理由を視覚定着させる

## ④ サムネ案

- テキスト：「両辺を割るだけ」
- 構図：左に漸化式 `aₙ₊₁ = 2aₙ + 2ⁿ`、右に変換後 `bₙ₊₁ = bₙ + 1/2`（等差！）、背景はオレンジ

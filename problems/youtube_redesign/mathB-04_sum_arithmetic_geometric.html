数学B 等差×等比型の和（差し引き法）
★★★☆☆ 標準偏差値55〜65
$n$ を正の整数とするとき、次の和を求めよ。

$$S = \sum_{k=1}^{n} k \cdot 2^k$$

等差数列の和？　等比数列の和？　どちらの公式も使えない…

よくある間違い Σを「積に分解」しようとする

❌ よくある発想①：Σを積に分解する

$$\sum_{k=1}^{n} k \cdot 2^k \;\overset{?}{=}\; \left(\sum_{k=1}^{n} k\right) \times \left(\sum_{k=1}^{n} 2^k\right)$$

→ Σは積に分解できない！ これは完全な間違い。

❌ よくある発想②：1項ずつ書き出して足す

$k=1,2,3,\ldots$ を代入して書き出す → 一般項が複雑で規則性が見えない

→ 有限個なら計算できるが、一般項 $a_n$ を求める形に直せない。

⚠️ この形は「等差」でも「等比」でも解けない。別のアプローチが必要！
本質ポイント S と 2S を並べて「差し引く」

$S$ の両辺に公比 $2$ をかけた $2S$ を作り、引き算する：

$S \;=\; \boldsymbol{1}{\cdot}2^1 + \boldsymbol{2}{\cdot}2^2 + \boldsymbol{3}{\cdot}2^3 + \cdots + \boldsymbol{n}{\cdot}2^n$
$2S = \phantom{1{\cdot}2^1\;+\;} \boldsymbol{1}{\cdot}2^2 + \boldsymbol{2}{\cdot}2^3 + \cdots + \boldsymbol{(n{-}1)}{\cdot}2^n + \boldsymbol{n}{\cdot}2^{n+1}$
$S - 2S$ を計算 → 右辺の大部分が打ち消し、等比数列の和が現れる！

※ 京大生は「等差×等比型」を見た瞬間に差し引き法を選びます

解説 4ステップで完全解答
STEP 1$S$ を書き出す
$S = 1{\cdot}2^1 + 2{\cdot}2^2 + 3{\cdot}2^3 + \cdots + n{\cdot}2^n$
STEP 2$2S$ を書き出す（右に1項ずつずらす）
$2S = \phantom{1{\cdot}2^1 + {}} 1{\cdot}2^2 + 2{\cdot}2^3 + \cdots + (n{-}1){\cdot}2^n + n{\cdot}2^{n+1}$
STEP 3差し引き：$S - 2S = -S$
$-S = \underbrace{(2^1 + 2^2 + \cdots + 2^n)}_{\text{等比数列の和}} - n{\cdot}2^{n+1} = (2^{n+1}-2) - n{\cdot}2^{n+1}$
STEP 4$S$ を求める： $S = (n-1)\cdot 2^{n+1} + 2$
一般化 等差×等比型の解法パターン
$\displaystyle S = \sum_{k=1}^{n} k \cdot r^k \quad (r \neq 1)$

差し引き法の流れ

  • $S$ を書き出し、両辺に $r$ をかけた $rS$ を作る
  • $S - rS$ を計算 → 右辺に等比数列の和 $+ $ 末尾の余り項が現れる
  • 等比数列の和の公式で整理し、$S$ を求める
答え：$S = \dfrac{(n-1)r^{n+1}}{r-1} \cdot r + \dfrac{r(r^n - 1)}{(r-1)^2}$ （$r=2$ なら $(n-1){\cdot}2^{n+1}+2$）
類題 同じ手順で解いてみよう
$n$ を正の整数とするとき、$\displaystyle T = \sum_{k=1}^{n} k \cdot 3^k$ を求めよ。
答　え
$T$ と $3T$ を書き出して差し引く：
$T - 3T = -2T = (3^1 + 3^2 + \cdots + 3^n) - n{\cdot}3^{n+1}$
$\phantom{-2T} = \dfrac{3(3^n - 1)}{2} - n{\cdot}3^{n+1} = \dfrac{3^{n+1}-3}{2} - n{\cdot}3^{n+1}$

$4T = (2n-1){\cdot}3^{n+1} + 3$

$\therefore\; T = \dfrac{(2n-1){\cdot}3^{n+1} + 3}{4}$

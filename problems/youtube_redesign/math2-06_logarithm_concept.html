数学II 対数の概念（指数から対数へ）
★★☆☆☆ 基本〜標準偏差値50〜60
次の値を求めよ。

（1）$\log_2 8$ 　　（2）$\log_3 \dfrac{1}{9}$

「$\log$」って何を求めているの？ 指数と対数の関係さえ押さえれば怖くない！

よくある間違い 底と真数を混同してしまう

❌ よくある間違い①：底と真数を逆に見る

$$\log_2 8 \;\overset{?}{=}\; 4$$

→ $2^4 = 16 \neq 8$。「$2$ を何乗すれば $8$ か」を答えるべきなのに $8$ の方を底と混同している。

❌ よくある間違い②：$\log_3 \dfrac{1}{9}$ の符号を忘れる

$$\log_3 \frac{1}{9} \;\overset{?}{=}\; 2$$

→ $3^2 = 9$ であり $\dfrac{1}{9}$ ではない。$\dfrac{1}{9} = 3^{-2}$ なので答えは $-2$。

⚠️ $\log_a b$ は「$a$ を何乗すれば $b$ になるか」という数。符号を含めて指数として考える！
本質ポイント 「何乗すれば底から真数になるか」が対数

対数は指数の「逆」の操作。$2^3 = 8$ という指数の式をひっくり返すと：

指数の式：$2^3 = 8$
　　　↕　（変換）
対数の式：$\log_2 8 = 3$

「$2$（底）を $3$（対数）乗すると $8$（真数）になる」
$\log_a b = c \;\Longleftrightarrow\; a^c = b$
（$a > 0,\; a \neq 1,\; b > 0$）
解説 定義に戻って確実に解く
（1）$\log_2 8$：「$2$ を何乗すれば $8$ か」
$2^1 = 2,\quad 2^2 = 4,\quad 2^3 = 8$ → $8 = 2^3$ より $\log_2 8 = 3$
（2）$\log_3 \dfrac{1}{9}$：「$3$ を何乗すれば $\dfrac{1}{9}$ か」
$\dfrac{1}{9} = \dfrac{1}{3^2} = 3^{-2}$ より $\log_3 \dfrac{1}{9} = -2$
確認変換公式で検証：$\log_2 8 = 3 \Leftrightarrow 2^3 = 8$ ✓　　$\log_3 \dfrac{1}{9} = -2 \Leftrightarrow 3^{-2} = \dfrac{1}{9}$ ✓
一般化 $\log_a b = c \Leftrightarrow a^c = b$
$\log_a b = c \;\Longleftrightarrow\; a^c = b \quad (a>0,\; a\neq 1,\; b>0)$

対数の基本性質

  • $\log_a 1 = 0$　（$a^0 = 1$ だから）
  • $\log_a a = 1$　（$a^1 = a$ だから）
  • $\log_a a^n = n$　（$a^n = a^n$ だから）
  • $\log_a (MN) = \log_a M + \log_a N$　（対数の積の法則）
まず「底を何乗すれば真数になるか」という問いかけに戻ること。
複雑な計算も定義の確認からスタートする習慣を！
類題 同じ手順で解いてみよう
$\log_5 125 - \log_5 5$ を計算せよ。
答　え
$\log_5 125 = \log_5 5^3 = 3$
$\log_5 5 = \log_5 5^1 = 1$

$\log_5 125 - \log_5 5 = 3 - 1 = 2$

（別解）対数の差の公式：$\log_5 \dfrac{125}{5} = \log_5 25 = \log_5 5^2 = 2$

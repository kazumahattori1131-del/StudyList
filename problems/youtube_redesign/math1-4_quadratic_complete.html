数学I 平方完成と頂点（計算ミスをなくす）
★★☆☆☆ 基本〜標準偏差値50〜60
$f(x) = 2x^2 - 8x + 5$ を平方完成し、最小値とそのときの $x$ の値を求めよ。

係数が $2$ のとき、どうやって平方完成する？ 符号と定数項の処理で失点する人が続出！

よくある間違い 定数項の処理を間違える

❌ よくある間違い①：$x^2$ の係数を無視して平方完成する

$$2x^2 - 8x + 5 \;\overset{?}{=}\; 2(x - 8)^2 + 5 - 128$$

→ $2$ でくくる前に $-8x$ だけを見て $(x-8)^2$ としてしまう誤り。

❌ よくある間違い②：引いた項の符号を間違える

$$2(x^2 - 4x) + 5 \;\overset{?}{=}\; 2(x-2)^2 + 5 + 8$$

→ $2(x-2)^2$ を展開すると $2x^2 - 8x + 8$ なので、引いた $8$ を足しているが符号が逆になる。

⚠️ 係数 $a$ があるときは「$a$ でくくる → 平方完成 → 余分に足した分だけ引く」の3ステップを守る！
本質ポイント $a(x-p)^2+q$ の形にする3ステップ

平方完成とは $ax^2 + bx + c$ を $a(x-p)^2 + q$ の形に書き直すこと。頂点は $(p,\, q)$。

ステップ1：$x^2$ の係数 $a$ でくくる
$2x^2 - 8x + 5 = 2(x^2 - 4x) + 5$

ステップ2：括弧の中を平方完成
$x^2 - 4x = (x - 2)^2 - 4$

ステップ3：$a$ をかけ直して定数を整理
$= 2\{(x-2)^2 - 4\} + 5 = 2(x-2)^2 - 8 + 5$
$f(x) = 2(x-2)^2 - 3$　→　頂点 $(2,\,-3)$、最小値 $-3$
解説 3ステップで完全解答
STEP 1係数 $2$ でくくる
$f(x) = 2(x^2 - 4x) + 5$
STEP 2括弧の中を平方完成：$-4 \div 2 = -2$ なので $(x-2)^2$ の形へ
$x^2 - 4x = (x-2)^2 - 4$　（$(x-2)^2 = x^2 - 4x + 4$ から $4$ を引いて調整）
STEP 3$2$ をかけ直して定数を整理
$f(x) = 2\{(x-2)^2 - 4\} + 5 = 2(x-2)^2 - 8 + 5 = 2(x-2)^2 - 3$
答え$f(x) \geq -3$（等号は $x=2$ のとき） 最小値 $-3$（$x=2$ のとき）
一般化 平方完成の公式と頂点のまとめ
$ax^2 + bx + c = a\!\left(x + \dfrac{b}{2a}\right)^{\!2} - \dfrac{b^2 - 4ac}{4a}$

ミスをなくす確認ポイント

  • くくり出す係数 $a$ は $x^2$ と $x$ の両方にかかる（定数 $c$ はくくらない）
  • $(x - p)^2 = x^2 - 2px + p^2$ なので、引いた $ap^2$ を外に出す
  • 計算後に展開して元の式と一致するか必ず確認する
頂点 $\left(-\dfrac{b}{2a},\; -\dfrac{b^2-4ac}{4a}\right)$　← これを暗記するより3ステップを手で覚える！
類題 同じ手順で解いてみよう
$f(x) = -x^2 + 4x + 1$ を平方完成し、最大値とそのときの $x$ の値を求めよ。
答　え
$f(x) = -(x^2 - 4x) + 1$
$x^2 - 4x = (x-2)^2 - 4$ より
$f(x) = -\{(x-2)^2 - 4\} + 1 = -(x-2)^2 + 4 + 1 = -(x-2)^2 + 5$

$a = -1 < 0$ なので上に凸。$x=2$ のとき最大値。

$\therefore\;$ 最大値 $5$（$x = 2$ のとき）

## スライド①　問題提示, 00:00

「ログ」という記号、見ただけで頭が真っ白になりませんか。

ログ 2 の 8、ログ 3 の 9ぶんの1。これらの値を求めよ。

対数は多くの高校生がつまずく単元です。でも、「対数とは何か」という定義さえ理解すれば、あとは指数の計算と同じです。今日は指数と対数の関係を完全に理解しましょう。

では問題です。

（1）ログ 2 の 8　（2）ログ 3 の 9ぶんの1 を求めよ。

---

## スライド②　よくある間違い, 00:30

まず、よくある間違いを見ていきます。

まちがい いち。底と真数を逆に見てしまうパターンです。

ログ 2 の 8 を「8を何乗すれば2になるか」と読んでしまい、4と答えてしまう。2の4乗は16で8ではありません。ログ 2 の 8 は「2を何乗すれば8になるか」を表しています。

まちがい に。ログ 3 の 9ぶんの1 の符号を忘れるパターンです。

9ぶんの1を見て「3の2乗は9だから答えは2」と答えてしまう。でも 9ぶんの1 は 9 ではありません。9ぶんの1 ＝ 3のマイナス2乗 なので、答えはマイナス2です。

対数の記号に慣れていないと、底と真数を混同したり符号を落としたりしやすいです。定義に立ち返ることが大切です。

---

## スライド③　本質ポイント, 01:30

対数の意味を説明します。

対数とは「指数の逆」です。2の3乗 ＝ 8、という指数の式を見ると、「底が2、指数が3、計算結果が8」という関係があります。

この式をひっくり返すと「2を何乗すれば8になるか、その答えは3」という情報に変換できます。これを対数の記号で書くと、ログ 2 の 8 ＝ 3 となります。

つまり、ログ エー の ビー ＝ シー と エーのシー乗 ＝ ビー は同じことを言っています。

このふたつの式を行き来できることが対数の核心です。定義は、ログ エー の ビー ＝ シー と エーのシー乗 ＝ ビー は同値。エー ＞ 0、エー ≠ 1、ビー ＞ 0 という条件もあわせて覚えましょう。

---

## スライド④　解説, 02:30

では定義に戻って解いていきます。

いち番。ログ 2 の 8。「2を何乗すれば8になるか」と考えます。

2の1乗 ＝ 2、2の2乗 ＝ 4、2の3乗 ＝ 8。8 ＝ 2の3乗 なので、ログ 2 の 8 ＝ 3 です。

に番。ログ 3 の 9ぶんの1。「3を何乗すれば9ぶんの1になるか」と考えます。

9ぶんの1 ＝ 3の2乗ぶんの1 ＝ 3のマイナス2乗。よってログ 3 の 9ぶんの1 ＝ マイナス2 です。

確認します。変換公式で検証します。ログ 2 の 8 ＝ 3 は 2の3乗 ＝ 8 と同値、正しい。ログ 3 の 9ぶんの1 ＝ マイナス2 は 3のマイナス2乗 ＝ 9ぶんの1 と同値、正しい。

## スライド⑤　一般化, 03:30

今回のポイントを一般化します。

対数の定義は、ログ エー の ビー ＝ シー と エーのシー乗 ＝ ビー が同値、です。

対数の基本性質をよっつ確認しましょう。

ひとつ目。ログ エー の 1 ＝ 0。エーの0乗 ＝ 1 だからです。

ふたつ目。ログ エー の エー ＝ 1。エーの1乗 ＝ エー だからです。

みっつ目。ログ エー の エーのエヌ乗 ＝ エヌ。エーのエヌ乗 ＝ エーのエヌ乗 だからです。

よっつ目。ログ エー の エムエヌ ＝ ログ エー の エム プラス ログ エー の エヌ。対数の積の法則です。

まず「底を何乗すれば真数になるか」という問いかけに戻ることが、対数の問題を解くときの基本姿勢です。

## スライド⑥　類題, 04:30

では類題を解いてみましょう。
ログ 5 の 125 マイナス ログ 5 の 5 を計算せよ。

ログ 5 の 125 から計算します。125 ＝ 5の3乗 なので、ログ 5 の 125 ＝ ログ 5 の 5の3乗 ＝ 3 です。

ログ 5 の 5 は ログ 5 の 5の1乗 ＝ 1 です。

よって、ログ 5 の 125 マイナス ログ 5 の 5 ＝ 3 マイナス 1 ＝ 2 です。

別解として対数の差の公式を使うと、ログ 5 の 125わる5 ＝ ログ 5 の 25 ＝ ログ 5 の 5の2乗 ＝ 2 と求めることもできます。

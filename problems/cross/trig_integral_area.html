数学II 融合 三角関数×積分 三角関数の交点と囲む面積
★★★★★ 発展目安 20 分
問　題
$0 \leq x \leq \pi$ において、曲線 $y = \sin x$ と曲線 $y = \cos 2x$ について答えよ。

(1)　2つの曲線の交点の $x$ 座標をすべて求めよ。

(2)　2つの曲線で囲まれた図形の面積 $S$ を求めよ。
なぜこの問題が重要か
「三角方程式」「上下関係の符号判定」「複数区間への積分分割」を一気に使う融合問題。
$\cos 2x = 1 - 2\sin^2 x$ の変換が鍵で、二重角公式と二次方程式の技術が試される。

（参考：京都大学 文系数学 2016年 入試問題 類題）

解説 (1) Step 1：交点の $x$ 座標を求める
1
$\sin x = \cos 2x$ を解く。$\cos 2x = 1 - 2\sin^2 x$ を代入： $$\sin x = 1 - 2\sin^2 x$$
2
整理して $\sin x$ の二次方程式にする： $$2\sin^2 x + \sin x - 1 = 0$$ 因数分解：$(2\sin x - 1)(\sin x + 1) = 0$
3
$\sin x = \dfrac{1}{2}$ または $\sin x = -1$

$0 \leq x \leq \pi$ の範囲では：
$\sin x = \dfrac{1}{2}$ → $x = \dfrac{\pi}{6},\; \dfrac{5\pi}{6}$
$\sin x = -1$ → $x = \dfrac{3\pi}{2}$（範囲外）
交点の $x$ 座標：$x = \dfrac{\pi}{6},\quad x = \dfrac{5\pi}{6}$
解説 (2) Step 2：各区間での上下関係を確認する
区間ごとに代表点で確認
$0 \leq x \leq \dfrac{\pi}{6}$
$x=0$ で試す：
$\sin 0 = 0$
$\cos 0 = 1$

$\cos 2x > \sin x$
$\dfrac{\pi}{6} \leq x \leq \dfrac{5\pi}{6}$
$x=\dfrac{\pi}{2}$ で試す：
$\sin\dfrac{\pi}{2} = 1$
$\cos\pi = -1$

$\sin x > \cos 2x$
$\dfrac{5\pi}{6} \leq x \leq \pi$
$x=\pi$ で試す：
$\sin\pi = 0$
$\cos 2\pi = 1$

$\cos 2x > \sin x$
方針
面積は 3 つの区間に分けて積分する： $$S = \underbrace{\int_0^{\pi/6}(\cos 2x - \sin x)\,dx}_{I_1} + \underbrace{\int_{\pi/6}^{5\pi/6}(\sin x - \cos 2x)\,dx}_{I_2} + \underbrace{\int_{5\pi/6}^{\pi}(\cos 2x - \sin x)\,dx}_{I_3}$$
解説 (2) Step 3：$I_1$ と $I_3$ を計算する
原始関数の準備
1
$$\int(\cos 2x - \sin x)\,dx = \frac{\sin 2x}{2} + \cos x + C$$
$I_1$
$$I_1 = \left[\frac{\sin 2x}{2} + \cos x\right]_0^{\pi/6} = \left(\frac{\sin(\pi/3)}{2} + \cos\frac{\pi}{6}\right) - \left(0 + 1\right)$$ $$= \left(\frac{\sqrt{3}/2}{2} + \frac{\sqrt{3}}{2}\right) - 1 = \left(\frac{\sqrt{3}}{4} + \frac{2\sqrt{3}}{4}\right) - 1 = \frac{3\sqrt{3}}{4} - 1$$
$I_3$
$$I_3 = \left[\frac{\sin 2x}{2} + \cos x\right]_{5\pi/6}^{\pi} = \left(0 + (-1)\right) - \left(\frac{\sin(5\pi/3)}{2} + \cos\frac{5\pi}{6}\right)$$ $$= -1 - \left(\frac{-\sqrt{3}/2}{2} + \left(-\frac{\sqrt{3}}{2}\right)\right) = -1 + \frac{\sqrt{3}}{4} + \frac{\sqrt{3}}{2} = \frac{3\sqrt{3}}{4} - 1$$
解説 (2) Step 4：$I_2$ を計算して合計する
$I_2$
$$\int(\sin x - \cos 2x)\,dx = -\cos x - \frac{\sin 2x}{2} + C$$ $$I_2 = \left[-\cos x - \frac{\sin 2x}{2}\right]_{\pi/6}^{5\pi/6}$$ 上端（$x = 5\pi/6$）：$-\cos\dfrac{5\pi}{6} - \dfrac{\sin(5\pi/3)}{2} = \dfrac{\sqrt{3}}{2} - \dfrac{-\sqrt{3}/2}{2} = \dfrac{\sqrt{3}}{2}+\dfrac{\sqrt{3}}{4} = \dfrac{3\sqrt{3}}{4}$
下端（$x = \pi/6$）：$-\cos\dfrac{\pi}{6} - \dfrac{\sin(\pi/3)}{2} = -\dfrac{\sqrt{3}}{2}-\dfrac{\sqrt{3}}{4} = -\dfrac{3\sqrt{3}}{4}$ $$I_2 = \frac{3\sqrt{3}}{4} - \left(-\frac{3\sqrt{3}}{4}\right) = \frac{3\sqrt{3}}{2}$$
合計
$$S = I_1 + I_2 + I_3 = \left(\frac{3\sqrt{3}}{4} - 1\right) + \frac{3\sqrt{3}}{2} + \left(\frac{3\sqrt{3}}{4} - 1\right)$$ $$= \frac{3\sqrt{3}}{4} + \frac{6\sqrt{3}}{4} + \frac{3\sqrt{3}}{4} - 2 = \frac{12\sqrt{3}}{4} - 2$$
$$S = 3\sqrt{3} - 2$$
答え 解答のまとめ
(1) 交点
$\cos 2x=1-2\sin^2 x$ を利用
二次方程式に帰着 $$x = \frac{\pi}{6},\quad \frac{5\pi}{6}$$
区間分割
3区間で上下が変わる
$I_1+I_2+I_3$ $$I_1 = I_3 = \frac{3\sqrt{3}}{4}-1$$
(2) 面積 $S$
$I_1+I_2+I_3$ を合計 $$S = 3\sqrt{3} - 2$$
答　え
(1)　$x = \dfrac{\pi}{6},\; \dfrac{5\pi}{6}$

(2)　$S = 3\sqrt{3} - 2$
ポイント 受験テクニック・確認事項
  • $\cos 2x$ の変換を使い分ける
    $\sin x$ と比べるときは $\cos 2x = 1-2\sin^2 x$（$\sin x$ だけの式に）。
    $\cos x$ と比べるときは $\cos 2x = 2\cos^2 x - 1$（$\cos x$ だけの式に）。
  • 交点の符号（$\sin x = -1$）の処理
    $0 \leq x \leq \pi$ では $\sin x \geq 0$ なので $\sin x = -1$ は解なし。
    範囲外の解を除くことを忘れずに。
  • 上下関係は必ず「代表点」で確認
    符号から判断するのではなく、具体的な代表値（$x=0,\, \pi/2,\, \pi$ など）を代入して確認する習慣を。
  • 対称性を活かす
    $I_1 = I_3$ になることは計算後でも確認できる。このような対称性チェックが計算ミス発見に役立つ。
  • 答えの数値確認
    $3\sqrt{3}-2 \approx 5.196-2 = 3.196$。面積の大きさとして妥当か（幅 $\pi$、高さ最大 2 程度のグラフなら数値的に合理的）確認する。
模範解答 記述答案例
三角関数と積分・面積
(1) $\cos 2x = 1-2\sin^2 x$ を利用して $\sin x = \cos 2x$ を整理する：
$2\sin^2 x + \sin x - 1 = 0 \;\Rightarrow\; (2\sin x-1)(\sin x+1)=0$
$0\leq x\leq\pi$ では $\sin x\geq 0$ なので $\sin x=-1$ は不適。
$$\sin x = \frac{1}{2} \;\Rightarrow\; x = \frac{\pi}{6},\quad \frac{5\pi}{6}$$
(2) 各区間の上下関係を代表点で確認すると、
$0\leq x\leq\tfrac{\pi}{6}$：$\cos 2x \geq \sin x$，　$\tfrac{\pi}{6}\leq x\leq\tfrac{5\pi}{6}$：$\sin x \geq \cos 2x$，　$\tfrac{5\pi}{6}\leq x\leq\pi$：$\cos 2x \geq \sin x$。
$$S = \int_0^{\pi/6}(\cos 2x-\sin x)\,dx + \int_{\pi/6}^{5\pi/6}(\sin x-\cos 2x)\,dx + \int_{5\pi/6}^{\pi}(\cos 2x-\sin x)\,dx$$
$I_1 = \left[\dfrac{\sin 2x}{2}+\cos x\right]_0^{\pi/6} = \dfrac{3\sqrt{3}}{4}-1$，　$I_3 = \left[\dfrac{\sin 2x}{2}+\cos x\right]_{5\pi/6}^{\pi} = \dfrac{3\sqrt{3}}{4}-1$
$I_2 = \left[-\cos x-\dfrac{\sin 2x}{2}\right]_{\pi/6}^{5\pi/6} = \dfrac{3\sqrt{3}}{4}-\left(-\dfrac{3\sqrt{3}}{4}\right) = \dfrac{3\sqrt{3}}{2}$
$$S = \left(\frac{3\sqrt{3}}{4}-1\right)+\frac{3\sqrt{3}}{2}+\left(\frac{3\sqrt{3}}{4}-1\right) = 3\sqrt{3}-2$$

類　題
$0 \leq x \leq \pi$ において、曲線 $y = \cos x$ と曲線 $y = \cos 2x$ で囲まれた図形の面積 $S$ を求めよ。
解法のヒント
$\cos 2x = 2\cos^2 x - 1$ を代入して $\cos x$ の二次方程式に帰着させる。

$2\cos^2 x - \cos x - 1 = 0 \;\Rightarrow\; (2\cos x + 1)(\cos x - 1) = 0$

$0 \leq x \leq \pi$ での解：$\cos x = -\dfrac{1}{2} \;\Rightarrow\; x = \dfrac{2\pi}{3}$、$\cos x = 1 \;\Rightarrow\; x = 0$（境界点）

区間 $(0, \tfrac{2\pi}{3})$ では $\cos x > \cos 2x$、$(\tfrac{2\pi}{3}, \pi)$ では $\cos 2x > \cos x$ になることを代表点で確認する。
類題 模範解答 記述答案例（類題）
三角関数と積分・面積
$\cos 2x = 2\cos^2 x - 1$ を代入して $\cos x = \cos 2x$ を整理する：
$2\cos^2 x - \cos x - 1 = 0 \;\Rightarrow\; (2\cos x+1)(\cos x-1)=0$
$0\leq x\leq\pi$ での解：$\cos x = -\dfrac{1}{2}$（$x=\dfrac{2\pi}{3}$）、$\cos x=1$（$x=0$、境界点）。
代表点で上下を確認：
$x=\dfrac{\pi}{2}$：$\cos\dfrac{\pi}{2}=0$、$\cos\pi=-1$ → $\cos x > \cos 2x$（$0 < x < \dfrac{2\pi}{3}$）
$x=\dfrac{5\pi}{6}$：$\cos\dfrac{5\pi}{6}=-\dfrac{\sqrt{3}}{2}$、$\cos\dfrac{5\pi}{3}=\dfrac{1}{2}$ → $\cos 2x > \cos x$（$\dfrac{2\pi}{3} < x \leq \pi$）
$$S = \int_0^{2\pi/3}(\cos x - \cos 2x)\,dx + \int_{2\pi/3}^{\pi}(\cos 2x - \cos x)\,dx$$
$\left[\sin x - \dfrac{\sin 2x}{2}\right]_0^{2\pi/3} = \dfrac{\sqrt{3}}{2} - \dfrac{\sin(4\pi/3)}{2} - 0 = \dfrac{\sqrt{3}}{2}+\dfrac{\sqrt{3}}{4} = \dfrac{3\sqrt{3}}{4}$
$\left[\dfrac{\sin 2x}{2} - \sin x\right]_{2\pi/3}^{\pi} = (0-0) - \left(\dfrac{\sin(4\pi/3)}{2}-\dfrac{\sqrt{3}}{2}\right) = 0 - \left(-\dfrac{\sqrt{3}}{4}-\dfrac{\sqrt{3}}{2}\right) ... $
$= \dfrac{\sqrt{3}}{4}+\dfrac{\sqrt{3}}{2} = \dfrac{3\sqrt{3}}{4}$
$$S = \frac{3\sqrt{3}}{4} + \frac{3\sqrt{3}}{4} = \frac{3\sqrt{3}}{2}$$

数学A・B 融合 確率漸化式 サイコロと確率漸化式
★★★★★ 発展目安 20 分
問　題
サイコロを繰り返し振る。$n$ 回目までに「3 の倍数の目（3 または 6）」が出た回数が奇数回である確率を $p_n$ とおく。

(1)　$p_1$ を求めよ。

(2)　$p_n$ を $n$ で表せ。
なぜこの問題が重要か
「確率」と「数列（漸化式）」を組み合わせた確率漸化式は、東大・京大・一橋大など難関大で頻出の融合テーマ。 「現在の状態」→「次の状態」の遷移を図式化し、漸化式を立てる思考法が核心。

（参考：東京大学 理系数学 確率の問題 類題）

解説 解法の全体像：状態の定義と遷移
「状態」を定義する
定義
$n$ 回目まで終えた時点で、3の倍数の目が奇数回出ている確率 $= p_n$
3の倍数の目が偶数回（0回含む）出ている確率 $= 1 - p_n$
1回の試行の確率
3の倍数の目（3, 6）
2通り / 6通り $$P = \frac{1}{3}$$
その他の目（1, 2, 4, 5）
4通り / 6通り $$P = \frac{2}{3}$$
状態遷移の図
奇数回（$p_n$）　← $\dfrac{2}{3}$（外れ）　← 奇数回（$p_{n-1}$）
← $\dfrac{1}{3}$（当たり）　← 偶数回（$1-p_{n-1}$）
解説 (1) $p_1$ を求める
考え方
1回だけ振って、3の倍数が「奇数回（＝1回）」出る確率を求める。
→ 1回目に 3 または 6 が出ればよい。
計算
$$p_1 = \frac{2}{6} = \frac{1}{3}$$
$$p_1 = \frac{1}{3}$$
解説 (2) Step 1：漸化式を立てる
$p_n$ と $p_{n-1}$ の関係を考える
場合①
$n-1$ 回目まで奇数回（確率 $p_{n-1}$）かつ $n$ 回目に3の倍数でない（確率 $\dfrac{2}{3}$）
→ 奇数回のまま → $p_{n-1} \cdot \dfrac{2}{3}$
場合②
$n-1$ 回目まで偶数回（確率 $1-p_{n-1}$）かつ $n$ 回目に3の倍数が出る（確率 $\dfrac{1}{3}$）
→ 奇数回になる → $(1-p_{n-1}) \cdot \dfrac{1}{3}$
漸化式
$$p_n = \frac{2}{3}p_{n-1} + \frac{1}{3}(1-p_{n-1}) = \frac{1}{3}p_{n-1} + \frac{1}{3}$$
解説 (2) Step 2：漸化式を解く
1
漸化式：$p_n = \dfrac{1}{3}p_{n-1} + \dfrac{1}{3}$

不動点 $\alpha$ を求める（$\alpha = \dfrac{1}{3}\alpha + \dfrac{1}{3}$）： $$\frac{2}{3}\alpha = \frac{1}{3} \;\Rightarrow\; \alpha = \frac{1}{2}$$
2
$p_n - \dfrac{1}{2} = \dfrac{1}{3}\left(p_{n-1} - \dfrac{1}{2}\right)$ と変形できる。

$\left\{p_n - \dfrac{1}{2}\right\}$ は公比 $\dfrac{1}{3}$ の等比数列。
3
初項 $p_1 - \dfrac{1}{2} = \dfrac{1}{3} - \dfrac{1}{2} = -\dfrac{1}{6}$ より： $$p_n - \frac{1}{2} = -\frac{1}{6} \cdot \left(\frac{1}{3}\right)^{n-1} = -\frac{1}{2 \cdot 3^n}$$
$$p_n = \frac{1}{2} - \frac{1}{2\cdot 3^n} = \frac{3^n - 1}{2 \cdot 3^n}$$
答え 解答のまとめ
(1) $p_1$
1回目に 3 または 6 $$p_1 = \frac{1}{3}$$
漸化式
状態遷移から導く $$p_n = \frac{1}{3}p_{n-1} + \frac{1}{3}$$
(2) $p_n$
等比数列に帰着 $$p_n = \frac{3^n - 1}{2\cdot 3^n}$$
答　え
(1)　$p_1 = \dfrac{1}{3}$

(2)　$p_n = \dfrac{3^n - 1}{2\cdot 3^n}$
検算
$n=2$ のとき：$\dfrac{9-1}{18} = \dfrac{8}{18} = \dfrac{4}{9}$
直接計算：$P(\text{奇数回}) = \binom{2}{1}\dfrac{1}{3}\cdot\dfrac{2}{3} = \dfrac{4}{9}$ ✓
ポイント 確率漸化式の解法パターン
  • ステップ①：状態を定義する
    「奇数回の確率 $p_n$」と「偶数回の確率 $1-p_n$」を明示。互いに補事象になるように設定するのがポイント。
  • ステップ②：1回分の遷移を書き出す
    「前の状態 × 今の試行の確率」を場合分けして加える。木（ツリー）図を描くと見通しがよい。
  • ステップ③：$\alpha = f(\alpha)$ で不動点を求める
    $p_n = r\,p_{n-1} + c$ の形なら $\alpha = c/(1-r)$。変形して等比数列に帰着させる。
  • よくあるミス
    「奇数回から偶数回になる」遷移を漏らしやすい。全パターンを書き出す習慣を。
  • 答えの確認
    $n \to \infty$ で $p_n \to \dfrac{1}{2}$ になるか確認（長期的に奇偶はほぼ半々に収束するはず）。
模範解答 記述答案例
確率漸化式
(1) 1回のサイコロで 3 の倍数の目（3 または 6）が出る確率は $\dfrac{2}{6}=\dfrac{1}{3}$。
$n=1$ のとき、3の倍数が奇数回（＝1回）出る確率は
$$p_1 = \frac{1}{3}$$
(2) $n-1$ 回目まで終わった時点で 3 の倍数が奇数回出ている確率は $p_{n-1}$、偶数回（0回含む）出ている確率は $1-p_{n-1}$。
$n$ 回目を終えて奇数回になるのは次の2通り：
（i）$n-1$ 回目まで奇数回、$n$ 回目が3の倍数でない：$p_{n-1} \cdot \dfrac{2}{3}$
（ii）$n-1$ 回目まで偶数回、$n$ 回目が3の倍数：$(1-p_{n-1}) \cdot \dfrac{1}{3}$
$$p_n = \frac{2}{3}p_{n-1} + \frac{1}{3}(1-p_{n-1}) = \frac{1}{3}p_{n-1} + \frac{1}{3}$$
変形すると $p_n - \dfrac{1}{2} = \dfrac{1}{3}\!\left(p_{n-1} - \dfrac{1}{2}\right)$。
よって $\left\{p_n - \dfrac{1}{2}\right\}$ は初項 $p_1 - \dfrac{1}{2} = -\dfrac{1}{6}$、公比 $\dfrac{1}{3}$ の等比数列。
$$p_n - \frac{1}{2} = -\frac{1}{6}\cdot\left(\frac{1}{3}\right)^{n-1} = -\frac{1}{2\cdot 3^n}$$
$$p_n = \frac{1}{2} - \frac{1}{2\cdot 3^n} = \frac{3^n - 1}{2\cdot 3^n}$$
類　題 数学A・B 融合 カードを引く確率漸化式
★★★★★ 発展目安 18 分
類　題
袋の中に 1, 2, 3, 4, 5 の番号が書かれたカードが 1 枚ずつ計 5 枚ある。カードを 1 枚引いて番号を確認してから元に戻す操作を $n$ 回繰り返す。

$n$ 回引いた後、偶数番号（2 または 4）のカードを引いた回数が奇数回である確率を $q_n$ とおく。$q_n$ を $n$ で表せ。
解法のヒント
偶数番号が出る確率：$\dfrac{2}{5}$　　奇数番号（1, 3, 5）が出る確率：$\dfrac{3}{5}$

漸化式：$q_n = \dfrac{3}{5}q_{n-1} + \dfrac{2}{5}(1-q_{n-1}) = \dfrac{1}{5}q_{n-1} + \dfrac{2}{5}$

不動点：$\alpha = \dfrac{1}{5}\alpha + \dfrac{2}{5}$ → $\alpha = \dfrac{1}{2}$
類題 模範解答 記述答案例（類題）
確率漸化式
偶数番号（2, 4）が出る確率 $= \dfrac{2}{5}$、奇数番号（1, 3, 5）が出る確率 $= \dfrac{3}{5}$。
$q_1 = \dfrac{2}{5}$（1回目に偶数が出る確率）。
$n$ 回目を終えて偶数回数が奇数になるのは次の2通り：
（i）$n-1$ 回まで奇数回、$n$ 回目が奇数番号：$q_{n-1}\cdot\dfrac{3}{5}$
（ii）$n-1$ 回まで偶数回、$n$ 回目が偶数番号：$(1-q_{n-1})\cdot\dfrac{2}{5}$
$$q_n = \frac{3}{5}q_{n-1} + \frac{2}{5}(1-q_{n-1}) = \frac{1}{5}q_{n-1} + \frac{2}{5}$$
変形すると $q_n - \dfrac{1}{2} = \dfrac{1}{5}\!\left(q_{n-1} - \dfrac{1}{2}\right)$。
初項 $q_1 - \dfrac{1}{2} = \dfrac{2}{5}-\dfrac{1}{2} = -\dfrac{1}{10}$、公比 $\dfrac{1}{5}$ の等比数列なので：
$$q_n - \frac{1}{2} = -\frac{1}{10}\cdot\left(\frac{1}{5}\right)^{n-1} = -\frac{1}{2\cdot 5^n}$$
$$q_n = \frac{1}{2} - \frac{1}{2\cdot 5^n} = \frac{5^n - 1}{2\cdot 5^n}$$

数学 III 積分法 三角置換による定積分
★★★★☆ 上級目安 15 分
問　題
次の定積分を求めよ。 $$\int_0^1 \frac{x^2}{\sqrt{1-x^2}}\,dx$$
考え方のヒント
被積分関数に $\sqrt{1-x^2}$ が含まれている。
→ $x = \sin\theta$ の三角置換を使うと根号が外れてシンプルになる。
置換後は半角公式 $\sin^2\theta = \dfrac{1-\cos 2\theta}{2}$ で計算する。

（参考：東京大学 理系数学 入試問題 類題）

解説 Step 1：置換の設定
1 $x = \sin\theta$（$0 \leq \theta \leq \dfrac{\pi}{2}$）とおく。
2 $\dfrac{dx}{d\theta} = \cos\theta$ より $dx = \cos\theta\,d\theta$
3 根号の処理：$\sqrt{1-x^2} = \sqrt{1-\sin^2\theta} = |\cos\theta| = \cos\theta$
（$0 \leq \theta \leq \dfrac{\pi}{2}$ の範囲では $\cos\theta \geq 0$ なので絶対値が外れる）
4 積分範囲の変換：
$x$$\theta$
$0$$0$
$1$$\dfrac{\pi}{2}$
解説 Step 2：被積分関数を $\theta$ で表す
5 各部分を $\theta$ で置き換える： $$\frac{x^2}{\sqrt{1-x^2}}\,dx = \frac{\sin^2\theta}{\cos\theta}\cdot\cos\theta\,d\theta$$
6 $\cos\theta$ が約分されてシンプルになる： $$= \sin^2\theta\,d\theta$$
7 積分全体を書き直すと： $$\int_0^1 \frac{x^2}{\sqrt{1-x^2}}\,dx = \int_0^{\pi/2}\sin^2\theta\,d\theta$$
解説 Step 3：$\displaystyle\int_0^{\pi/2}\sin^2\theta\,d\theta$ を計算
8 半角公式：$\sin^2\theta = \dfrac{1-\cos 2\theta}{2}$ を使う。 $$\int_0^{\pi/2}\sin^2\theta\,d\theta = \int_0^{\pi/2}\frac{1-\cos 2\theta}{2}\,d\theta$$
9 積分を実行する： $$= \left[\frac{\theta}{2} - \frac{\sin 2\theta}{4}\right]_0^{\pi/2}$$
10 上端 $\theta = \dfrac{\pi}{2}$ を代入：$\quad \dfrac{\pi/2}{2} - \dfrac{\sin\pi}{4} = \dfrac{\pi}{4} - 0 = \dfrac{\pi}{4}$
下端 $\theta = 0$ を代入：$\quad \dfrac{0}{2} - \dfrac{\sin 0}{4} = 0$

$$= \frac{\pi}{4} - 0 = \frac{\pi}{4}$$
答え 計算結果
答　え
$$\int_0^1 \frac{x^2}{\sqrt{1-x^2}}\,dx = \frac{\pi}{4}$$
計算の流れをまとめると
$x = \sin\theta$ で置換 $\;\Rightarrow\;$ $\displaystyle\int_0^{\pi/2}\sin^2\theta\,d\theta$ $\;\Rightarrow\;$ 半角公式 $\;\Rightarrow\;$ $\dfrac{\pi}{4}$
ポイント 三角置換の使い分けと注意点
  • $\sqrt{a^2 - x^2}$ 型：$x = a\sin\theta$（$-\dfrac{\pi}{2} \leq \theta \leq \dfrac{\pi}{2}$）で置換
  • $\sqrt{a^2 + x^2}$ 型（参考）：$x = a\tan\theta$ で置換 → $\sqrt{a^2+a^2\tan^2\theta}=\dfrac{a}{\cos\theta}$
  • $\sqrt{x^2 - a^2}$ 型（参考）：$x = \dfrac{a}{\cos\theta}$ で置換 → $\sqrt{\frac{a^2}{\cos^2\theta}-a^2} = a\tan\theta$
  • 置換後に $\cos\theta$ が正かどうかを必ず確認する。積分範囲を $\left[0,\dfrac{\pi}{2}\right]$ に収めれば $\cos\theta \geq 0$ が保証される
  • 必須公式：$\sin^2\theta = \dfrac{1-\cos 2\theta}{2}$、$\cos^2\theta = \dfrac{1+\cos 2\theta}{2}$
類　題 数学 III $\sqrt{1-x^2}$ を含む定積分
★★★★☆ 上級目安 15 分
類　題
次の定積分を求めよ。 $$\int_0^{\sqrt{3}/2}\sqrt{1-x^2}\,dx$$
解法のヒント
$x = \sin\theta$ で置換する。$x = \dfrac{\sqrt{3}}{2}$ のとき $\sin\theta = \dfrac{\sqrt{3}}{2}$、 すなわち $\theta = \dfrac{\pi}{3}$。

置換後：$\displaystyle\int_0^{\pi/3}\cos^2\theta\,d\theta$

半角公式 $\cos^2\theta = \dfrac{1+\cos 2\theta}{2}$ を使って計算する。
類題 模範解答 記述答案例（類題）
三角置換・半角公式
$x = \sin\theta$（$0 \leq \theta \leq \dfrac{\pi}{2}$）とおくと $dx = \cos\theta\,d\theta$、$\sqrt{1-x^2} = \cos\theta$。
積分範囲：$x: 0 \to \dfrac{\sqrt{3}}{2}$ のとき $\theta: 0 \to \dfrac{\pi}{3}$
$$\int_0^{\sqrt{3}/2}\sqrt{1-x^2}\,dx = \int_0^{\pi/3}\cos\theta \cdot \cos\theta\,d\theta = \int_0^{\pi/3}\cos^2\theta\,d\theta$$
半角公式 $\cos^2\theta = \dfrac{1+\cos 2\theta}{2}$ を用いると、
$$= \int_0^{\pi/3}\frac{1+\cos 2\theta}{2}\,d\theta = \left[\frac{\theta}{2}+\frac{\sin 2\theta}{4}\right]_0^{\pi/3}$$
$$= \frac{\pi}{6} + \frac{\sin\!\left(\dfrac{2\pi}{3}\right)}{4} - 0 = \frac{\pi}{6} + \frac{\dfrac{\sqrt{3}}{2}}{4} = \frac{\pi}{6} + \frac{\sqrt{3}}{8}$$
$$\int_0^{\sqrt{3}/2}\sqrt{1-x^2}\,dx = \frac{\pi}{6}+\frac{\sqrt{3}}{8}$$

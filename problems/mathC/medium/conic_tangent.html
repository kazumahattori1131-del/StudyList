数学 C 二次曲線 楕円と直線の接線
★★★☆☆ 中級目安 12 分
問　題
楕円 $\dfrac{x^2}{9} + \dfrac{y^2}{4} = 1$ に直線 $y = x + k$ が接するとき、$k$ の値を求めよ。また、そのときの接点の座標を求めよ。
考え方のヒント
直線の式を楕円の方程式に代入して $x$ の二次方程式を作る。
直線が楕円に接する条件は、その二次方程式の判別式 $D = 0$（重解をもつ）。
$k$ が求まったら重解 $x$ の値から接点の座標を計算する。
解説 Step 1：直線を楕円の方程式に代入する
1 $y = x + k$ を $\dfrac{x^2}{9} + \dfrac{y^2}{4} = 1$ に代入する： $$\frac{x^2}{9} + \frac{(x+k)^2}{4} = 1$$
2 両辺に $36$ を掛けて分母を払う： $$4x^2 + 9(x+k)^2 = 36$$ $$4x^2 + 9(x^2 + 2kx + k^2) = 36$$ $$4x^2 + 9x^2 + 18kx + 9k^2 = 36$$
3 整理する： $$13x^2 + 18kx + 9k^2 - 36 = 0 \quad \cdots (*)$$
代入後の方程式
$$13x^2 + 18kx + (9k^2 - 36) = 0$$
解説 Step 2：接する条件（判別式 $= 0$）
4 直線が楕円に接する $\Longleftrightarrow$ $(*)$ が重解をもつ $\Longleftrightarrow$ 判別式 $D = 0$。
$(*)$ の判別式 $D$ を計算する： $$D = (18k)^2 - 4 \cdot 13 \cdot (9k^2 - 36)$$ $$= 324k^2 - 52(9k^2 - 36)$$ $$= 324k^2 - 468k^2 + 1872$$ $$= -144k^2 + 1872$$
5 $D = 0$ とおく： $$-144k^2 + 1872 = 0 \quad \Longrightarrow \quad k^2 = \frac{1872}{144} = 13$$
$k$ の値
$$k = \pm\sqrt{13}$$
解説 Step 3：接点の座標を求める
6 $k = \sqrt{13}$ のとき：
$(*)$ は $13x^2 + 18\sqrt{13}\,x + 9 \cdot 13 - 36 = 0$、すなわち $13x^2 + 18\sqrt{13}\,x + 81 = 0$。
重解なので $x = \dfrac{-18\sqrt{13}}{2 \times 13} = \dfrac{-18\sqrt{13}}{26} = -\dfrac{9\sqrt{13}}{13}$。
$y = x + \sqrt{13} = -\dfrac{9\sqrt{13}}{13} + \sqrt{13} = \dfrac{-9\sqrt{13} + 13\sqrt{13}}{13} = \dfrac{4\sqrt{13}}{13}$。
7 $k = -\sqrt{13}$ のとき：
楕円と直線はともに原点対称なので、接点は $k=\sqrt{13}$ のときの接点と原点対称。
接点の座標は $\left(\dfrac{9\sqrt{13}}{13},\ -\dfrac{4\sqrt{13}}{13}\right)$。
答　え
$k = \sqrt{13}$ のとき接点 $\left(-\dfrac{9\sqrt{13}}{13},\ \dfrac{4\sqrt{13}}{13}\right)$、
$k = -\sqrt{13}$ のとき接点 $\left(\dfrac{9\sqrt{13}}{13},\ -\dfrac{4\sqrt{13}}{13}\right)$。
ポイント 受験テクニック・確認事項
  • 楕円と直線の接線問題の定番アプローチ：
    代入 → 整理（$x$ の二次方程式）→ 判別式 $D = 0$
  • 一般公式：楕円 $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$ に接する傾き $m$ の直線は $$y = mx \pm \sqrt{a^2 m^2 + b^2}$$
  • この問題では $a^2 = 9$、$b^2 = 4$、$m = 1$ なので $k = \pm\sqrt{9 \cdot 1 + 4} = \pm\sqrt{13}$ と素早く確認できる。
  • 接点の座標は重解を $x$ 座標として求める。$D=0$ の重解公式 $x = -\dfrac{b}{2a}$ を使うと速い。
  • 対称性（$k$ の符号変化 $\Leftrightarrow$ 接点が原点対称）を利用すると計算を半減できる。
模範解答 記述答案例（そのまま使えるレベル）
楕円への接線・判別式法
$y = x + k$ を $\dfrac{x^2}{9} + \dfrac{y^2}{4} = 1$ に代入し、両辺に $36$ を掛けると、
$$4x^2 + 9(x+k)^2 = 36 \quad \Longrightarrow \quad 13x^2 + 18kx + 9k^2 - 36 = 0 \quad \cdots (*)$$
直線が楕円に接する条件は $(*)$ の判別式 $D = 0$：
$$D = (18k)^2 - 4 \cdot 13 \cdot (9k^2-36) = 324k^2 - 468k^2 + 1872 = -144k^2 + 1872 = 0$$
$$k^2 = 13 \quad \therefore k = \pm\sqrt{13}$$
$k = \sqrt{13}$ のとき：$(*)$ は $13x^2 + 18\sqrt{13}\,x + 81 = 0$。
重解 $x = -\dfrac{18\sqrt{13}}{26} = -\dfrac{9\sqrt{13}}{13}$。
$y = -\dfrac{9\sqrt{13}}{13} + \sqrt{13} = \dfrac{4\sqrt{13}}{13}$。
接点 $\left(-\dfrac{9\sqrt{13}}{13},\ \dfrac{4\sqrt{13}}{13}\right)$。
$k = -\sqrt{13}$ のとき：対称性より接点 $\left(\dfrac{9\sqrt{13}}{13},\ -\dfrac{4\sqrt{13}}{13}\right)$。
$k = \sqrt{13}$ のとき接点 $\left(-\dfrac{9\sqrt{13}}{13},\ \dfrac{4\sqrt{13}}{13}\right)$、$k = -\sqrt{13}$ のとき接点 $\left(\dfrac{9\sqrt{13}}{13},\ -\dfrac{4\sqrt{13}}{13}\right)$。

類　題
楕円 $\dfrac{x^2}{4} + \dfrac{y^2}{3} = 1$ に直線 $y = 2x + k$ が接するとき、$k$ の値を求めよ。
解答
1 $y = 2x+k$ を代入し $12$ を掛けて整理： $$\frac{x^2}{4} + \frac{(2x+k)^2}{3} = 1 \;\Longrightarrow\; 3x^2 + 4(2x+k)^2 = 12$$ $$3x^2 + 16x^2 + 16kx + 4k^2 = 12 \;\Longrightarrow\; 19x^2 + 16kx + (4k^2-12) = 0$$
2 判別式 $D = 0$： $$D = (16k)^2 - 4 \cdot 19 \cdot (4k^2-12) = 256k^2 - 76(4k^2-12)$$ $$= 256k^2 - 304k^2 + 912 = -48k^2 + 912 = 0$$ $$k^2 = 19 \quad \therefore k = \pm\sqrt{19}$$
答　え
$$k = \pm\sqrt{19}$$ （一般公式でも確認：$\pm\sqrt{a^2m^2+b^2} = \pm\sqrt{4\cdot4+3} = \pm\sqrt{19}$ ✓）
類題 模範解答 記述答案例（類題）
楕円 $\dfrac{x^2}{4}+\dfrac{y^2}{3}=1$、直線 $y=2x+k$
$y = 2x + k$ を $\dfrac{x^2}{4} + \dfrac{y^2}{3} = 1$ に代入し、両辺に $12$ を掛けると、
$$3x^2 + 4(2x+k)^2 = 12 \quad \Longrightarrow \quad 19x^2 + 16kx + 4k^2 - 12 = 0 \quad \cdots (*)$$
直線が楕円に接する条件は $(*)$ の判別式 $D = 0$：
$$D = (16k)^2 - 4 \cdot 19 \cdot (4k^2-12) = 256k^2 - 304k^2 + 912 = -48k^2 + 912 = 0$$
$$k^2 = \frac{912}{48} = 19$$
$$k = \pm\sqrt{19}$$

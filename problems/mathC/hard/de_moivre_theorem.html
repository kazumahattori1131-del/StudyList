【数学C】ド・モアブルの定理と多倍角公式

★★★★☆ 上級 ⏱ 目安：18分

問題

(1) ド・モアブルの定理を用いて、次の等式を示せ。

$$\cos 3\theta = 4\cos^3\theta - 3\cos\theta$$

(2) (1) の結果を利用して、$\cos 20°$ は方程式

$$8x^3 - 6x - 1 = 0$$

の解であることを示せ。

解法のヒント
  • (1)：$(\cos\theta+i\sin\theta)^3$ を展開し、実部を比較する
  • (2)：(1) に $\theta=20°$ を代入し、$\cos60°=\dfrac{1}{2}$ を利用する

（参考：東京大学 理系数学 複素数平面（類題））

ド・モアブルの定理（復習）

定理の内容

$$(\cos\theta+i\sin\theta)^n = \cos n\theta + i\sin n\theta$$

$n$ は整数、$\theta$ は実数

この定理が強力な理由
  • 複素数の累乗を、偏角の $n$ 倍という単純な操作に帰着できる
  • 左辺を二項展開 $\rightarrow$ 実部・虚部を比較することで $\cos n\theta$・$\sin n\theta$ の公式を導ける
  • $n=2$：$\cos2\theta=\cos^2\theta-\sin^2\theta$、$\sin2\theta=2\sin\theta\cos\theta$（倍角公式）が即座に得られる
今回は $n=3$ の場合を使って $\cos3\theta$ の公式を導く

(1) Step 1：左辺を展開

二項定理による展開

ド・モアブルの定理より：

$$(\cos\theta+i\sin\theta)^3 = \cos3\theta + i\sin3\theta$$

左辺を二項定理で展開する：

展開
$$\cos^3\theta + 3\cos^2\theta(i\sin\theta) + 3\cos\theta(i\sin\theta)^2 + (i\sin\theta)^3$$
整理
$i^2=-1$、$i^3=-i$ を代入： $$= \cos^3\theta + 3i\cos^2\theta\sin\theta - 3\cos\theta\sin^2\theta - i\sin^3\theta$$
実虚
実部：$\cos^3\theta - 3\cos\theta\sin^2\theta$
虚部：$3\cos^2\theta\sin\theta - \sin^3\theta$

(1) Step 2：実部を比較して完成

実部の比較と $\sin^2\theta$ の置き換え

両辺の実部を比較すると：

$$\cos3\theta = \cos^3\theta - 3\cos\theta\sin^2\theta$$
置換
$\sin^2\theta = 1-\cos^2\theta$ を代入： $$= \cos^3\theta - 3\cos\theta(1-\cos^2\theta)$$
展開
$$= \cos^3\theta - 3\cos\theta + 3\cos^3\theta$$
結論
$$= 4\cos^3\theta - 3\cos\theta \quad \square$$
$\cos3\theta = 4\cos^3\theta - 3\cos\theta$  が示された。

(2) $\theta=20°$ を代入

既知値の利用

(1) で示した $\cos3\theta = 4\cos^3\theta-3\cos\theta$ に $\theta=20°$ を代入する：

1
$$\cos(3\times20°) = 4\cos^320°-3\cos20°$$ $$\cos60° = 4\cos^320°-3\cos20°$$
2
$\cos60°=\dfrac{1}{2}$ を代入： $$\frac{1}{2} = 4\cos^320°-3\cos20°$$
3
両辺に $2$ をかける： $$1 = 8\cos^320°-6\cos20°$$ $$8\cos^320°-6\cos20°-1=0$$
$x=\cos20°$ とおくと $8x^3-6x-1=0$ を満たす。∴ $\cos20°$ は $8x^3-6x-1=0$ の解である。 □

発展：$8x^3-6x-1=0$ の3つの解

解の美しい対称性

$8x^3-6x-1=0$ は3次方程式なので解が3つある。$\cos3\theta=\dfrac{1}{2}$ の解を考えると：

$$3\theta = 60°+360°k \quad (k=0,1,2)$$ $$\theta = 20°,\quad 140°,\quad 260°$$
$k$$\theta$解 $x=\cos\theta$別表示
$0$$20°$$\cos20°$約 $0.940$
$1$$140°$$\cos140°$$-\cos40°\approx -0.766$
$2$$260°$$\cos260°$$-\cos80°\approx -0.174$
  • 3つの解は $\cos(20°+120°k)$（$k=0,1,2$）という規則的な形をしている
  • 解の和：$\cos20°+\cos140°+\cos260° = 0$（解と係数の関係：$x^3$ の係数が $0$）

解法のポイント

  • ド・モアブルの定理の活用
    $(\cos\theta+i\sin\theta)^n = \cos n\theta+i\sin n\theta$ — 右辺は "偏角を $n$ 倍"、左辺は "二項展開"
  • 実部・虚部の分離
    展開後、実部比較 $\Rightarrow$ $\cos n\theta$、虚部比較 $\Rightarrow$ $\sin n\theta$ の公式が得られる
  • $\sin^2\theta = 1-\cos^2\theta$ の置き換え
    $\cos\theta$ だけの式に整理するために必須。虚部も同様に $\cos^2\theta=1-\sin^2\theta$ を使う
  • 既知の特殊角値の活用
    $\cos60°=\dfrac{1}{2}$、$\cos90°=0$ など既知値を代入して特殊角の方程式を構成できる
  • 類題
    同様の手法で $\cos5\theta = 16\cos^5\theta-20\cos^3\theta+5\cos\theta$ も示せる

模範解答

解答

(1) ド・モアブルの定理より：
$$(\cos\theta+i\sin\theta)^3 = \cos3\theta + i\sin3\theta \quad \cdots (*)$$
左辺を二項定理で展開する：
$$(\cos\theta+i\sin\theta)^3 = \cos^3\theta + 3i\cos^2\theta\sin\theta - 3\cos\theta\sin^2\theta - i\sin^3\theta$$
$= (\cos^3\theta - 3\cos\theta\sin^2\theta) + i(3\cos^2\theta\sin\theta - \sin^3\theta)$
$(*)$ の実部を比較する：
$$\cos3\theta = \cos^3\theta - 3\cos\theta\sin^2\theta$$
$\sin^2\theta = 1-\cos^2\theta$ を代入：
$$= \cos^3\theta - 3\cos\theta(1-\cos^2\theta) = \cos^3\theta - 3\cos\theta + 3\cos^3\theta$$
$$\cos3\theta = 4\cos^3\theta - 3\cos\theta \quad \square$$
(2) (1) の等式に $\theta = 20°$ を代入する：
$$\cos60° = 4\cos^320°-3\cos20°$$
$\cos60°=\dfrac{1}{2}$ より：
$$\frac{1}{2} = 4\cos^320°-3\cos20°$$
両辺に $2$ をかけると：
$$1 = 8\cos^320°-6\cos20°$$
$$8\cos^320°-6\cos20°-1=0$$
$x=\cos20°$ とおくと $8x^3-6x-1=0$ を満たす。したがって $\cos20°$ は方程式 $8x^3-6x-1=0$ の解である。 □

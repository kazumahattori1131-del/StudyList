数学 I 二次不等式 二次不等式（連立不等式と整数解）
★★★☆☆ 中級目安 10 分
問　題
次の連立不等式を解き、整数解をすべて求めよ。 $$x^2 - 3x - 4 < 0 \qquad \cdots ①$$ $$2x + 1 > x - 2 \qquad \cdots ②$$
考え方のヒント
各不等式を別々に解いてから共通範囲を求めるのが基本方針。
①は二次不等式なので因数分解、②は一次不等式なのでそのまま整理する。
解説 Step 1：① $x^2 - 3x - 4 < 0$ を解く
1 左辺を因数分解する。 $$x^2 - 3x - 4 = (x + 1)(x - 4)$$
2 不等式を変形する。 $$(x + 1)(x - 4) < 0$$
3 $(x - \alpha)(x - \beta) < 0 \;(\alpha < \beta)$ の解は $\alpha < x < \beta$ であるから、 $$\therefore\; -1 < x < 4$$
解説 Step 2：② $2x + 1 > x - 2$ を解く
1 $x$ の項を左辺、定数項を右辺に移項する。 $$2x - x > -2 - 1$$
2 整理する。 $$x > -3$$
まとめ
① の解：$-1 < x < 4$
② の解：$x > -3$
次のスライドで数直線を使って共通範囲を確認する。
解説 Step 3：共通範囲を数直線で整理し、整数解を求める
① の範囲
$$-1 < x < 4$$
② の範囲
$$x > -3$$
共通範囲
$$-1 < x < 4$$
整数解の列挙
$-1 < x < 4$ を満たす整数は（端点 $-1$, $4$ は含まないことに注意）： $$x = 0,\; 1,\; 2,\; 3$$
答　え
$$x = 0,\ 1,\ 2,\ 3$$
ポイント 受験テクニック・確認事項
  • 二次不等式の解き方の手順：「因数分解 → 解の順序を確認 → 不等号の方向に注意」
  • 公式として覚える：$(x-\alpha)(x-\beta) < 0 \;(\alpha < \beta)$ の解は $\alpha < x < \beta$
  • 連立不等式は数直線で：各不等式の解を数直線上に図示して共通範囲を視覚的に確認する
  • 整数解を求める際の注意：開区間（$<$）では端点を含まない。$-1 < x < 4$ の整数解は $-1$ と $4$ を除いた $0, 1, 2, 3$ の 4 個
模範解答 記述答案例（そのまま使えるレベル）
二次不等式・連立
① $x^2 - 3x - 4 < 0$ を解く。
左辺を因数分解すると $(x+1)(x-4) < 0$
$$\therefore\; -1 < x < 4 \qquad \cdots \text{(i)}$$
② $2x + 1 > x - 2$ を解く。
$x > -3 \qquad \cdots \text{(ii)}$
(i), (ii) の共通範囲を求めると、
$$-1 < x < 4$$
整数解は $x = 0,\; 1,\; 2,\; 3$

元問題との違いを確認
元問題類題
二次不等式$x^2 - 3x - 4 < 0$（開区間）$x^2 - 5x + 4 \leq 0$（閉区間）
一次不等式$2x + 1 > x - 2$$2x - 3 > x - 1$
不等号$<$（端点含まず）$\leq$（端点含む）に注意
$\leq$ と $<$ で整数解の個数が変わることがあるので不等号に注意。
類題 模範解答 記述答案例（類題）
$\leq$ の二次不等式
① $x^2 - 5x + 4 \leq 0$ を解く。
左辺を因数分解すると $(x-1)(x-4) \leq 0$
$$\therefore\; 1 \leq x \leq 4 \qquad \cdots \text{(i)}$$
② $2x - 3 > x - 1$ を解く。
$x > 2 \qquad \cdots \text{(ii)}$
(i), (ii) の共通範囲を求めると、
$$2 < x \leq 4$$
整数解は $x = 3,\; 4$

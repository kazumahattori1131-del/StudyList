数学 I 二次関数 二次関数の最小値（パラメータ場合分け）
★★★☆☆ 中級目安 15 分
問　題
実数 $a$ を定数とする。関数 $$f(x) = x^2 - 2ax + 2a + 3$$ について、$0 \leq x \leq 3$ における $f(x)$ の最小値 $m(a)$ を求めよ。
考え方のヒント
放物線の頂点の $x$ 座標が定義域 $0 \leq x \leq 3$ のどこにあるかによって最小値の場所が変わる。
→ 頂点の位置で場合分け が解法の核心！
解説 Step 1：平方完成で頂点を求める
1 平方完成： $$f(x) = (x - a)^2 - a^2 + 2a + 3$$ 頂点 $\left(a,\ -a^2 + 2a + 3\right)$
2 場合分けの境界は頂点 $x=a$ が区間端と一致するとき： $a=0$（左端） と $a=3$（右端） の 2 点
解説 Step 2：場合分けして最小値を求める
場合 1：$a < 0$
頂点が左外 → 単調増加
最小は $x=0$ $$m(a) = f(0) = 2a+3$$
場合 2：$0 \leq a \leq 3$
頂点が区間内
最小は頂点の値 $$m(a) = -a^2+2a+3$$
場合 3：$a > 3$
頂点が右外 → 単調減少
最小は $x=3$ $$m(a) = f(3) = -4a+12$$
答　え
$$m(a) = \begin{cases} 2a + 3 & (a < 0) \\ -a^2 + 2a + 3 & (0 \leq a \leq 3) \\ -4a + 12 & (a > 3) \end{cases}$$
ポイント 受験テクニック・確認事項
  • 場合分けの境界は「頂点 $x=a$ が区間端 $x=0$，$x=3$ と一致するとき」
  • 各境界で答えが一致するか確認（連続性チェック）
    $a=0$：$2(0)+3=3$，$-(0)^2+2(0)+3=3$ ✓
    $a=3$：$-(3)^2+2(3)+3=0$，$-4(3)+12=0$ ✓
  • 青チャート・フォーカスゴールドともに「最重要例題」に分類される形式
  • 「場合の数だけ場合分け境界がある」→ 定義域の端点の数 = 場合の数 − 1
模範解答 記述答案例（そのまま使えるレベル）
二次関数の最小値
$f(x)$ を平方完成すると $f(x) = (x-a)^2 - a^2 + 2a + 3$ となり、頂点の $x$ 座標は $a$。
定義域 $0 \leq x \leq 3$ に対する頂点の位置で次のように場合分けする。
［i］ $a < 0$ のとき
$f(x)$ は $0 \leq x \leq 3$ において単調増加ゆえ、最小値は $x=0$ で実現される。 $$m(a) = f(0) = 2a + 3$$
［ii］ $0 \leq a \leq 3$ のとき
頂点が区間内に含まれるから、最小値は頂点の $y$ 座標に等しい。 $$m(a) = -a^2 + 2a + 3$$
［iii］ $a > 3$ のとき
$f(x)$ は $0 \leq x \leq 3$ において単調減少ゆえ、最小値は $x=3$ で実現される。 $$m(a) = f(3) = 9 - 6a + 2a + 3 = -4a + 12$$
以上より $$m(a) = \begin{cases} 2a + 3 & (a < 0) \\ -a^2 + 2a + 3 & (0 \leq a \leq 3) \\ -4a + 12 & (a > 3) \end{cases}$$
類　題 数学 I 定義域・係数を変えた類似問題

類　題
実数 $a$ を定数とする。関数 $$g(x) = x^2 - 2ax + 3a - 2$$ について、$0 \leq x \leq 4$ における $g(x)$ の最小値 $n(a)$ を求めよ。
元問題との違いを確認
元問題類題
関数$x^2 - 2ax + 2a + 3$$x^2 - 2ax + 3a - 2$
定義域$0 \leq x \leq 3$$0 \leq x \leq 4$
場合分け境界$a=0,\ a=3$$a=0,\ a=4$
→ 解法の構造は全く同じ。定数項と定義域だけが変化
類題解説 Step 1 → 2：平方完成と場合分け
1 平方完成： $$g(x) = (x - a)^2 - a^2 + 3a - 2$$ 頂点 $\left(a,\ -a^2 + 3a - 2\right)$　　境界：$a=0$（左端）, $a=4$（右端）
場合 1：$a < 0$
頂点が左外 → 単調増加
最小は $x=0$ $$n(a) = g(0) = 3a-2$$
場合 2：$0 \leq a \leq 4$
頂点が区間内
最小は頂点の値 $$n(a) = -a^2+3a-2$$
場合 3：$a > 4$
頂点が右外 → 単調減少
最小は $x=4$ $$n(a) = g(4) = -5a+14$$
連続性チェック： $a=0$：$3(0)-2=-2$，$-(0)^2+3(0)-2=-2$ ✓ $a=4$：$-(16)+12-2=-6$，$-5(4)+14=-6$ ✓
類題 模範解答 記述答案例（類題）
定義域 $[0,4]$・定数項 $3a-2$
$g(x)$ を平方完成すると $g(x) = (x-a)^2 - a^2 + 3a - 2$ となり、頂点の $x$ 座標は $a$。
定義域 $0 \leq x \leq 4$ に対する頂点の位置で次のように場合分けする。
［i］ $a < 0$ のとき
$g(x)$ は $0 \leq x \leq 4$ において単調増加ゆえ、最小値は $x=0$ で実現される。 $$n(a) = g(0) = 3a - 2$$
［ii］ $0 \leq a \leq 4$ のとき
頂点が区間内に含まれるから、最小値は頂点の $y$ 座標に等しい。 $$n(a) = -a^2 + 3a - 2$$
［iii］ $a > 4$ のとき
$g(x)$ は $0 \leq x \leq 4$ において単調減少ゆえ、最小値は $x=4$ で実現される。 $$n(a) = g(4) = 16 - 8a + 3a - 2 = -5a + 14$$
以上より $$n(a) = \begin{cases} 3a - 2 & (a < 0) \\ -a^2 + 3a - 2 & (0 \leq a \leq 4) \\ -5a + 14 & (a > 4) \end{cases}$$

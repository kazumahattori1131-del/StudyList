数学 I 二次関数 二次関数の最小値（パラメータ場合分け）
★★★☆☆ 中級目安 15 分
問　題
実数 $a$ を定数とする。関数 $$f(x) = x^2 - 2ax + 2a + 3$$ について、$0 \leq x \leq 3$ における $f(x)$ の最小値 $m(a)$ を求めよ。
考え方のヒント
放物線の頂点の $x$ 座標が定義域 $[0,\ 3]$ のどこにあるかによって最小値の場所が変わる。
→ 頂点の位置で場合分け が解法の核心！
解説 Step 1：平方完成で頂点を求める
1 平方完成： $$f(x) = (x - a)^2 - a^2 + 2a + 3$$ 頂点 $\left(a,\ -a^2 + 2a + 3\right)$
2 場合分けの境界は頂点 $x=a$ が区間端と一致するとき： $a=0$（左端） と $a=3$（右端） の 2 点
解説 Step 2：場合分けして最小値を求める
場合 1：$a < 0$
頂点が左外 → 単調増加
最小は $x=0$ $$m(a) = f(0) = 2a+3$$
場合 2：$0 \leq a \leq 3$
頂点が区間内
最小は頂点の値 $$m(a) = -a^2+2a+3$$
場合 3：$a > 3$
頂点が右外 → 単調減少
最小は $x=3$ $$m(a) = f(3) = -4a+12$$
答　え
$$m(a) = \begin{cases} 2a + 3 & (a < 0) \\ -a^2 + 2a + 3 & (0 \leq a \leq 3) \\ -4a + 12 & (a > 3) \end{cases}$$
ポイント 受験テクニック・確認事項
  • 場合分けの境界は「頂点 $x=a$ が区間端 $x=0$，$x=3$ と一致するとき」
  • 各境界で答えが一致するか確認（連続性チェック）
    $a=0$：$2(0)+3=3$，$-(0)^2+2(0)+3=3$ ✓
    $a=3$：$-(3)^2+2(3)+3=0$，$-4(3)+12=0$ ✓
  • 青チャート・フォーカスゴールドともに「最重要例題」に分類される形式
  • 発展：最小値が常に正となる $a$ の範囲は？（不等式との融合）
発展 発展問題：最小値が常に正となる $a$ の範囲
前問で求めた $m(a)$ を用いて、すべての $a$ に対して $m(a) > 0$ が成り立つような $a$ の範囲を求めよ。
1 場合 1（$a < 0$）： $m(a) = 2a + 3 > 0 \Rightarrow a > -\dfrac{3}{2}$ $$\therefore\quad -\frac{3}{2} < a < 0$$
2 場合 2（$0 \leq a \leq 3$）： $m(a) = -a^2 + 2a + 3 > 0$ $$a^2 - 2a - 3 < 0 \;\Rightarrow\; (a+1)(a-3) < 0 \;\Rightarrow\; -1 < a < 3$$ 定義域 $0 \leq a \leq 3$ との共通部分：$0 \leq a < 3$
3 場合 3（$a > 3$）： $m(a) = -4a + 12 > 0 \Rightarrow a < 3$　→　矛盾。解なし
答　え
各場合の和集合：$\displaystyle -\frac{3}{2} < a < 3$
解法の流れ
  • 前問の $m(a)$ をそのまま利用する
  • 各場合ごとに $m(a)>0$ を解く
  • 場合の条件との共通部分を取る
  • 最後に3つの結果を合わせる
数直線での確認
場合1：$\left(-\dfrac{3}{2},\ 0\right)$
場合2：$\left[0,\ 3\right)$
合計：$\left(-\dfrac{3}{2},\ 3\right)$
場合3で「解なし」になっても省略しない。
記述試験では「矛盾するため解なし」と明記すること。
よく出る派生問題
  • $m(a) \geq 0$ となる範囲（$\geq$ への変更）
  • 最大値が負となる $a$ の範囲
模範解答 記述答案例（そのまま使えるレベル）
二次関数の最小値
$f(x)$ を平方完成すると $f(x) = (x-a)^2 - a^2 + 2a + 3$ となり、頂点の $x$ 座標は $a$。
定義域 $[0,\ 3]$ に対する頂点の位置で次のように場合分けする。
［i］ $a < 0$ のとき
$f(x)$ は $[0,\ 3]$ 上で単調増加ゆえ、最小値は $x=0$ で実現される。 $$m(a) = f(0) = 2a + 3$$
［ii］ $0 \leq a \leq 3$ のとき
頂点が区間内に含まれるから、最小値は頂点の $y$ 座標に等しい。 $$m(a) = -a^2 + 2a + 3$$
［iii］ $a > 3$ のとき
$f(x)$ は $[0,\ 3]$ 上で単調減少ゆえ、最小値は $x=3$ で実現される。 $$m(a) = f(3) = 9 - 6a + 2a + 3 = -4a + 12$$
以上より $$m(a) = \begin{cases} 2a + 3 & (a < 0) \\ -a^2 + 2a + 3 & (0 \leq a \leq 3) \\ -4a + 12 & (a > 3) \end{cases}$$

数学 I 二次関数 二次関数の最大・最小（軸の移動）
★★★★☆ 上級目安 15 分
問　題
実数 $a$ を定数とする。関数 $$f(x) = x^2 - 2ax + 3a$$ の $0 \leq x \leq 2$ における最小値 $m(a)$ を、$a$ の値によって場合分けして求めよ。
考え方のヒント
放物線の軸 $x = a$ が定義域 $0 \leq x \leq 2$ のどこにあるかで最小値の場所が決まる。
→ 軸 $x=a$ の位置で場合分け が解法の核心！

（参考：センター試験 数学I・A 2015年 入試問題 類題）

解説 準備：軸・頂点・端点の値を整理する
1 平方完成して頂点を求める： $$f(x) = (x - a)^2 - a^2 + 3a$$ 軸：$x = a$　　頂点の $y$ 座標：$f(a) = -a^2 + 3a$
2 端点の値を計算する： $$f(0) = 0 - 0 + 3a = 3a$$ $$f(2) = 4 - 4a + 3a = 4 - a$$
3 放物線は上に凸ではなく上に開く（$x^2$ の係数 $= 1 > 0$）ので、 軸が区間内にあれば最小値は頂点の値、区間外にあれば端点が最小値となる。
解説 Step 1：$a < 0$ のとき
1 軸 $x = a$ は $a < 0$ より定義域 $[0, 2]$ の左外にある。
2 放物線は軸より右側では単調増加するため、$0 \leq x \leq 2$ において $f(x)$ は単調増加。
3 最小値は $x = 0$（左端）で実現される： $$m(a) = f(0) = 3a$$
イメージ
軸 $x=a$（$a<0$）　　　 $[0 \longrightarrow 2]$
← 軸 ─────── 区間全体が右肩上がり ───────→
最小値は左端 $x=0$
解説 Step 2：$0 \leq a \leq 2$ のとき
1 軸 $x = a$ が定義域 $[0, 2]$ の内部または端点にある。
2 放物線は軸 $x = a$ で最小値をとる（上に開く放物線）。
軸が区間内にあるため、最小値は頂点の $y$ 座標に等しい。
3 $$m(a) = f(a) = a^2 - 2a \cdot a + 3a = -a^2 + 3a$$
イメージ
$[0 \longrightarrow$ 軸 $x=a$ $\longrightarrow 2]$
区間内に頂点あり → 最小値 $= -a^2 + 3a$
解説 Step 3：$a > 2$ のとき
1 軸 $x = a$ は $a > 2$ より定義域 $[0, 2]$ の右外にある。
2 放物線は軸より左側では単調減少するため、$0 \leq x \leq 2$ において $f(x)$ は単調減少。
3 最小値は $x = 2$（右端）で実現される： $$m(a) = f(2) = 4 - a$$
イメージ
$[0 \longrightarrow 2]$　　　軸 $x=a$（$a>2$）→
区間全体が右肩下がり → 最小値は右端 $x=2$
まとめ 場合分けの結果まとめ
場合 1：$a < 0$
軸が左外 → 単調増加
最小は $x=0$ $$m(a) = 3a$$
場合 2：$0 \leq a \leq 2$
軸が区間内
最小は頂点の値 $$m(a) = -a^2 + 3a$$
場合 3：$a > 2$
軸が右外 → 単調減少
最小は $x=2$ $$m(a) = 4 - a$$
答　え
$$m(a) = \begin{cases} 3a & (a < 0) \\ -a^2 + 3a & (0 \leq a \leq 2) \\ 4 - a & (a > 2) \end{cases}$$
ポイント 受験テクニック・確認事項
  • 解法の核心は「軸の位置」と「区間の端点」を比較すること
  • 軸が区間内 → 最小値は頂点の $y$ 座標 $= -a^2 + 3a$
  • 軸が区間の左外 → 区間全体で単調増加 → 最小は左端 $x=0$
  • 軸が区間の右外 → 区間全体で単調減少 → 最小は右端 $x=2$
  • 各境界で答えが一致するか確認（連続性チェック）：
    $a=0$：$3 \times 0 = 0$、$-(0)^2 + 3(0) = 0$ ✓
    $a=2$：$-(2)^2 + 3(2) = -4+6 = 2$、$4-2 = 2$ ✓
  • 場合分けの境界は「軸 $x=a$ が区間端 $x=0$，$x=2$ と一致するとき」であり、境界は $a=0$ と $a=2$ の 2 点
模範解答 記述答案例（そのまま使えるレベル）
二次関数の最小値（軸の移動）
$f(x)$ を平方完成すると $f(x) = (x - a)^2 - a^2 + 3a$ となり、軸は $x = a$、頂点の $y$ 座標は $-a^2 + 3a$。
また端点の値は $f(0) = 3a$、$f(2) = 4 - a$。放物線は上に開くので、軸の位置で次のように場合分けする。
［i］ $a < 0$ のとき
軸 $x=a$ は定義域 $[0,2]$ の左外にあり、$f(x)$ は $0 \leq x \leq 2$ において単調増加。 よって最小値は $x=0$ で実現される。 $$m(a) = f(0) = 3a$$
［ii］ $0 \leq a \leq 2$ のとき
軸 $x=a$ が区間 $[0,2]$ 内に含まれるから、最小値は頂点の $y$ 座標に等しい。 $$m(a) = f(a) = -a^2 + 3a$$
［iii］ $a > 2$ のとき
軸 $x=a$ は定義域 $[0,2]$ の右外にあり、$f(x)$ は $0 \leq x \leq 2$ において単調減少。 よって最小値は $x=2$ で実現される。 $$m(a) = f(2) = 4 - a$$
以上より $$m(a) = \begin{cases} 3a & (a < 0) \\ -a^2 + 3a & (0 \leq a \leq 2) \\ 4 - a & (a > 2) \end{cases}$$

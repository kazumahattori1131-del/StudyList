数学 B 数列 漸化式（特性方程式による解法）
★★★☆☆ 中級目安 12 分
問　題
数列 $\{a_n\}$ が次の条件を満たすとき、一般項 $a_n$ を求めよ。 $$a_1 = 1, \quad a_{n+1} = 3a_n - 4$$
考え方のヒント
$a_{n+1} = 3a_n - 4$ は「$a_{n+1}$ と $a_n$ の一次式」。この型は特性方程式を使って等比数列に変換できる。
特性方程式：$\alpha = 3\alpha - 4$ を解いて $\alpha$ を求め、$b_n = a_n - \alpha$ とおく。
解説 Step 1：特性方程式で等比数列に変換する
1 特性方程式を解く： $$\alpha = 3\alpha - 4 \quad \Longrightarrow \quad -2\alpha = -4 \quad \Longrightarrow \quad \alpha = 2$$
2 $b_n = a_n - 2$ とおく。漸化式から $b_n$ の漸化式を導く： $$a_{n+1} = 3a_n - 4$$ $$b_{n+1} + 2 = 3(b_n + 2) - 4$$ $$b_{n+1} = 3b_n$$
3 $\{b_n\}$ は公比 $3$ の等比数列。初項 $b_1 = a_1 - 2 = 1 - 2 = -1$。 $$b_n = -1 \cdot 3^{n-1} = -3^{n-1}$$
解説 Step 2：$a_n$ を求めて検証する
4 $b_n = a_n - 2$ より $a_n$ を求める： $$a_n = b_n + 2 = -3^{n-1} + 2$$
5 検証：
確認内容計算結果
初期条件 $a_1$$-3^0 + 2 = -1 + 2$$= 1$ ✓
漸化式（$n=1$）$3a_1 - 4 = 3 - 4$$= -1 = a_2$ ✓
答　え
$$a_n = 2 - 3^{n-1}$$
ポイント 受験テクニック・確認事項
  • 特性方程式の意味：$\alpha = p\alpha + q$ の解 $\alpha$ は「$a_n = \alpha$ が成り立てば次も $\alpha$」という固定点
  • 手順は3ステップ：① 特性方程式を解く → ② $b_n = a_n - \alpha$ とおく → ③ 等比数列として解く
  • 最後に初期条件と漸化式の両方で検証する習慣をつける
  • 頻出パターン：$a_{n+1} = pa_n + q$（$p \neq 1$）はすべてこの手法で解ける
模範解答 記述答案例（そのまま使えるレベル）
$a_{n+1} = pa_n + q$ 型漸化式
特性方程式 $\alpha = 3\alpha - 4$ を解くと $\alpha = 2$。
$b_n = a_n - 2$ とおくと、
$$b_{n+1} = a_{n+1} - 2 = (3a_n - 4) - 2 = 3(a_n - 2) = 3b_n$$
よって $\{b_n\}$ は公比 $3$ の等比数列。
初項 $b_1 = a_1 - 2 = -1$ より、
$$b_n = -1 \cdot 3^{n-1} = -3^{n-1}$$
$$a_n = b_n + 2 = 2 - 3^{n-1}$$

手順は全く同じ。特性方程式の解が負になることに注意
類題解説 Step 1 → 2：特性方程式と等比数列への変換
1 特性方程式： $$\alpha = 2\alpha + 1 \quad \Longrightarrow \quad -\alpha = 1 \quad \Longrightarrow \quad \alpha = -1$$
2 $b_n = a_n - (-1) = a_n + 1$ とおく： $$b_{n+1} = a_{n+1} + 1 = (2a_n + 1) + 1 = 2a_n + 2 = 2(a_n + 1) = 2b_n$$ $\{b_n\}$ は公比 $2$ の等比数列。初項 $b_1 = a_1 + 1 = 4$。
3 $$b_n = 4 \cdot 2^{n-1} = 2^{n+1} \quad \Longrightarrow \quad a_n = b_n - 1 = 2^{n+1} - 1$$
検証：$a_1 = 4 - 1 = 3$ ✓　$a_2 = 2(3)+1 = 7 = 8-1$ ✓
